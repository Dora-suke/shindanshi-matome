生産管理の全体像

! 生産管理ってなに？

ひとことで言うと

「ものづくり」をうまくやりくりすることです。

原料や材料を使って、もっと価値のあるもの（＝製品）に変えること。それが「生産」。
そして、その生産をムダなく・品質よく・期日どおりに行うために管理するのが「生産管理」です。

やることは大きく3つ：

  • 開発・設計 ー どんな製品を作るか決める
  • 材料調達 ー 必要な材料を集める
  • 製品の製造 ー 実際に作る
料理に例えるとわかりやすいよ！レシピを考える（設計）→ 食材を買う（調達）→ 料理する（製造）。この3つをうまく回すのが生産管理だと思ってね！

★ 生産の目標 ＝ QCD

QCDとは？

生産管理で一番大事な3つの目標です。

  • Quality（品質）ー 良いものを作ること
  • Cost（コスト）ー できるだけ安く作ること
  • Delivery（納期）ー 約束の期日どおりに届けること

ただし、この3つはトレードオフの関係。つまり、1つを良くしようとすると、他のどれかに悪影響が出がちです。

「トレードオフ」ってめちゃくちゃ大事な言葉！たとえば、品質を上げるために高い材料を使うとコストが上がるし、コストを下げるために急いで作ると品質が下がる。この「あちら立てればこちらが立たず」がトレードオフだよ！

★ 実例で学ぶ生産管理

みんなが知ってる会社の事例で、生産管理のイメージをつかもう。

実際の企業の話を読むと「あ、これが教科書で言ってたアレか！」ってなるよ。試験の事例問題対策にもなるから、しっかり読んでね！
トヨタ自動車 ー 世界が学ぶ「トヨタ生産方式」
ライン生産 セル生産 段取り替え 5S QCサークル

トヨタはライン生産方式をベースにしながら、世界トップレベルの効率を実現しています。

「シングル段取り」の元祖
トヨタでは1960年代に、プレス機の金型交換を数時間から10分未満（シングル段取り）に短縮。「ラインを止めなくてもできる準備は、止める前にやっておこう」という発想で内段取りを外段取りに転換したのです。
QCサークルと5S
トヨタの工場では、作業者が自分たちでQCサークルを作り、毎日改善を重ねます。5Sは全工場の基本ルール。「工具は使ったら元の場所に戻す」という整頓の徹底が、ムダな探し時間をなくし、リードタイムの短縮に直結しています。
トヨタ生産方式（TPS）は試験に超出る！覚えるべきは「ジャストインタイム」と「自働化（にんべんの自動化）」の2本柱。段取り替え改善の代表例としても頻出だよ！

試験対策ポイント

トヨタ生産方式 = 「ジャストインタイム」+「自働化」。段取り替え改善のキーワードは「内段取りから外段取りへの転換」。

セブン-イレブン ー コンビニで学ぶ「見込生産」と「需要予測」
見込生産 受注生産化 QCD PDSサイクル

コンビニのおにぎりやお弁当は、「これぐらい売れるだろう」と予想して先に作る見込生産の典型例。

PDSサイクルを毎日回す
各店舗が天気・イベント・曜日を見て発注数を決める（Plan）→ 工場で作って店に届ける（Do）→ 売れ残りや品切れをデータで確認（See）。このサイクルを毎日回して予測の精度を上げています。
「受注生産化」の好例
セブン-イレブンの「予約弁当」や「クリスマスケーキ予約」は、見込生産に受注生産の要素を組み合わせたしくみ。事前に注文を取ることで、作りすぎのムダを減らしています。
コンビニは「見込生産」の身近な例として使いやすいよ。見込生産の課題 = 需要予測の精度。これを改善するためにPDSサイクルを毎日回してるんだね。

試験対策ポイント

見込生産の課題 = 需要予測の精度向上。QCDのD（納期・数量）管理が特に重要。「受注生産化」は見込生産企業の改善トレンド。

町の和菓子店「花月堂」 ー 中小企業のリアルな生産管理
受注生産 見込生産 個別生産 歩留り 4M

試験では大企業だけでなく、こういう中小企業の事例がたくさん出ます。和菓子店で考えてみましょう。

受注生産と見込生産の両方をやっている
店頭に並べる和菓子は見込生産（今日はこのくらい売れるかなと予想して朝作る）。法事や結婚式の注文菓子は受注生産（注文を受けてから作る）。同じ工房で2つの方式を使い分けています。
歩留りで差がつく
小豆100kgから作れるあんこの量が歩留り。ベテラン職人は小豆をムダなく使って歩留りが高い（たとえば70%）。新人は皮を多く捨ててしまい歩留りが低い（たとえば55%）。この差がそのまま材料コストの差＝利益の差になります。
4Mで原因を見つける
「最近味が変わった」とお客さんからクレーム → Man（職人が変わった？）、Machine（蒸し器の温度は正確？）、Material（小豆の産地が変わった？）、Method（レシピどおり作ってる？）で1つずつ確認。
和菓子店のような身近な例で考えると、4Mや歩留りがグッとわかりやすくなるでしょ？試験の事例問題も、こうやって身近なものに置き換えて考えるのがコツだよ！
超入門で全体像をつかんだら、次はここ！生産管理で使うキーワードと考え方を覚えよう。3S、5S、ECRS、生産形態、生産方式 — これが試験の土台になるよ！

⚖ 重要公式（くわしい解説付き）

試験で計算問題が出る公式を全て整理しました。全8公式、しっかり押さえましょう。

1. 生産性

生産性 = 産出量（Output）/ 投入量（Input）

産出量には生産量・生産金額・付加価値などが使われ、投入量には労働量・投入資本・設備・材料などが使われます。

例：100人で1000個作れる工場A（生産性=10）vs 50人で800個の工場B（生産性=16）
→ 人数が少ないBの方が「生産性が高い」

2. 労働生産性

労働生産性 = 付加価値額 / 従業員数

生産性の中でも頻出の指標。従業員1人あたりが生み出す付加価値を表します。財務・会計の科目とも横断的に出題されます。

例：付加価値額5000万円、従業員10人 → 労働生産性 = 500万円/人

3. 歩留り（ぶどまり）

歩留り = 産出された品物の量 / 投入された主原材料の量 × 100（%）

材料をどれだけムダなく使えたかの指標。不良品が出ると歩留りは下がる。手直しして良品にすれば計算上は向上するが、手直しにはコストがかかるため、根本的には不良品を出さないことが重要。

例：鉄100kgを投入、製品85kg → 歩留り85%（残り15kgはスクラップ）

4. 稼働率

稼働率 = 有効稼働時間 / 利用可能時間

有効稼働時間＝生産に直接役立っている時間。利用可能時間＝就業時間（機械が使える時間全体）。段取り替え・故障・待ち時間で有効稼働時間は減る。

例：利用可能時間100時間、そのうち実際に生産した時間30時間 → 稼働率 = 30%

5. 生産リードタイム

生産リードタイム = 生産の着手時期から完了時期に至るまでの時間

原材料の状態から製品として出荷できる状態になるまでの全体時間。複数工程の場合、工程間の滞留時間も含むのがポイント。加工時間だけではない。

ここまでの5つはどれも「良いもの / 全体」の形で、数値が高いほど良い。この共通点を覚えておくと混乱しにくいよ！

6. サイクルタイム（ピッチタイム）

サイクルタイム = 最も作業時間が長い工程の時間

ライン全体で製品を生産する速度を決める時間。一番遅い工程（＝ボトルネック）がライン全体のペースを決めます。合計でも平均でもないことに注意。

例：工程A=3分, B=5分, C=4分 → サイクルタイム = 5分（B工程が最大）

7. ライン編成効率（ラインバランス効率）

ライン編成効率 = 作業時間の合計 / （サイクルタイム × 作業ステーション数）

作業時間がどれだけ均等に配分されているか。100%＝完璧にバランスが取れた状態。

例：A=3分, B=5分, C=4分（3工程）
サイクルタイム=5分、ライン編成効率 = (3+5+4)/(5×3) = 12/15 = 80%

8. バランスロス率

バランスロス率 = 100% - ライン編成効率

0%が理想（ムダが全くない）。バランスロスが大きい＝手待ちや仕掛品停滞が多い。

例：上の例ではバランスロス率 = 100% - 80% = 20%
山崩しで全工程4分にすると → 12/(4×3) = 100%、ロス率 = 0%
全8公式まとめ：生産性・労働生産性・歩留り・稼働率・リードタイム → 高いほど良い。サイクルタイム・バランスロス率 → 低いほど良い。ライン編成効率 → 高い（100%に近い）ほど良い。この区別が大事！

受注生産 vs 見込生産

受注生産
  • 注文を受けてから作り始める
  • 例：注文住宅、産業用機械
  • 受注生産はさらに2つに分かれる：
    • 個別受注生産：毎回オーダーメイド。顧客ごとに設計から始める

    💡 違いのポイント：「設計が毎回必要か？」→ Yes なら個別、No なら繰返し
  • 課題：リードタイム短縮、納期を守る、受注の平準化
見込生産
  • 「これぐらい売れるだろう」と予想して先に作る
  • 例：家電、書籍、食品
  • 課題：需要予測をどれだけ正確にできるか

受注の平準化ってなに？ — 受注生産って、注文が来月に10件ドカッと来て、再来月はゼロ…みたいに波があるのが悩みなんだよね。忙しい月は残業だらけ、暇な月は人が余る。これを「平準化」＝なるべく均等にならすのが目標。具体的には、閑散期に営業をかけて注文を前倒しでもらったり、納期を調整してもらったりして、工場の負荷を一定に保つようにする。試験では「注文の偏りをならす」「負荷の山崩し」みたいな表現で出るよ。
見込生産の「受注生産化」は最近のトレンド。たとえばコンビニのクリスマスケーキを予約制にするのがまさにコレ。作りすぎを防げるから無駄が減るんだよ。

作る製品を切り替えるときの準備作業のこと。

⚙ 生産方式（くわしく）

ライン生産方式

ベルトコンベアで製品が流れていき、各担当者が自分の工程だけをこなす方式。

メリット

  • 生産性が高い
  • 管理がしやすい
  • 単能工（1つだけできる人）でOK

デメリット

  • 製品の変更に弱い
  • 設備レイアウトが固定的
  • 作業が単調でモチベーション低下

種類：単一品種ライン（1種類だけ作る）/ 多品種ライン（ライン切替方式・混合ライン方式）

ラインバランシング

各工程の作業時間のバラつきを減らして、ラインを効率よくする手法です。

  • ピッチダイアグラム：各工程の作業時間を棒グラフにしたもの
  • サイクルタイム（＝ピッチタイム）：一番遅い工程の時間。ライン全体のスピードはここで決まる
  • 山崩し：遅い工程（ボトルネック）の作業を、余裕のある工程に移すこと
サイクルタイム = 最も作業時間が長い工程の時間
ライン編成効率 = 作業時間の合計 / （サイクルタイム × 作業ステーション数）
バランスロス率 = 100% - ライン編成効率

※ ピッチダイアグラム図・詳しい計算例は応用タブで解説しています。

セル生産方式

似た部品や工程をグループ（セル）にまとめて、1人〜少人数で複数の工程を担当する方式。

  • グループテクノロジー：似た部品をグループ化して、少量生産でも大量生産のような効率を出す考え方
  • U字ライン方式：工程をU字に並べて、1人の作業者がぐるぐる回る。待ち時間がほぼゼロ
  • 1人生産方式：1人で最初から最後まで全部やる。仕掛品（作りかけの製品）がゼロ

メリット

  • バランスロスが少ない
  • 仕掛品を減らせる
  • やりがいを感じやすい

デメリット

  • 多能工が必要（育成に時間かかる）
  • 作業者の能力差が生産量に直結

モジュール生産方式

部品をあらかじめ小さなまとまり（モジュール＝ユニット）として組み立てておいて、最終組立ではモジュール同士をくっつけるだけにする方式。自動車産業でよく使われます。

QCサークル

現場で働く人たちが、自分たちで小さなグループを作り、品質を良くするためのアイデアを出し合う活動。日本のものづくりが強い理由の1つ。

ライン生産 vs セル生産は超頻出テーマ！「大量に同じものを作るならライン」「多品種少量で柔軟にやるならセル」と覚えてね。試験では「この会社はどっちが向いてる？」って聞かれることが多いよ！

⚖ 重要な数式（全17公式）

ここからは計算問題対策！公式は丸暗記じゃなくて「何を測ってるのか」をイメージしながら覚えると忘れにくいよ。

1. 生産性

生産性 = 産出量（Output）/ 投入量（Input）

産出量には生産量・生産金額・付加価値などが使われ、投入量には労働量・投入資本・設備・材料などが使われます。

例：100人で1000個作れる工場A（生産性=10）vs 50人で800個の工場B（生産性=16）
→ 人数が少ないBの方が「生産性が高い」

2. 労働生産性

労働生産性 = 付加価値額 / 従業員数

生産性の中でも頻出の指標。従業員1人あたりが生み出す付加価値を表します。財務・会計の科目とも横断的に出題されます。

例：付加価値額5000万円、従業員10人 → 労働生産性 = 500万円/人

3. 歩留り（ぶどまり）

歩留り = 産出された品物の量 / 投入された主原材料の量 × 100（%）

材料をどれだけムダなく使えたかの指標。不良品が出ると歩留りは下がる。手直しして良品にすれば計算上は向上するが、手直しにはコストがかかるため、根本的には不良品を出さないことが重要。

例：鉄100kgを投入、製品85kg → 歩留り85%（残り15kgはスクラップ）

4. 稼働率

稼働率 = 有効稼働時間 / 利用可能時間

有効稼働時間＝生産に直接役立っている時間。利用可能時間＝就業時間（機械が使える時間全体）。段取り替え・故障・待ち時間で有効稼働時間は減る。

例：利用可能時間100時間、そのうち実際に生産した時間30時間 → 稼働率 = 30%

5. 生産リードタイム

生産リードタイム = 生産の着手時期から完了時期に至るまでの時間

原材料の状態から製品として出荷できる状態になるまでの全体時間。複数工程の場合、工程間の滞留時間も含むのがポイント。加工時間だけではない。

ここまでの5つはどれも「良いもの / 全体」の形で、数値が高いほど良い。この共通点を覚えておくと混乱しにくいよ！

6. サイクルタイム（ピッチタイム）

サイクルタイム = 最も作業時間が長い工程の時間

ライン全体で製品を生産する速度を決める時間。一番遅い工程（＝ボトルネック）がライン全体のペースを決めます。合計でも平均でもないことに注意。

例：工程A=3分, B=5分, C=4分 → サイクルタイム = 5分（B工程が最大）

7. ライン編成効率（ラインバランス効率）

ライン編成効率 = 作業時間の合計 / （サイクルタイム × 作業ステーション数）

作業時間がどれだけ均等に配分されているか。100%＝完璧にバランスが取れた状態。

例：A=3分, B=5分, C=4分（3工程）
サイクルタイム=5分、ライン編成効率 = (3+5+4)/(5×3) = 12/15 = 80%

8. バランスロス率

バランスロス率 = 100% - ライン編成効率

0%が理想（ムダが全くない）。バランスロスが大きい＝手待ちや仕掛品停滞が多い。

例：上の例ではバランスロス率 = 100% - 80% = 20%
山崩しで全工程4分にすると → 12/(4×3) = 100%、ロス率 = 0%
全8公式まとめ：生産性・労働生産性・歩留り・稼働率・リードタイム → 高いほど良い。サイクルタイム・バランスロス率 → 低いほど良い。ライン編成効率 → 高い（100%に近い）ほど良い。この区別が大事！

9. 設備総合効率（3-6 設備管理）

設備総合効率 = 時間稼働率 × 性能稼働率 × 良品率

設備のロスを3つに分けて測る指標。3つを掛け算するから、どれか1つでも低いと全体がガクッと下がります。

指標何を測る？ロスの原因
時間稼働率停止ロス故障・段取り替え
性能稼働率性能ロス空転・チョコ停・速度低下
良品率不良ロス不良品・手直し
「チョコ停」ってかわいい名前だけど、設備が一瞬だけ止まること。センサーが反応して止まったり、材料が詰まったり。1回は短くても積み重なると大きなロスになるよ！

10. MTBF（平均故障間隔）（3-6 設備管理）

MTBF = 稼働時間の合計 / 故障回数

故障してから次に故障するまでの平均時間。長いほど壊れにくい＝信頼性が高い設備。

例：稼働400時間→故障→稼働600時間→故障→稼働500時間→故障
MTBF = (400+600+500)/3 = 500時間

11. MTTR（平均修復時間）（3-6 設備管理）

MTTR = 修理時間の合計 / 修理回数

故障した設備を直すのにかかる平均時間。短いほど修理が早い＝保全性が高い。

例：1回目の修理20時間、2回目18時間 → MTTR = (20+18)/2 = 19時間

12. 可用率（アベイラビリティ）（3-6 設備管理）

可用率 = MTBF / （MTBF + MTTR）

設備が「使いたいときに使える確率」。MTBFが長く、MTTRが短いほど可用率は高い。

例：MTBF=500時間, MTTR=19時間 → 可用率 = 500/(500+19) = 約96.3%
MTBF・MTTR・可用率は3点セットで覚えよう！MTBFは「壊れるまでの時間」、MTTRは「直すまでの時間」、可用率は「結局どのくらい使えるか」。経営情報システムの科目でも出るから一石二鳥！

13. 故障強度率（3-6 設備管理）

故障強度率 = （故障停止時間の合計 / 負荷時間の合計）× 100

故障のために設備が停止した割合。設備の信頼性を表す指標で、低いほど良い。

14. 標準時間（3-5 IE）

標準時間 = 正味時間 + 余裕時間
正味時間 = 観測時間 × レイティング係数

「習熟した作業者が、普通のペースで作業したときにかかる時間」のこと。正味時間（実際に作業する時間）に余裕時間（休憩・機械調整など）を足して求めます。

15. 余裕率と標準時間の計算（3-5 IE）

余裕率には外掛け法と内掛け法の2つがあります。試験ではどちらかを指定して出題されます。

外掛け法内掛け法
余裕率の式余裕時間 / 正味時間余裕時間 / 標準時間
標準時間の式正味時間 × (1 + 余裕率)正味時間 / (1 - 余裕率)
外掛け法と内掛け法、どっちが出ても対応できるようにしよう！外掛け＝正味時間が基準（分母）、内掛け＝標準時間が基準（分母）。「外は正味、内は標準」と覚えてね。

16. 度数率（労働安全衛生）

度数率 = （労働災害による死傷者数 / 延べ実労働時間数）× 1,000,000

100万延べ実労働時間あたりの労働災害の発生頻度。PQCDSMEのS（Safety）に直結する指標。

17. 強度率（労働安全衛生）

強度率 = （延べ労働損失日数 / 延べ実労働時間数）× 1,000

1,000延べ実労働時間あたりの労働損失日数。度数率が「頻度」、強度率が「重さ」と覚えましょう。

🏪 「今日（強）、ローソン（労損）で働いたら（労働）1,000円もらった」

強度率 ＝ 労働損失日数 ÷ 延べ実労働時間数 × 1,000

🎸 「ど（度）でかいフェスで死傷者が出た！ 観客100万人」

キヤノン ー ライン生産からセル生産への大転換
セル生産 ライン生産 多能工 ラインバランシング

キヤノンは2000年代に、プリンターやカメラの生産をライン生産からセル生産に大きく変えました。

なぜセル生産に変えたの？
製品の種類が増えて、ライン生産だと段取り替えが頻繁に必要になり、バランスロス（各工程の時間のばらつきによるムダ）も増えた。セル生産なら、1人〜少人数が全工程を担当するので、品種を柔軟に切り替えられます。
多能工の育成
セル生産には多能工が必要。キヤノンは徹底した教育プログラムで作業者を多能工に育て上げました。時間はかかりましたが、結果として仕掛品の削減、リードタイムの短縮、作業者のモチベーション向上を実現。
キヤノンの事例は「ライン vs セルの使い分け」を考える好例！「同じものを大量に→ライン」「多品種で柔軟に→セル」という判断基準を覚えてね。

試験対策ポイント

セル生産のメリット＝バランスロス少・仕掛品削減・モチベーション向上。デメリット＝多能工が前提・習熟に時間。U字ラインと1人生産方式がキーワード。

① 生産性 ＝ 産出量（Output） / 投入量（Input）
② 労働生産性 ＝ 付加価値額 / 従業員数
③ 歩留り ＝ 産出された品物の量 / 投入された主原材料の量 × 100（%）
Q13. 重要公式（稼働率・リードタイム）
① 稼働率 ＝ 有効稼働時間 / 利用可能時間
② 生産リードタイム ＝ 生産の着手時期から完了時期に至るまでの時間
※ 複数工程の場合、工程間の滞留時間も含む
Q14. ラインバランシング
① サイクルタイム ＝ 最も作業時間が長い工程の時間（＝ボトルネック）
② ライン編成効率 ＝ 作業時間の合計 / （サイクルタイム × 作業ステーション数）
③ バランスロス率 ＝ 100% − ライン編成効率（0%が理想）
例：A=3分, B=5分, C=4分 → サイクルタイム=5分、編成効率=80%、ロス率=20%

・ 繰返し受注生産 ＝ 仕様は決まっていて、注文が来たら同じものを作る
・ 違いのポイント：「設計が毎回必要か？」→ Yesなら個別、Noなら繰返し
Q16. セル生産のメリット・デメリット
メリット：バランスロスが少ない・仕掛品を減らせる・やりがいを感じやすい
デメリット：多能工が必要（育成に時間がかかる）・作業者の能力差が生産量に直結
Q17. PDSサイクル
Plan（計画） → Do（実行） → See（確認）
PDCAのCheck（確認）とAct（改善）をまとめて「See」にしたもの
D. 応用の範囲
Q18. 段取りの種類と効率順
効率が高い順：ゼロ段取り ＞ シングル段取り ＞ 外段取り ＞ 内段取り
・ ゼロ段取り ＝ 3分以内で限りなくゼロを目指す
・ シングル段取り ＝ 10分未満で行う
Q19. 段取り頻度（仕事の流し方別）
段取り頻度が高い順：個別生産 ＞ ロット生産 ＞ 連続生産
Q20. 設備総合効率
設備総合効率 ＝ 時間稼働率 × 性能稼働率 × 良品率
・ 時間稼働率 → ロスの原因は故障・段取り替え
・ 性能稼働率 → ロスの原因は空転・チョコ停・速度低下
・ 良品率 → ロスの原因は不良品・手直し
Q21. MTBF・MTTR・可用率
① MTBF（平均故障間隔）＝ 稼働時間の合計 / 故障回数　→ 長いほど良い
② MTTR（平均修復時間）＝ 修理時間の合計 / 修理回数　→ 短いほど良い
③ 可用率 ＝ MTBF / （MTBF + MTTR）
Q22. 標準時間と余裕率
標準時間 ＝ 正味時間 + 余裕時間
正味時間 ＝ 観測時間 × レイティング係数
外掛け法：標準時間 ＝ 正味時間 × （1 + 余裕率）
内掛け法：標準時間 ＝ 正味時間 / （1 − 余裕率）
覚え方：「外は正味（分母）、内は標準（分母）」
Q23. 度数率と強度率
度数率 ＝（労働災害による死傷者数 / 延べ実労働時間数）× 1,000,000（百万）
→ 災害の頻度を表す
強度率 ＝（延べ労働損失日数 / 延べ実労働時間数）× 1,000（千）
→ 災害の重さを表す
掛ける数字が大きい方（百万）が「度数率」、小さい方（千）が「強度率」
Q24. 公式の「高い/低いほど良い」判定
・ 生産性・労働生産性・歩留り・稼働率 → 高いほど良い
・ ライン編成効率・可用率・設備総合効率 → 高いほど良い
・ サイクルタイム・生産リードタイム・MTTR → 短い（低い）ほど良い
・ バランスロス率・故障強度率・度数率・強度率 → 低いほど良い
・ MTBF（平均故障間隔）→ 長いほど良い（壊れにくい＝信頼性が高い）
