需要予測とは

! 需要予測の概要

需要予測とは？

将来の製品やサービスに対する需要量を科学的に見積もること。生産計画・在庫管理・販売計画の基盤となる重要なプロセス。予測の精度が経営効率を大きく左右する。

需要予測の位置づけ
需要予測将来の需要を推定
➜
生産計画生産量・時期を決定
➜
在庫管理適正在庫の維持
➜
販売計画売上目標設定
需要予測は「生産計画のスタート地点」。予測がズレると過剰在庫や欠品が発生する。「当たるも八卦」ではなく「科学的な方法」で精度を上げていくのがポイント！
予測手法の分類

★ 定性的予測と定量的予測

需要予測の手法は大きく定性的予測法と定量的予測法の2つに分類されます。

ざっくり言うと、定性的＝「人の勘と経験（プロの直感）」、定量的＝「数字でゴリゴリ計算」。新製品で過去データがないときは勘に頼るしかないし、データがたまってきたら計算で攻める、という使い分け！
定性的予測 vs 定量的予測 ざっくり比較
定性的予測人の判断・経験データ少ないとき
↔
定量的予測数値データで計算データ豊富なとき

定性的予測法（質的予測法）

人の判断や経験に基づく予測。数値データが不足している場合や新製品の需要予測に使用。

  • デルファイ法：専門家に匿名でアンケートを繰り返し、意見を収束
  • 市場調査法：消費者・販売員へのアンケートやヒアリング
  • 類推法：類似製品の過去データから類推
  • 歴史的類推法：過去の類似ケースからの推定

定量的予測法（量的予測法）

過去の数値データに基づく予測。データが蓄積されている既存製品の需要予測に適する。

  • 時系列分析法：過去のデータの時間的推移パターンを分析
  • 因果関係分析法：需要と関連要因の因果関係を分析
デルファイ法は試験で超頻出！ポイントは「匿名」「繰り返し」「統計的処理」の3つ。専門家同士が直接議論すると権威に引きずられるから、匿名にするのがミソ。名前の由来は古代ギリシャのデルフォイの神託だよ！
時系列分析の基本

⚖ 時系列データの4つの変動要因

時系列データには以下の4つの変動要因が含まれています。

変動要因内容例
傾向変動（トレンド）長期的な上昇・下降の傾向人口増加による長期的な需要増
循環変動景気循環などによる数年周期の変動好景気・不景気の波
季節変動季節・月ごとの周期的変動夏のエアコン需要増
不規則変動予測不能な偶発的変動災害・突発的な事象
傾・循・季・不
「傾向」「循環」「季節」「不規則」＝ 「けい・じゅん・き・ふ」で覚えよう！
時系列分析は、これら4つの変動要因を分離・把握することが目的。移動平均法で季節変動や不規則変動を平滑化（ならす）して傾向変動を取り出すイメージだよ！

移動平均法

★ 移動平均法

移動平均法とは

過去の一定期間（N期間）のデータの平均値を求め、それを次期の予測値とする方法。データの変動を平滑化して傾向を把握する。

単純移動平均法の計算式
Ft+1 = (Dt + Dt-1 + ... + Dt-N+1) / N
F = 予測値、D = 実績値、N = 期間数

移動平均法のポイント

  • Nが大きい：変動が平滑化されるが、トレンドへの反応が遅い
  • Nが小さい：トレンドへの反応は速いが、変動の影響を受けやすい
  • 全てのデータに等しい重みが付く（最近のデータも古いデータも同じ扱い）
  • 上昇トレンドのときは実際より低め、下降トレンドのときは実際より高めの予測になる
移動平均法の弱点は「過去のデータを全部同じ重みで扱う」こと。直近の変化を重視したいなら、次に出てくる指数平滑法の出番！
指数平滑法

★ 指数平滑法（一次指数平滑法）

指数平滑法とは

直近のデータほど大きな重みを付けて予測する方法。平滑化定数（α）を使い、直前の予測値と実績値の誤差を修正して次の予測値を算出する。

指数平滑法の計算式
Ft+1 = Ft + α(Dt − Ft)
= α × Dt + (1 − α) × Ft
α = 平滑化定数（0 < α < 1）

αが大きい（1に近い）

  • 直近のデータを重視
  • 変化への反応が速い
  • 変動の影響を受けやすい

αが小さい（0に近い）

  • 過去のデータを重視
  • 変化への反応が遅い
  • 安定した予測値になる
指数平滑法の式は「前回の予測値 ＋ α × 前回の予測誤差」と覚えよう。予測が外れた分をαの割合で修正するイメージ。αの大小と特徴の対応は必ず出題される！
回帰分析

⚖ 因果関係分析法（回帰分析）

回帰分析とは

需要に影響を与える要因（説明変数）と需要量（目的変数）の関係を数式で表す方法。

単回帰分析の回帰式
Y = a + bX
Y = 需要量（目的変数）、X = 説明変数、a = 切片、b = 回帰係数

回帰分析のポイント

  • 単回帰分析：説明変数が1つ（例：広告費 → 売上）
  • 重回帰分析：説明変数が2つ以上（例：広告費＋価格＋気温 → 売上）
  • 相関係数（r）：-1～+1の範囲。|r|が1に近いほど関係が強い
  • 決定係数（r²）：説明変数で目的変数をどの程度説明できるかの指標
  • 時系列分析とは異なり、「なぜ需要が変動するか」の要因分析が可能
回帰分析は「XがこうなるとYはこうなる」という因果関係をモデル化するもの。移動平均法や指数平滑法が「過去の延長線」で予測するのに対し、回帰分析は「原因→結果」で予測する違いを押さえよう！
線形計画法（LP）

★ 線形計画法（リニアプログラミング）

線形計画法（LP: Linear Programming）とは

限られた経営資源（人・設備・材料・時間）のもとで、利益の最大化やコストの最小化を実現する最適な生産量の組み合わせを求める数理計画法。

線形計画法の3つの構成要素
目的関数最大化/最小化する式
+
制約条件資源の制限
+
非負条件x ≥ 0, y ≥ 0
目的関数の例（利益最大化）
Z = ax + by → 最大化
制約条件の例
c1x + d1y ≤ E1（資源1の制約）
c2x + d2y ≤ E2（資源2の制約）
x ≥ 0, y ≥ 0（非負条件）

⚖ 図解法（グラフ法）による解法

図解法の3ステップ
Step1制約条件を直線で描く
➜
Step2実行可能領域を特定
➜
Step3頂点で目的関数を計算

図解法の手順

  1. 制約条件をグラフ上に直線で描く
  2. 全ての制約条件を満たす領域（実行可能領域）を特定する
  3. 目的関数の値が最大（最小）となる点を実行可能領域の頂点から探す
  4. 最適解は必ず実行可能領域の頂点（端点）のいずれかにある

最適解の性質（超重要！）

  • 最適解は実行可能領域の端点（頂点）に存在する
  • 目的関数の傾きと制約条件の傾きが一致する場合、辺上の全ての点が最適解になる（多重最適解）
  • 実行可能領域が存在しない場合 → 解なし（実行不能）
  • 実行可能領域が無限に広がる場合 → 非有界（最適解が存在しないことも）
線形計画法の計算問題はほぼ毎年出題される超重要テーマ！図解法では「制約条件を線で引く → 実行可能領域を見つける → 頂点で目的関数を計算」の3ステップ。頂点さえ見つかれば、あとは代入するだけだよ！

⚙ 感度分析とシャドープライス

感度分析

最適解が得られた後、制約条件や目的関数の係数が変化した場合に最適解がどう変わるかを分析する手法。

  • 係数がどの範囲なら最適解が変わらないかを確認
  • 経営環境の変化に対するロバスト性を評価

シャドープライス（影の価格）

制約条件の右辺値（資源量）を1単位増加させたとき、目的関数の値がどれだけ改善するかを示す値。

  • 余裕のない制約（ボトルネック）のシャドープライスは正
  • 余裕のある制約のシャドープライスはゼロ
  • 投資判断（どの資源を増やすべきか）に活用
シャドープライスを身近に例えると、ラーメン屋の行列と同じ。席数がボトルネック（制約）なら、席を1つ増やせば売上が増える（シャドープライスが正）。でも麺の在庫が余っているなら、麺を増やしても意味がない（シャドープライスは0）！
シャドープライスは「もし機械をあと1時間使えたら、利益がいくら増えるか？」を示す値。ボトルネックの解消にいくら投資すべきかを判断するのに使う。余裕がある資源のシャドープライスは0（増やしても意味がない）！

  • LP3要素＝目的関数・制約条件・非負条件／最適解＝頂点

このタブで学ぶこと

1
専門家に匿名でアンケートを繰り返し行い、意見を収束させる需要予測手法はどれか。
  • A. 回帰分析法
  • B. デルファイ法
  • C. 指数平滑法
  • D. 移動平均法
正解：B. デルファイ法
デルファイ法は定性的予測法の代表格。「匿名性」「反復」「統計的処理」が特徴。専門家の意見を匿名で集め、結果をフィードバックして再度回答を求めることで意見を収束させる。
2
指数平滑法において、平滑化定数αを大きくした場合の特徴として最も適切なものはどれか。
  • A. 過去のデータを重視した安定的な予測になる
  • B. 季節変動が完全に除去される
  • C. 直近のデータを重視し、変化への反応が速くなる
  • D. 移動平均法と同じ結果になる
正解：C. 直近のデータを重視し、変化への反応が速くなる
αが大きい（1に近い）ほど直近の実績値に大きな重みが付く。逆にαが小さい（0に近い）と過去のデータを重視した安定的な予測になる。
3
線形計画法において、最適解が存在する場所として最も適切なものはどれか。
  • A. 実行可能領域の端点（頂点）
  • B. 実行可能領域の中心
  • C. 制約条件の交点すべて
  • D. 原点
正解：A. 実行可能領域の端点（頂点）
線形計画法の基本定理により、最適解は実行可能領域の端点（頂点）に存在する。そのため、全ての頂点で目的関数の値を計算し、最大（最小）となる頂点を見つければよい。
4
時系列データの変動要因のうち、景気循環などによる数年周期の変動はどれか。
  • A. 傾向変動
  • B. 循環変動
  • C. 季節変動
  • D. 不規則変動
正解：B. 循環変動
循環変動は景気循環など数年単位の周期的変動。傾向変動は長期的な上昇・下降トレンド、季節変動は1年以内の周期的変動、不規則変動は偶発的な変動。
5
ある製品の過去3期間の売上実績が100, 120, 110であった。3期間の単純移動平均法による次期の予測値はいくらか。
  • A. 100
  • B. 105
  • C. 110
  • D. 120
正解：C. 110
単純移動平均法：(100 + 120 + 110) / 3 = 330 / 3 = 110。過去3期間の平均値がそのまま次期の予測値となる。
6
シャドープライス（影の価格）の説明として最も適切なものはどれか。
  • A. 製品の市場価格から原価を引いた利益額
  • B. 制約条件の資源を1単位増加させたときの目的関数の改善額
  • C. 需要予測の誤差を金額に換算した値
  • D. 在庫保管コストの1単位あたりの金額
正解：B. 制約条件の資源を1単位増加させたときの目的関数の改善額
シャドープライスはボトルネックとなっている制約条件の資源量を1単位増やした場合に、利益（目的関数）がどれだけ改善するかを示す。余裕のある制約のシャドープライスは0。
7
指数平滑法で前期の予測値が200、前期の実績値が220、平滑化定数α=0.3のとき、今期の予測値はいくらか。
  • A. 200
  • B. 206
  • C. 210
  • D. 214
正解：B. 206
F = Ft + α(Dt - Ft) = 200 + 0.3 × (220 - 200) = 200 + 0.3 × 20 = 200 + 6 = 206。または、0.3 × 220 + 0.7 × 200 = 66 + 140 = 206。
8
製品Aと製品Bを生産する。利益は製品A：5万円/個、製品B：4万円/個。機械加工時間の制約が2A+B≤10、組立時間の制約がA+2B≤10、非負条件あり。利益を最大化する最適解はどれか。
  • A. A=5, B=0（利益25万円）
  • B. A=0, B=5（利益20万円）
  • C. A=10/3, B=10/3（利益30万円）
  • D. A=3, B=3（利益27万円）
正解：C. A=10/3, B=10/3（利益30万円）
制約条件の交点を求める。2A+B=10 と A+2B=10 を連立すると A=B=10/3。利益 Z=5×(10/3)+4×(10/3)=50/3+40/3=90/3=30万円。各頂点で計算：(5,0)→25万、(0,5)→20万、(10/3,10/3)→30万。よって(10/3, 10/3)が最適解。

計算例1

! 移動平均法の計算例

問題：月別売上データから3期移動平均を求める

月1月2月3月4月5月6月
売上（個）100120110130120140

解法

  • 4月の予測値 = (100+120+110)/3 = 110
  • 5月の予測値 = (120+110+130)/3 = 120
  • 6月の予測値 = (110+130+120)/3 = 120
  • 7月の予測値 = (130+120+140)/3 = 130

3期間の「窓」をスライドさせて平均を計算していくイメージ。

移動平均法は「窓をずらしていく」だけのシンプルな計算。試験では計算ミスに注意。足し算・割り算を丁寧に！ 回転寿司のレーンみたいに、古いデータが外れて新しいデータが入ってくるイメージだよ。
計算例2

★ 指数平滑法の計算例

問題：α=0.2で予測値を求める

1月の予測値 F1 = 100、1月の実績値 D1 = 110 のとき、2月の予測値は？

指数平滑法の計算フロー
前回予測F=100
+
α × 誤差0.2×(110-100)=2
=
今回予測F=102

解法

F2 = F1 + α(D1 - F1)

= 100 + 0.2 × (110 - 100)

= 100 + 0.2 × 10

= 100 + 2 = 102

さらに：2月の実績値 D2 = 108 のとき、3月の予測値は？

F3 = 102 + 0.2 × (108 - 102) = 102 + 1.2 = 103.2

指数平滑法では「予測が外れた分のα倍だけ修正する」と覚えれば簡単！予測値100で実績が110なら、予測誤差10のα（=0.2）倍＝2だけ上方修正するイメージ。
計算例3

★ 線形計画法の計算例

問題

ある工場で製品Xと製品Yを生産する。

  • 製品Xの利益：3万円/個、製品Yの利益：2万円/個
  • 機械Aの制約：1X + 1Y ≤ 6（合計6時間まで）
  • 機械Bの制約：2X + 1Y ≤ 8（合計8時間まで）
  • x ≥ 0, y ≥ 0

利益Z = 3x + 2y を最大化せよ。

Step 1：制約条件をグラフに描く

  • x + y = 6 → 切片(6,0)と(0,6)を結ぶ直線
  • 2x + y = 8 → 切片(4,0)と(0,8)を結ぶ直線
  • x ≥ 0, y ≥ 0 → 第1象限のみ

Step 2：頂点を求める

実行可能領域の頂点は以下の4つ：

  • (0, 0)：原点
  • (4, 0)：2x + y = 8 と x軸の交点
  • (2, 4)：x + y = 6 と 2x + y = 8 の交点
    連立方程式：2x+y=8 から x+y=6 を引くと x=2、y=4
  • (0, 6)：x + y = 6 と y軸の交点

Step 3：各頂点で目的関数Z = 3x + 2y を計算

頂点Z = 3x + 2y結果
(0, 0)3×0 + 2×00万円
(4, 0)3×4 + 2×012万円
(2, 4)3×2 + 2×414万円 ★最大
(0, 6)3×0 + 2×612万円

答え

最適解は x=2, y=4 のとき、最大利益Z = 14万円

製品Xを2個、製品Yを4個生産するのが利益最大化の最適な組み合わせ。

線形計画法の問題は「頂点を全部出して、目的関数に代入して比較」すればOK。連立方程式を正確に解くことがカギ。試験では2変数の問題がほとんどなので、図解法で十分対応できる！ 実はやってることは「宝探し」みたいなもの。宝（最適解）は必ず領域の角っこにあるから、角だけ調べればいいんだ！

まとめ：試験対策のポイント

  • 需要予測の分類：定性的（デルファイ法等）vs 定量的（時系列・因果）を区別
  • デルファイ法の3特徴：匿名性・反復・統計的処理
  • 時系列データの4変動要因：傾向・循環・季節・不規則
  • 移動平均法：Nの大小による特性の違い（大→安定だが反応遅い）
  • 指数平滑法の計算式とαの特性（大→直近重視・反応速い）
  • 回帰分析：単回帰 vs 重回帰、相関係数・決定係数
  • 線形計画法：目的関数・制約条件・非負条件の3要素
  • 図解法の手順：制約線を引く→実行可能領域→頂点で目的関数計算
  • シャドープライス：ボトルネック資源の1単位増加による利益改善額
  • 計算問題は毎年出題！移動平均・指数平滑・LPの計算を確実に
