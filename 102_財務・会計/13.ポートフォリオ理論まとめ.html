🔢 「加重平均」ってなに？

🧑
新人くん
「加重平均」って難しそう...普通の平均と何が違うの？
👨
先生
普通の平均は「全部同じ重さ」。加重平均は「重要度に差をつけて平均する」ってこと。学校のテストで例えてみよう！
普通の平均 vs 加重平均

普通の平均

国語 80点 + 数学 60点

(80 + 60) ÷ 2 = 70点

2科目を同じ重さで計算

加重平均

国語 80点(配点3) + 数学 60点(配点7)

80x0.3 + 60x0.7 = 66点

数学の方が「重い」ので結果が変わる！

🔤 ギリシャ文字はこわくない！

🧑

ギリシャ文字だよ。でもたった3つ覚えるだけ！しかもそれぞれ「あだ名」をつけちゃえば簡単。
覚える記号はたった3つ！
σ
シグマ = リスク（ブレ幅）
あだ名：「ブレブレくん」
ρ
ロー = 相関係数（連動度）
あだ名：「なかよし度」
β
ベータ = 市場との連動度
あだ名：「ゆれゆれ度」

🏫 日常に置き換えると...

記号日常の例え値が大きいと
σ（シグマ）テストの点数のバラつき毎回の点数がバラバラ
ρ（ロー）友達との趣味の一致度好みがまったく同じ
β（ベータ）ジェットコースターの激しさ激しく揺れる（ハイリスク）
⚠️
試験の注意
試験では記号が必ず出る。「σ（シグマ）= 標準偏差 = リスク」「ρ（ロー）= 相関係数」の対応を暗記しておくのが最低ライン！

  • ポートフォリオ = 資産の組み合わせ（おかずセット）
  • リスク = 危険ではなく「ブレ幅」
  • σ ρ β = たった3つの記号を覚えるだけ

この3つを下の語呂で一気に覚えよう！

📮 ポストの中に3匹の猫
玄関のポスト（= ポートフォリオ）を開けたら、中で
σ（シグマ猫）がブルブル震え（= ブレ幅 = リスク）、
ρ（ロー猫）が隣の猫とピッタリくっついて寝ていて（= 相関 = 連動度）、
β（ベータ猫）がジェットコースターの模型にしがみついていた（= 市場との連動度）。
— 変な光景だけど、3匹の名前と特徴はもう忘れない！

🎯 リターンとリスク

💰 リターン（期待収益率）

「平均でどれくらい儲かるか」
= 各シナリオの収益率 x 確率 の合計

例: 好景気50%で+20%、不景気50%で-4%
→ 0.5x20% + 0.5x(-4%) = 8%

VS

📉 リスク（標準偏差）

「リターンがどれくらいブレるか」
= ブレ幅が大きい ＝ リスクが高い

定期預金はブレ小 → 低リスク
株式はブレ大 → 高リスク

🎬 実例：映画の興行収入。大作映画は大ヒットor大コケ（高リスク）、ドキュメンタリーは安定した小収入（低リスク）。

💡
身近な例え
リターン = お弁当屋さんの「1日の平均売上」
リスク = 日によって売上がどれくらいバラつくか
毎日安定して10万円売れるお店は低リスク。5万〜20万円とバラつくお店は高リスク。
⚠️
試験の大原則
ハイリスク・ハイリターン：リスクが高い資産ほど、見返りとして高いリターンが期待される。逆にローリスクならローリターン。この関係は試験で大前提として問われるよ！
リスクとリターンの関係
🏦 定期預金低リスク・低リターン
→
📄 債券中リスク・中リターン
→
📈 株式高リスク・高リターン

キーワード意味

標準偏差リターンのブレ幅＝リスクの大きさ

試験のツボ：ポートフォリオ理論は「リスクを下げたいなら相関係数が低い資産を組み合わせる」がすべての基本。これだけ覚えれば、応用問題も解ける！

📊 期待収益率の計算

👨
先生
ポートフォリオの期待収益率は、各資産の期待収益率を投資比率で加重平均するだけ！シンプルだよ。
E(Rp) = w₁ x E(R₁) + w₂ x E(R₂)

w = 投資比率、E(R) = 各資産の期待収益率

📝 計算例

資産A（期待収益率10%）に60%、資産B（期待収益率4%）に40% 投資する場合：

E(Rp) = 0.6 x 10% + 0.4 x 4% = 6% + 1.6% = 7.6%

🍔 実例：ハンバーガー店で「肉60%・パン40%」のバーガーの総カロリーを計算するのと同じ。各素材のカロリーを配分比で加重平均するだけ！

📉 ポートフォリオのリスク（標準偏差）

🧑
受験生
リターンは加重平均でかんたんだけど、リスクは同じじゃないの？
👨
先生
ここがポートフォリオ理論の核心！リスクは単純な加重平均にはならない。相関係数によって分散効果が生まれるから、たいてい加重平均よりリスクは小さくなるんだ。
σp² = w₁²σ₁² + w₂²σ₂² + 2w₁w₂σ₁σ₂ρ₁₂

σ = 標準偏差、ρ = 相関係数

公式の3つのパーツを分解
第1項Aだけのリスク
+
第2項Bだけのリスク
+
第3項AとBの連動分
← ρで変わる！
💡
覚え方のコツ
料理に例えると、第1項=塩の辛さ、第2項=砂糖の甘さ、第3項=混ぜたときの味変。相関係数ρが「味変の強さ」を決める。ρ=-1なら塩と砂糖が打ち消し合ってマイルドに！
試験最頻出ポイント！
この公式の第3項「2w₁w₂σ₁σ₂ρ」がカギ。
• ρ = +1 の場合 → リスクは単純な加重平均（分散効果なし）
• ρ = -1 の場合 → リスクを最小（ゼロ）にできる可能性あり
• ρ が低いほど → 分散効果が大きい

🔗 相関係数をビジュアルで

相関係数と値動きのイメージ
相関係数値動き分散効果身近な例
ρ = +1まったく同じ方向なしトヨタ株とホンダ株
ρ = 0無関係あり自動車株と食品株
ρ = -1まったく逆方向最大傘屋とアイス屋

⚽ 実例：サッカーチームで「攻撃的FW」と「守備的MF」は逆の役割（相関-1に近い）。両方いればチームが安定する。FWばかり（相関+1）だと守備崩壊！

🛁 風呂場に相撲取りと菊の花
お相撲さん（= 相関）が低い天井（= 低い）の風呂場で頭をぶつけ、
驚いて干していた洗濯物をバラバラに散らかした（= 分散）ら、
なぜか浴室中に菊の花（= 効く）の香りが広がった。
— 「相撲が低い天井で散らかして菊の香り」= 相関が低いほど分散が効く！

🎯 効率的フロンティアとCAPM

📈 効率的フロンティア

同じリスクの中で最大のリターンが得られる組み合わせの集合。グラフ上では左上の曲線になる。

= 「これ以上リスクを減らすにはリターンを諦めるしかない」という最適ライン

💡
身近な例え
回転寿司の「満足度（リターン）」と「お会計（リスク）」のバランス。効率的フロンティア＝同じ予算で最も満足できるメニューの組み合わせ。それより上はお金が足りない、下は損してる。

💱 CAPM（資本資産価格モデル）

E(Ri) = Rf + βi x (E(Rm) - Rf)

Rf = リスクフリーレート（安全資産の利回り。例：国債の利回り0.5〜2%程度）
β = ベータ値（市場全体に対する感応度）
E(Rm) - Rf = マーケット・リスクプレミアム（市場全体のリターン - 安全資産）

⚠️
試験の引っかけ
CAPM計算はRf + β x (Rm - Rf) の代入だけ！よくある引っかけは「β x Rm」と間違えること。Rfを引いてからかけるのを忘れずに！
💡
CAPMの身近な例え
遊園地のジェットコースター（= 市場全体）に比べて、β=1.5の株は「1.5倍激しく揺れる乗り物」。揺れが激しい分、スリルの報酬（= リターン）も大きい。β=0.5なら半分しか揺れないメリーゴーラウンド。安定だけどリターンも控えめ。

🛡 システマティック・リスク

市場全体のリスク。分散投資では消せない。

例：景気後退、金利変動、戦争

🎲 アンシステマティック・リスク

個別企業のリスク。分散投資で消せる。

例：不祥事、特定製品の失敗

🏫 実例：学校のテスト。全教科まとめて中止（台風=システマティック）vs 数学だけ追試（自分の体調不良=アンシステマティック）。他の教科を頑張れば補える！

リスクの分類ツリー
📊 投資の総リスク
⬇
システマティック✖ 分散で消せない
⬇
アンシステマティック○ 分散で消せる
💡
身近な例え
学校で例えると、システマティック=台風で学校ごと休校（全員に影響、避けられない）。アンシステマティック=自分だけ風邪で欠席（個人の問題、予防接種=分散で対策可能）。
試験頻出ポイント
• 「分散投資で除去できるのはアンシステマティック・リスクのみ」
• CAPMで評価されるのはシステマティック・リスク（β）だけ
• β=1 は市場平均と同じ動き

💬 相談2：新事業への投資判断

🏪 小売業の鈴木社長の悩み

🧑
鈴木社長
今は実店舗だけだけど、EC事業を始めようか迷っている。でもリスクが心配...
👨
診断士
実はポートフォリオ理論の考え方は、事業ポートフォリオにも応用できます。実店舗とEC事業の売上は「天候」や「立地」で相関が低い可能性が高いです。

📊 分析してみると...

シナリオ確率実店舗のみEC追加後
好景気30%+15%+12%
普通40%+5%+6%
不景気30%-8%+1%

実店舗のみの期待収益率 = 0.3x15 + 0.4x5 + 0.3x(-8) = 4.1%（ブレ大）
EC追加後の期待収益率 = 0.3x12 + 0.4x6 + 0.3x1 = 6.3%（ブレ小）

🛒 実例：ECサイトの「実店舗受取」と「宅配」の2チャネル。大雪で来店客が減っても、宅配注文は増える。チャネルの分散=事業リスクの分散！

👨
診断士
EC事業を加えると、不景気でもプラスを維持。リターンが上がりつつリスクが下がる = まさにポートフォリオ理論の効果です！これが「事業の分散」の力。
⚠️
試験の引っかけ
「期待収益率が高い方を選べばよい」はバツ！リスク（ブレ幅）も必ず確認する。リターンだけ見て判断するのは、料理の味だけ見て値段を無視するようなもの。

💬 相談3：CAPMで投資判断

🏭 製造業・山田社長の悩み

🧑
山田社長
設備投資の案が2つある。どっちを選べばいい？ハイリスク・ハイリターンの新製品ラインか、安定した既存ライン増設か。
👨
診断士
CAPMの考え方を応用しましょう。リスクに見合うリターンが得られるかが判断基準です。

📝 CAPM での比較

リスクフリーレート(Rf) = 1%、マーケットリスクプレミアム = 5%

投資案β要求収益率予想収益率判定
新製品ライン1.81%+1.8x5%=10%14%○ 投資価値あり
既存ライン増設0.61%+0.6x5%=4%3%✖ 不十分

🏃 実例：スポーツジムの入会判断。月会費（=要求収益率）に見合う健康効果（=予想収益率）があるか？効果が会費以上なら入会すべき！

予想収益率 ＞ 要求収益率 なら投資すべき。新製品ラインは14% > 10%でリスクに見合うリターンあり！

💡
覚え方
要求収益率＝「このリスクなら最低これだけほしい」というハードル。予想収益率がハードルを超えれば"お買い得"、下回れば"割高"。ネット通販で「送料込みでも安いか？」と判断するのと同じ感覚！

基本
資産A（期待収益率8%）に70%、資産B（期待収益率2%）に30%投資した場合、ポートフォリオ全体の期待収益率は？
  • 6.2%
  • 5.0%
  • 7.0%
  • 4.6%
✅ 正解：6.2%
0.7 x 8% + 0.3 x 2% = 5.6% + 0.6% = 6.2%。ポートフォリオの期待収益率は各資産の加重平均！

相関係数
海水浴場の売店（夏に儲かる）とスキー場のレストラン（冬に儲かる）。この2つの事業の相関係数に最も近いのは？
  • +1
  • +0.5
  • 0
  • -1に近い
✅ 正解：-1に近い
夏に儲かる事業と冬に儲かる事業は逆の値動き。まさに傘屋とアイス屋の関係と同じで、相関係数は-1に近い。組み合わせると年間通して安定する！

分散投資
分散投資によって除去できるリスクはどれ？
  • システマティック・リスク
  • アンシステマティック・リスク
  • マーケット・リスク
  • すべてのリスク
✅ 正解：アンシステマティック・リスク
個別企業のリスク（不祥事、製品の失敗など）は分散で除去可能。一方、市場全体のリスク（景気後退など）= システマティック・リスクは分散しても消せない！
5
CAPM
あなたの会社のベータ値は1.2。リスクフリーレートが2%、マーケットリスクプレミアムが5%のとき、CAPMによる要求収益率は？
  • 8.0%
  • 7.0%
  • 8.5%
  • 6.0%
✅ 正解：8.0%
E(Ri) = Rf + β x (Rm - Rf) = 2% + 1.2 x 5% = 2% + 6% = 8.0%。ベータが1より大きいので、市場平均より高いリターンが要求される！
6
効率的フロンティア
効率的フロンティア上のポートフォリオの特徴として正しいものは？
  • リスクがゼロのポートフォリオである
  • リターンが最大のポートフォリオである
  • すべての投資家が選ぶべきポートフォリオである
  • 同じリスクの中でリターンが最大の組み合わせである
✅ 正解：同じリスクの中でリターンが最大の組み合わせである
効率的フロンティアは「同じリスク水準で最もリターンが高い」（= 同じリターンで最もリスクが低い）ポートフォリオの集合。リスクゼロやリターン最大とは限らない。
7
応用
地元のパン屋さんが、カフェ併設とネット通販の2つの新事業を検討中。ポートフォリオ理論の観点から最もよいアドバイスは？
  • リターンが最も高い事業だけに集中投資すべき
  • 両方やるとコストが倍になるのでどちらか一方にすべき
  • 売上の相関が低い事業を組み合わせるとリスクが下がる
  • 新事業はリスクが高いのでどちらもやめるべき
✅ 正解：売上の相関が低い事業を組み合わせるとリスクが下がる
カフェ（立地・天候に依存）とネット通販（立地に依存しない）は相関が低い可能性が高い。ポートフォリオ理論では、相関の低い資産を組み合わせることでリスクを低減できる！
8
CAPM
β（ベータ値）が1.0の株式について正しいものは？
  • 市場全体と同じ動きをする
  • 市場より2倍激しく動く
  • リスクフリーレートと同じリターンになる
  • 分散投資の効果がない
✅ 正解：市場全体と同じ動きをする
β=1.0は「市場平均と同じ感応度」。市場が10%上がればこの株も約10%上がり、10%下がれば約10%下がる。β>1なら市場より激しく、β<1なら市場より穏やか。

🏦 GPIF（年金積立金管理運用独立行政法人）

🌏 世界最大級の機関投資家による分散投資

私たちの年金を運用しているGPIF。運用資産は約200兆円超（2024年時点）。世界最大級の機関投資家です。

📊 GPIFの基本ポートフォリオ（2020年〜）

資産クラス配分比率特徴
国内債券25%低リスク・低リターン
外国債券25%為替リスクあり・やや高リターン
国内株式25%中リスク・中リターン
外国株式25%高リスク・高リターン
💡
ポートフォリオ理論との関連
4つの資産クラスに均等に分散 = 相関の低い資産を組み合わせてリスクを抑制。国内・海外、債券・株式で値動きが異なるからこそ分散効果が働く。まさにポートフォリオ理論の教科書的な実践例！

GPIFは2001年の運用開始以来、年平均約3.9%（2024年3月末時点）のリターンを達成。「分散投資は長期で力を発揮する」ことを証明しています。

👨
先生
定期預金の利率が0.01%だった時代と比べると、年3.9%は約390倍の差。分散投資の威力がよくわかるね。コンビニで例えると、おにぎりだけ売る店 vs おにぎり・飲料・デザート・日用品を売る店の安定感の違い！

📱 ソフトバンクグループのビジョン・ファンド

🚀 集中投資のリスクを体現した事例

孫正義氏率いるソフトバンクグループは、2017年に約10兆円規模のビジョン・ファンドを設立。AI・テクノロジー企業に集中投資しました。

✅ 成功例

  • Armの上場で巨額の含み益
  • Coupang（韓国EC大手）の成長

❌ 失敗例

  • WeWork：上場延期→経営危機で大損
  • 複数のスタートアップが破綻
⚠️
ポートフォリオ理論の教訓
テクノロジー企業ばかりに投資 = 相関の高い資産に集中。業界全体が不調になると一気に損失が膨らむ。アンシステマティック・リスクは分散できても、テクノロジーセクター全体のリスク（システマティック・リスクに近い）は残るという好例。
💡
わかりやすく言うと
学校で例えると「友達全員サッカー部」なら、雨が降ったら全員練習できない。サッカー部・吹奏楽部・将棋部に分かれていれば、雨でも誰かは活動できる。これが業種分散の考え方！
集中投資 vs 分散投資
集中投資同じ業界ばかり
✖ 共倒れリスク大

🏪 イオンの事業ポートフォリオ

🛒 小売業の分散戦略

イオンは総合スーパー（GMS）だけでなく、金融・不動産・ヘルス&ウェルネスなど多角的に事業展開しています。

📊 イオンの事業セグメント

セグメント内容景気感応度
GMS（総合スーパー）食品・衣料・日用品低（生活必需品）
ディベロッパーイオンモール運営中
金融イオン銀行・イオンカード中
ヘルス&ウェルネスドラッグストア（ウエルシア）低

🏫 実例：学校の部活動が「運動部だけ」vs「運動部+文化部+帰宅部」。雨天で運動部が活動停止しても、文化部は平常運転。事業セグメントの分散と同じ！

💡
ポートフォリオ理論で読み解く
食品スーパーは不景気でも安定（低β）。金融事業は景気に連動しやすい（高β）。βの異なる事業を組み合わせることで、グループ全体のリスクを抑制。中小企業でも「本業＋副業」で同じ発想が使える！

👨‍🍳 中小企業の身近な成功事例

🍣 地方の寿司チェーン「くら寿司」の多角化

回転寿司チェーンのくら寿司は、国内だけでなく海外（アメリカ・台湾）にも積極展開。国内の少子高齢化リスクを、成長する海外市場で分散しています。

👨
診断士の視点
国内市場と海外市場の景気は完全には連動しない（相関が低い）。為替リスクは増えるものの、売上全体のブレ幅（リスク）を抑える効果がある。まさに「地理的分散」というポートフォリオ戦略！
🧑
受験生
「為替リスク」ってどういうこと？
👨
先生
海外で稼いだドルを円に換えるとき、円高になると目減りするリスクのこと。1ドル=150円で稼いだのに、換金時に1ドル=130円だと20円分損する。旅行で「帰国したらレートが変わってた」と同じ感覚！
