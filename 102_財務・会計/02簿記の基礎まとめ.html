簿記の基礎まとめ

中小企業診断士 財務・会計｜友人に教えるつもりで、ざっくり丁寧に

📒 1-1. 簿記って何？ Rank A

簿記とは、ざっくり言うと「会社のお金に関する出来事を、一定のルールに従って帳簿に記録する手続き」のことです。

家庭での家計簿のように、企業でもお金の動きを記録します。ただし企業の場合は、投資家や銀行に報告する必要があるので、みんな共通のルール（＝複式簿記）で記録するんです。

家計簿は「今月いくら使ったかな？」を知るためにつけますよね。簿記は企業版の家計簿。ただし、「いくら持っているか（財産）」と「いくら儲けたか（成績）」の両方を記録するのがポイントです。

簿記の3つの目的

  1. 帳簿に記録する ― 取引を正確に記録しておく
  2. 外部に報告する ― 財務諸表を作って投資家・債権者に報告
  3. 経営を管理する ― 経営状況を確認し、意思決定に活用
試験で聞かれるポイント
  • 「簿記上の取引」は5要素（資産・負債・純資産・収益・費用）が増減する場合のみ
  • 契約書にサインしただけでは簿記上の取引にならない
  • 火災で商品が焼失 → 資産が減少するので簿記上の取引になる
🧩 1-2. 簿記の5つの要素 Rank A

簿記では、すべての取引を5つの要素に分類します。これが簿記の土台です！

💰 資産 持っているもの
📝 負債 借りているもの
🏦 純資産 自分の持ち分
📈 収益 稼いだお金
📉 費用 使ったお金

B/S（貸借対照表）の等式

資産 ＝ 負債 ＋ 純資産

P/L（損益計算書）の等式

収益 − 費用 ＝ 利益
B/Sには 資産・負債・純資産、P/Lには 収益・費用 が載ります。B/Sは「ある時点の財産」、P/Lは「一定期間の成績」を表します。
⚖️ 1-3. 複式簿記の原則（借方・貸方） Rank A

複式簿記では、1つの取引を必ず2つの側面から記録します。左側を借方（かりかた）、右側を貸方（かしかた）と呼びます。

借方（左）
資産の増加
負債の減少
純資産の減少
費用の発生
収益の減少
貸方（右）
資産の減少
負債の増加
純資産の増加
収益の発生
費用の減少
「かりかたはひだり」の「か」は左に払う！
「か」の書き始めが左方向 → 借方＝左。「し」（かしかた）は右に払う → 貸方＝右。
超重要ルール
  • 借方と貸方の金額は必ず一致する（貸借平均の原則）
  • 資産・費用は左側（B/S・P/Lの左）の要素
  • 負債・純資産・収益は右側の要素
  • 左の要素が増加 → 借方（左）に記入
  • 右の要素が増加 → 貸方（右）に記入

仕訳の例：現金100を借り入れた

借方科目金額貸方科目金額
現金100借入金100

→ 現金（資産）が増加 → 借方。借入金（負債）が増加 → 貸方。

🔄 1-5. 簿記一巡の流れ Rank A

取引の発生から財務諸表の完成までの一連の流れを簿記一巡と呼びます。

STEP 1取引発生
→
STEP 2仕訳帳
→
STEP 3総勘定元帳
→
STEP 4試算表
→
STEP 5決算整理
→
STEP 6財務諸表

期中の取引

  • 仕訳帳：取引が発生したら最初に仕訳を記入。時系列に全取引を記録
  • 総勘定元帳：勘定科目別に仕訳を集約。T字型の勘定で管理
仕訳帳と総勘定元帳を合わせて「主要簿」と呼びます。これを補うのが「補助簿」（仕入先元帳、得意先元帳など）。

決算手続

  1. 決算整理前試算表の作成
  2. 決算整理仕訳（経過勘定、棚卸資産、貸倒引当金、減価償却 etc.）
  3. 決算整理後試算表の作成
  4. 帳簿の締切り
  5. 貸借対照表・損益計算書の作成

商品売買は三分法で記録します。「仕入」「売上」「繰越商品」の3つの勘定科目を使います。

現金取引

仕入時：商品100円を現金で仕入れ

借方金額貸方金額
仕入100現金100

売上時：商品150円を現金で販売

借方金額貸方金額
現金150売上150

掛取引

実際の企業では掛取引（後払い）が主流です。

取引借方貸方ポイント
掛仕入仕入 100買掛金 100買掛金＝後で払う義務（負債）
掛売上売掛金 150売上 150売掛金＝後で受け取る権利（資産）
買掛金の支払い買掛金 100現金 100負債の減少→借方
売掛金の回収現金 150売掛金 150資産の振替

返品・値引き

返品も値引きも、仕入時・売上時の逆仕訳を行います。

📊 1-7. 決算手続き Rank A

期末に決算整理仕訳を行い、正しい損益と財政状態を表すように修正します。

① 経過勘定の処理

名前意味B/S表示仕訳例
前払費用先に払った来期分の費用流動資産前払保険料 / 支払保険料
未収収益まだもらってない当期分の収益流動資産未収利息 / 受取利息
前受収益先にもらった来期分の収益流動負債受取利息 / 前受利息
未払費用まだ払ってない当期分の費用流動負債支払利息 / 未払利息
「くりのべ」と「みこし」
費用の繰延（前払費用） ＆ 収益の繰延（前受収益）
費用の見越（未払費用） ＆ 収益の見越（未収収益）

② 減価償却

固定資産の価値減少を費用として配分します。

方法計算式特徴
定額法（取得原価 − 残存価額）÷ 耐用年数毎年同額
定率法（取得原価 − 減価償却累計額）× 償却率初期に多く、だんだん減少

記帳方法：

  • 直接法：減価償却費 / 建物（資産から直接控除）
  • 間接法：減価償却費 / 減価償却累計額（累計額で間接控除）

③ 貸倒引当金

売掛金などの債権が回収不能になるリスクに備えます。

  • 差額補充法：見積額と残高の差額を追加。貸倒引当金繰入額 / 貸倒引当金
  • 洗替法：いったん全額戻入れしてから再設定

④ 売上原価の算定

三分法で記録した仕入勘定を使い、「しくりくりし」で売上原価を計算します（1-6参照）。

📄 1-8. 有価証券 Rank B

有価証券は保有目的によって4つに分類され、期末評価が異なります。

分類目的期末評価評価差額の処理
売買目的有価証券値上がり益を狙う時価評価する当期の損益（P/L）
満期保有目的の債券満期まで保有時価評価しない−

取得原価の計算
  • 有価証券の取得原価 ＝ 購入代価 ＋ 付随費用（手数料）
🏗️ 1-9. 固定資産・減価償却 Rank B

有形固定資産（建物・備品など）は長期間使用するため、減価償却で費用を各期に配分します。

取得原価

有形固定資産の取得原価 ＝ 購入代価 ＋ 付随費用

直接法 vs 間接法

方法仕訳B/S表示
直接法減価償却費 100 / 建物 100帳簿価額のみ表示
間接法減価償却費 100 / 減価償却累計額 100取得原価を表示し、累計額をマイナス表示

減損会計

固定資産の収益性が大きく低下し、投資額を回収できないと見込まれる場合に帳簿価額を減額します。

減損損失 ＝ 固定資産の帳簿価額 − 回収可能価額
減損損失は原則として特別損失に計上。戻入れは認められていません。
🛡️ 1-10. 引当金 Rank B

将来の支出や損失に備えて、当期の費用として計上するのが引当金です。

引当金の計上4要件

  1. 将来の特定の費用・損失であること
  2. その発生が当期以前の事象に起因すること
  3. 発生の可能性が高いこと
  4. 金額を合理的に見積もれること

分類体系

大分類小分類例B/S表示
評価性引当金−貸倒引当金資産のマイナス
負債性引当金債務性賞与引当金、退職給付引当金、商品保証引当金負債の部
非債務性修繕引当金、債務保証損失引当金負債の部
🤝 1-11. のれん Rank B

のれんとは、M&A（企業買収）のときに発生する差額です。

  • 正ののれん：買収価格 ＞ 純資産の時価 → 無形固定資産に計上。20年以内に均等償却
  • 負ののれん：買収価格 ＜ 純資産の時価 → 発生時に特別利益として一括計上
お店を買ったとき、「建物や在庫の価値」より高い値段を払った分が「のれん」。ブランド力や顧客基盤など、目に見えない価値に対するプレミアムです。

📖 田中さんの文房具屋 ― 開業から初めての決算まで

田中さんは脱サラして、念願の文房具屋「ペンギン堂」を開業することにしました。ここから田中さんと一緒に、簿記の基本を体験しましょう！

全8問の4択クイズです。間違えても大丈夫！解説を読んで学べます。

Q1Chapter 1: 開業準備 ― 資本金の出資

田中さんは自己資金500万円を元手に、ペンギン堂を開業しました。当座預金口座を開設し、500万円を預け入れました。

この取引の仕訳として正しいものは？
正解：B
当座預金（資産）が増加 → 借方。資本金（純資産）が増加 → 貸方。
自分のお金を出資したので「資本金」。銀行から借りたわけではないので「借入金」ではありません。
Q2Chapter 2: 銀行借入 ― お金を借りる

開業資金が足りないため、田中さんは銀行から300万円を借り入れ、当座預金に振り込まれました。

この取引の仕訳として正しいものは？
正解：C
当座預金（資産）が増加 → 借方。借入金（負債）が増加 → 貸方。
銀行から借りたお金は「資本金」ではなく「借入金」です。返済義務がある＝負債！
Q3Chapter 3: 商品仕入 ― 三分法・掛仕入

田中さんは文房具メーカーから商品200万円分を仕入れました。代金は翌月払い（掛取引）です。

三分法での仕訳として正しいものは？
正解：B
三分法では商品を仕入れたら「仕入」勘定（費用）を使います。「商品」勘定は使いません！
掛取引なので相手科目は「買掛金」（負債）。「売掛金」は販売側の勘定です。
Q4Chapter 4: 初めての売上 ― 掛売上

開店初日！近所の会社に文房具セットを350万円で販売しました。代金は翌月回収予定です。

この掛売上の仕訳として正しいものは？
正解：C
売掛金（資産）が増加 → 借方。売上（収益）が発生 → 貸方。
現金をもらっていないので「現金」ではなく「売掛金」。収益が増加したら右側（貸方）に書きます！
Q5Chapter 5: 返品発生 ― 仕入返品の仕訳

仕入れた商品のうち、品違いのものが30万円分見つかりました。田中さんはメーカーに返品しました。

仕入返品の仕訳として正しいものは？
正解：B
返品は仕入の逆仕訳！ 元の仕入仕訳が「仕入/買掛金」だったので、その逆の「買掛金/仕入」になります。
これにより、仕入（費用）が減少し、買掛金（負債）も減少します。
Q6Chapter 6: 備品購入 ― 減価償却の計算

田中さんはお店のレジ用パソコン（備品）を50万円で現金購入しました。耐用年数5年、残存価額ゼロ、定額法で減価償却します。

1年分の減価償却費はいくら？
正解：A）10万円
定額法：（取得原価 − 残存価額）÷ 耐用年数 ＝（50万 − 0）÷ 5年 ＝ 10万円/年
間接法の仕訳：減価償却費 10万 / 減価償却累計額 10万
Q7Chapter 7: 決算① ― 売上原価（しくりくりし）

決算日を迎えました！ 期首商品は0万円（新規開業のため）。当期仕入高は200万−30万（返品分）＝170万円。期末に在庫を数えたら40万円分の商品が残っていました。

売上原価はいくら？
正解：C）130万円
売上原価 ＝ 期首商品（0）＋ 当期仕入（170万）− 期末商品（40万）＝ 130万円
「しくりくりし」の仕訳：
① 仕入 0 / 繰越商品 0（期首分、今回は0）
② 繰越商品 40万 / 仕入 40万（期末分）
Q8Chapter 8: 決算② ― 貸倒引当金（差額補充法）

決算時の売掛金残高は350万円。貸倒引当金の残高は5万円。田中さんは来期の貸倒率を3%と見積もりました。

差額補充法での仕訳として正しいものは？
正解：B）5.5万円
貸倒見積額 ＝ 350万 × 3% ＝ 10.5万円
差額補充法なので：10.5万 − 残高5万 ＝ 5.5万円を追加計上。
貸倒引当金繰入額 5.5万 / 貸倒引当金 5.5万

📋 3-6. 減価償却の計算例

毎期の償却費 ＝（200 − 20）÷ 5 ＝ 36万円/年

📋 3-7. のれん・負ののれん詳細

正ののれん（買収価格 ＞ 純資産時価）

例：純資産時価800万の会社を1,000万で買収

借方金額貸方金額
諸資産800万諸負債0
のれん200万現金1,000万
  • のれんは無形固定資産に計上
  • 20年以内に均等償却（毎期 のれん償却 / のれん）

負ののれん（買収価格 ＜ 純資産時価）

例：純資産時価800万の会社を600万で買収

借方金額貸方金額
諸資産800万諸負債0
現金600万
負ののれん発生益200万
  • 負ののれんは発生した事業年度の特別利益として一括計上
  • 償却は不要（一括処理）
正ののれん vs 負ののれん 比較
  • 正ののれん → 資産計上 → 20年以内に償却
  • 負ののれん → 特別利益で一括計上（償却なし）
