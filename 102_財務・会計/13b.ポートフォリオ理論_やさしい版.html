📖 第1話：麦田さんの悩み — 「ポートフォリオって何？」
🧑‍🍳
麦田店長
パン屋を10年やって、やっと貯金が500万円できた。でも全額を銀行に置いておくのはもったいない気がする...どうしたらいいの？
👨‍🏫
先生
いい質問！500万円をいくつかに分けて持つことを考えよう。これを「ポートフォリオ」と言うよ。

📖 ポートフォリオ = 組み合わせ

パン屋さんに例えると「食パンだけ」のお店は不安。「食パン + クロワッサン + サンドイッチ」と品揃えを組み合わせる方が安心。投資も同じ！

📖 第2話：もうけとブレ幅 — 「リターンとリスク」
🧑‍🍳
麦田店長
でも、何に投資すればいいの？銀行預金？株？
👨‍🏫
先生
まず2つの言葉を覚えよう。リターン（どれくらい儲かるか）とリスク（どれくらいブレるか）。

⚠️
ここ大事！
リスク = 「危険」じゃない！リスク = 「ブレ幅」。儲かる方にブレるのもリスク。試験では「リスクとは標準偏差である」と出るよ！

📖 第3話：混ぜると安定する — 「分散効果」
🧑‍🍳
麦田店長
株はブレが大きくて怖い...でも預金だけじゃ儲からない...
👨‍🏫
先生
だから混ぜるんだ！値動きが逆のものを組み合わせると、全体のブレが小さくなるよ。

逆に動くもの同士（= 相関が低い）を組み合わせると安定する

🔑 「相関係数」を一言で

2つの値動きがどれくらい連動するかを表す数字。-1〜+1の間。

数字意味パン屋で例えると
+1まったく同じ動き食パンとミニ食パン（同じ客層）
0関係なしパンと文房具
-1まったく逆の動きあんぱん（雨）とサンドイッチ（晴れ）

-1に近いほど、組み合わせたときに安定する（= 分散効果が大きい）

📖 第4話：麦田さんの決断 — 「500万円をどう分ける？」
🧑‍🍳
麦田店長
よし、500万円を分けてみよう！でも、平均でどれくらい儲かるか知りたいな。
👨‍🏫
先生
それが「期待収益率」。計算はかんたん。各投資先の儲け x 配分の割合を足すだけ！

📝 麦田さんの投資プラン

投資先配分期待収益率
預金50%（250万）年0.5%
株50%（250万）年6.0%

全体の期待収益率は？

0.5 x 0.5% + 0.5 x 6.0% = 0.25% + 3.0% = 3.25%

預金だけなら0.5%。でも半分を株にするだけで3.25%に！しかも全額株より安定。

💡
覚え方
期待収益率の計算は「配分 x 儲けを足すだけ」。カレーの材料配分と同じ。じゃがいも50%とにんじん50%を混ぜたら、味は平均になるよね！
📖 第5話：記号はたった3つ — 「σ ρ β」
🧑‍🍳
麦田店長
テキストに変な記号が出てきた...σとかρとか...
👨‍🏫
先生
ギリシャ文字だけど3つだけ覚えればOK。パン屋さんの仕事に例えてみよう！
覚える記号は3つだけ！
σ
シグマ = ブレ幅（リスク）
売上の安定度
ρ
ロー = 連動度（相関係数）
パンAとBの売れ方の関係
β
ベータ = 市場との連動度
景気に敏感かどうか
🍞 パン工房の棚に3つの瓶
朝、パン工房に行ったら棚に見たことない3つの瓶が並んでいた。
σの瓶はプルプル震えている（= ブレ幅 = リスク）、
ρの瓶は隣の瓶とロープでつながっている（= 連動度）、
βの瓶だけ窓の外の天気で色が変わる（= 市場との連動）。
— 3つの瓶の特徴を覚えれば、記号はもう怖くない！
📖 最終話：試験に出る「たった1つの公式」
🧑‍🍳
麦田店長
公式がたくさん出てくるって聞いたけど...
👨‍🏫
先生
初心者はCAPMだけ覚えればOK！「この投資はリスクに見合う儲けがあるか？」を計算する公式だよ。

💱 CAPM（きゃっぷえむ）

儲け = 安全な利回り + ベータ x 市場のご褒美

正式には E(Ri) = Rf + β x (E(Rm) - Rf)

安全な利回り（Rf）= 銀行預金の利率みたいなもの（例：1%）
ベータ（β）= その投資が市場全体と比べてどれくらい揺れるか
市場のご褒美= 「市場全体の儲け - 安全な利回り」（例：6% - 1% = 5%）

📝 麦田さんがやってみた

安全な利回り = 1%、市場のご褒美 = 5%、パン屋関連株のβ = 0.8

1% + 0.8 x 5% = 1% + 4% = 5%

「この株は最低5%の儲けがないと、リスクに見合わない」という判断基準になる！

💡
超かんたんまとめ
CAPMの計算は「安全 + ベータ x ご褒美」の3つを足し算するだけ。パン屋の原価計算（材料費 + 手間賃 x 個数）と同じ感覚！

1️⃣ ポートフォリオ = 組み合わせ

🍞 一言で

いろいろな投資先を組み合わせて持つこと。1つに集中しない。

パン屋の品揃えと同じ。食パンだけより3種類あった方が安心。

2️⃣ リスク = ブレ幅（危険ではない！）

📉 一言で

儲けの結果がどれくらいバラつくか。正式名は「標準偏差（σ）」。

毎日10万円の店 = 低リスク。5万〜20万円の店 = 高リスク。

3️⃣ 相関が低いほど分散効果が大きい

🔗 一言で

逆の動きをするもの（相関係数が-1に近い）を組み合わせると、全体のブレが一番小さくなる。

雨の日パン + 晴れの日パン = 毎日安定。

4️⃣ 期待収益率 = 配分 x 儲けの合計

💰 一言で

各投資先の「配分 x 期待収益率」を足すだけ。かんたんな掛け算と足し算。

全体の儲け = 配分A x 儲けA + 配分B x 儲けB

5️⃣ CAPM = 安全 + ベータ x ご褒美

💱 一言で

「この投資はリスクに見合うか？」を判断する公式。3つの数字を代入するだけ。

必要な儲け = Rf + β x (Rm - Rf)

Rf=安全な利回り、β=揺れ度、Rm-Rf=市場のご褒美

🛀 お風呂で5つの指を折って確認
お風呂に入ったら、湯船で5本の指を折りながら復習。
① ポートフォリオ（組み合わせ）
② リスク（ブレ幅）
③ 相関低い→分散効果大
④ 期待収益率（配分 x 儲け）
⑤ CAPM（安全+βxご褒美）
— 5本全部折れたらお風呂から出てOK！
記号カード：σ（シグマ）= ブレ幅 　 ρ（ロー）= 連動度 　 β（ベータ）= 市場との揺れ度
2大引っかけ
• 「リスクの意味は？」→ ブレ幅（標準偏差）。「損する確率」ではない！
• 「分散投資ですべてのリスクを消せる？」→ 消せない。市場全体のリスクは残る！
