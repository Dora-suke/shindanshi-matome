始点投入とは
直接材料費は始点投入 = 工程の最初にまとめて投入される

→ 換算量 = 個数 x 進捗度（月末400個 x 50% = 200個分の加工しか済んでいない）

先入先出法

当月コストだけ

月初+当月を混ぜる
月初仕掛品のコストも単価計算に入れる。
月初と当月を合わせて平均単価を出す。

STEP 3: 平均単価を出す
平均法では「月初＋当月の合計」÷「総換算量」で平均単価を出す。月初コストも入れて平均する。

直接材料費の平均単価

平均単価 = (8,000 + 27,000) ÷ (600 + 400) = 35,000 ÷ 1,000 = 35円
分母 1,000 = 完成品600 + 月末400（直接材料費は始点投入なので換算量=個数そのまま）

加工費の平均単価

平均単価 = (5,000 + 15,000) ÷ (600 + 200) = 20,000 ÷ 800 = 25円
分母 800 = 完成品600 + 月末200（400 x 50%）
★ ポイント: 月初のコストも含めて平均する → 月初の割高コストが混ざって単価が上がる。
先入先出法: 材料30円/加工20円 → 平均法: 材料35円/加工25円（どちらも上がる）

なぜ差が出る？
月初の1個あたりコストが、当月より高いからです。
平均法は割高な月初コストを混ぜるので単価が上がり、月末仕掛品が高くなる。
その分だけ完成品原価（= 総原価 - 月末）は下がる。

直接材料費

月初: 8,000円 ÷ 100個 = 80円
当月: 27,000円 ÷ 900個 = 30円
→ 月初が2.7倍割高

加工費

月初: 5,000円 ÷ 50個 = 100円
当月: 15,000円 ÷ 750個 = 20円
→ 月初が5倍割高

まとめ
月初と当月の1個あたり単価のギャップが大きいほど、2つの方法の差は大きくなる。
もし月初と当月の単価が同じ（ギャップゼロ）なら、先入先出法と平均法の結果は完全に一致する。
✏️ 穴埋め初級｜総合原価計算の基本
総合原価計算は大量・連続生産に適用
2大方法：先入先出法と平均法
仕掛品の評価には完成品換算量を使う
✏️ 穴埋め中級｜計算式
平均法：完成品単価＝（期首仕掛品＋当期投入）÷（完成品＋期末仕掛品換算量）
先入先出：当期投入分のみ÷完成品換算量−期首仕掛品換算量＝当期単価
完成品換算量＝個数×加工進捗度
✏️ 穴埋め上級｜2方法の使い分け
平均法：期首仕掛品と当期投入を混ぜて単価計算
先入先出：期首仕掛品が先に完成する前提で、当期投入は別単価
価格変動が大きい場合、先入先出の方が情報価値が高い
✏️ 穴埋め通常｜全体まとめ
2方法：先入先出法と平均法
完成品換算量＝個数×加工進捗度
平均法：（期首＋投入）÷（完成品＋期末換算量）
先入先出：当期投入÷（完成品換算量−期首換算量）
