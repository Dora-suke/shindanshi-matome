📌 勘定科目ってなに？

勘定科目とは、お金の出入りに名前をつけたラベルのこと。

スーパーの「レシート」を想像してください。「野菜」「お菓子」「飲料」など分類されていますよね。会社のお金の動きも同じように分類するのが勘定科目です。

全ての勘定科目は、次の5つのグループのどれかに属します。

🏷️ 5つのグループ（5大分類）
💰資産 持っているもの・権利
📝負債 返す義務があるもの
🏛️純資産 自分のお金（元手＋利益）
📈収益 稼いだお金
💸費用 使ったお金
資産・負債・純資産 → 貸借対照表（B/S）に載る
収益・費用 → 損益計算書（P/L）に載る
⚖️ 借方・貸方 ＝ 増える側のルール

各グループの科目が増えるときに記入する側が決まっています。

借方（左）で増える
資産（現金が増えた等）
費用（給料を払った等）
貸方（右）で増える
負債（借入が増えた等）
純資産（出資を受けた等）
収益（売上があった等）
「借方はかりかた → 左の「か」の払いが左向き」
「貸方はかしかた → 右の「し」の払いが右向き」で覚える！
借方で増えるもの：「し（資産）ひ（費用）＝しひ」
→ 「資産と費用は左（借方）で増える」の頭文字。残りの負債・純資産・収益は右（貸方）。
回転寿司で例えると、左手でお皿を取る（資産GET・費用発生）、右手でお会計を出す（負債・純資産・収益）。左右の皿の枚数は必ず一致します！
💰 資産の勘定科目

「会社が持っているもの・もらう権利」です。増えたら借方（左）。

資産のツリー構造
  • 資産（もっているもの）
    • 流動資産 ← 1年以内に現金化できる
      • 現金 手元のお金
      • 普通預金・当座預金 銀行口座
      • 売掛金 本業のツケ売り
      • 受取手形 約束手形をもらった
      • 商品 売るための在庫
      • 前払金 先に払った（権利）
      • 貸付金（短期） 1年以内に返ってくる
    • 固定資産 ← 長期間使い続けるもの
      • 建物 オフィス・店舗
      • 土地 駐車場・敷地
      • 備品 パソコン・机
      • 車両運搬具 トラック・社用車
      • のれん 買収時のブランド価値
「1年以内にお金に変わるか？」が流動 vs 固定の分かれ目。ワンイヤールールと呼ばれます。
コンビニの商品棚を想像してください。おにぎり（すぐ売れる）＝流動資産。レジの機械（何年も使う）＝固定資産。「すぐ現金化できるか？」がポイントです。

📝 負債の勘定科目

「会社が返さなければならない義務」です。増えたら貸方（右）。

負債のツリー構造
  • 負債（返す義務）
    • 流動負債 ← 1年以内に返すもの
      • 買掛金 本業の仕入ツケ
      • 支払手形 約束手形を渡した
      • 未払金 本業以外の後払い
      • 前受金 先にもらった（義務）
      • 預り金 他人から預かったお金
    • 固定負債 ← 返済が1年超のもの
      • 借入金（長期） 銀行ローン等
      • 社債 会社が発行した借金の証書
負債も「1年以内に返すか？」で流動 vs 固定に分かれます。資産と同じワンイヤールール！
科目名ひとこと身近な例
買掛金ツケで買った代金（後で払う）お店への未払い
支払手形「後で払います」と書いた紙約束手形を渡した
借入金銀行などから借りたお金住宅ローン的なもの
未払金商品以外の未払い備品代の後払い
前受金まだ届けていない商品の代金予約の手付金
社債会社が発行した借金の証書会社版の借用書
預り金他人から預かったお金従業員の源泉税
🏛️ 純資産の勘定科目

「返す必要がない自分のお金」です。増えたら貸方（右）。

まず大前提！
会社のお金の出どころは2つだけ：
 (1) 他人から借りたお金 → 負債（いつか返す）
 (2) 自分たちのお金 → 純資産（返さなくてOK）

つまり 「資産 − 負債 ＝ 純資産」。
純資産が大きいほど「借金に頼っていない健全な会社」ということ！
マイホームでイメージしよう
家の価値（資産）3,000万円のうち…
 住宅ローン（負債）＝ 2,000万 → いつか返すお金
 自分の持ち分（純資産）＝ 1,000万 → 返さなくていいお金

この1,000万の中身をさらに分解すると…
 頭金として最初に払った分 → 資本金（株主が出資したお金）
 毎月コツコツ返済して増えた持ち分 → 利益剰余金（稼いで貯めたお金）
これが純資産の基本イメージです！
純資産のツリー構造 ― ここが一番わかりにくい！
  • 純資産（返さなくていい自分のお金）
    • 株主資本 ← 純資産のメイン。試験で最頻出！
      • 資本金 ← 株主が「出資します！」と払い込んだお金
        • 起業時：「会社を作ろう！」と出した元手
        • 増資時：「もっと大きくしよう！」と追加で集めたお金
        • 登記簿に載る公式な金額
      • 資本剰余金 ← 出資なのに資本金に入れなかった「おまけ」部分
        • 資本準備金 ← 株式発行額の半分まで、資本金にしなくてOK。その分がここ
        • その他資本剰余金 ← 自己株式の処分差益など（出題頻度は低い）
      • 利益剰余金 ← 会社が稼いで貯めた利益の合計。「会社の貯金箱」
        • 利益準備金 ← 法律で「配当するなら10分の1を積み立てなさい」と決まっている分
        • 繰越利益剰余金 ← まだ「配当する？再投資する？」を決めていない利益のプール
      • 自己株式 ← 自社の株を買い戻したもの（マイナス項目！）
        • 純資産から差し引かれる。持っていると純資産が減る
    • 評価・換算差額等 ← まだ確定していない含み損益
      • その他有価証券評価差額金 持っている株の値上がり・値下がり分
超重要！ 2つの「出どころ」を見分けるコツ

株主資本はお金の出どころで2つに分かれます：

 株主が出してくれた → 資本金 ＋ 資本剰余金
  例：友人3人が100万ずつ出資 → 資本金150万＋資本準備金150万

 会社が自分で稼いだ → 利益剰余金
  例：今期の利益100万 → 繰越利益剰余金に追加

試験での聞かれ方：「資本金と利益剰余金の違いは？」
→ 出どころが違う！ 出資されたお金 vs 稼いで貯めたお金
資本金 vs 資本剰余金 ― なぜ分ける？

1株 1,000円で100株を発行して10万円を調達！
 → 法律では「最低半分（5万円）を資本金にすればOK」
 → 残りの5万円は資本準備金（資本剰余金の一部）に

なぜわざわざ分けるの？
 → 資本金は「会社の信用の看板」。簡単に減らせないように法律で保護
 → 資本準備金は「いざという時の予備のお金」。少し柔軟に使える
 → どちらも株主がくれたお金という点は同じ！
利益剰余金の流れ ― 稼いだ後どうなる？

Step 1：今年100万円の利益が出た！
 → まず全額「繰越利益剰余金」に入る

Step 2：株主総会で使い道を決める
 → 30万円は株主に配当
 → 10万円は利益準備金に積立て（法律上の義務）
 → 残り60万円はそのまま繰越利益剰余金に残る

ポイント：利益剰余金 ＝ 利益準備金 ＋ 繰越利益剰余金（上位概念）

📈 収益の勘定科目

「会社が稼いだお金」です。増えたら貸方（右）。

収益のツリー構造
  • 収益（稼いだお金）
    • 営業収益 ← 本業で稼いだ
      • 売上 商品・サービスの販売
    • 営業外収益 ← 本業以外で定期的に入る
      • 受取利息 預金や貸付金の利息
      • 受取配当金 持っている株の配当
      • 雑収入 自販機手数料など
    • 特別利益 ← たまにしか起こらない
      • 固定資産売却益 土地等を帳簿より高く売った
P/L（損益計算書）では「営業利益 → 経常利益 → 当期純利益」と段階的に計算します。上の分類は各段階のどこに入るかに対応しています。

受取利息お金を貸して得た利息預金の利息
受取配当金持っている株の配当金株の配当

💸 費用の勘定科目

「会社が使ったお金」です。増えたら借方（左）。

費用のツリー構造
  • 費用（使ったお金）
    • 売上原価 ← 売った商品の仕入コスト
      • 仕入 材料・商品の購入
    • 販売費及び一般管理費（販管費） ← 日常の運営コスト
      • 給料 従業員の報酬
      • 広告宣伝費 チラシ・Web広告
      • 水道光熱費 電気・ガス・水道
      • 通信費 電話・ネット代
      • 支払家賃 オフィスの家賃
      • 減価償却費 設備の年間の価値低下
      • 租税公課 固定資産税・印紙代
    • 営業外費用 ← 本業以外の定期コスト
      • 支払利息 借入金の利息
    • 特別損失 ← たまにしか起こらない
      • 貸倒損失 回収不能になった売掛金
      • 固定資産売却損 帳簿より安く売った損

⚠️ 間違えやすい！紛らわしい勘定科目

売掛金 vs 未収入金

売掛金 資産

本業の商品・サービスを売った未回収金

例：パン屋がパンをツケで売った

未収入金 資産

本業以外のもので未回収のお金

例：パン屋が不要な車を売った代金が未回収

買掛金 vs 未払金

買掛金 負債

本業の仕入を後払いにしたもの

例：小麦粉をツケで仕入れた

未払金 負債

本業以外の支払いが残っている

例：パソコンを買って後払い

前払金 vs 前受金

前払金 資産

商品を受け取る前に払ったお金

自分が先に払った → 商品をもらう権利

前受金 負債

商品を届ける前にもらったお金

先にもらった → 届ける義務

映画チケットの前売りで覚えよう！ お客さん目線：先にお金を払った → 映画を見る権利 → 前払金（資産）。映画館目線：先にお金をもらった → 映画を見せる義務 → 前受金（負債）。同じ取引でも立場が逆なら科目も逆！

貸付金 vs 借入金

貸付金 資産

他人に貸したお金 → 返してもらう権利

借入金 負債

他人から借りたお金 → 返す義務

試験での頻出ひっかけ
  • 「売掛金」と「未収入金」 → 本業かどうかで判断
  • 「前払」と「前受」 → 自分が払ったか・もらったかで判断
  • 「受取手形」は資産、「支払手形」は負債
  • 「のれん」は資産（買収時にのみ発生）
  • 「減価償却累計額」は資産のマイナス（控除項目）

5グループ（資産・負債・純資産・収益・費用）を押さえれば
どの科目も「どこに属するか」で借方・貸方が決まる！

📐 区分の全体マップ

B/S（貸借対照表）の区分

資産の区分

流動資産：1年以内に現金化
（現金、売掛金、商品 等）

固定資産：長期間使う
（建物、土地、備品、のれん 等）

負債の区分

流動負債：1年以内に返す
（買掛金、前受金、預り金 等）

固定負債：返済が1年超
（長期借入金、社債 等）

純資産の区分

資本金：株主が出資したお金
（起業時の元手、増資 等）

資本剰余金：出資のうち資本金に入れなかった分
（資本準備金 等）

未収入金 資産

DRILL 3仕入先から材料を50万円分購入、支払いは来月末
この「来月払う義務」を記録する勘定科目は？
買掛金は増えた。借方（左）？ 貸方（右）？
買掛金 負債
本業の材料の仕入れ → 「買掛金」。未払金は本業以外の後払い（例：パソコン購入）。支払手形は手形を渡した場合。
貸方（右）
買掛金は負債。負債が増える → 貸方（右）。同時に仕入（費用）が借方（左）に入ります。
DRILL 4新しいパソコンを20万円で購入、来月払い
この「来月払う義務」を記録する勘定科目は？
未払金は増えた。借方（左）？ 貸方（右）？
未払金 負債
パソコンは本業の商品ではない → 買掛金ではなく「未払金」。「本業の仕入=買掛金」「それ以外=未払金」これだけ！
貸方（右）
未払金は負債。負債が増える → 貸方（右）。買掛金と同じ負債だから同じ側です。
DRILL 5お客さんから商品の予約金として3万円をもらった（まだ届けていない）
この「先にもらったお金」を記録する勘定科目は？
前受金は増えた。借方（左）？ 貸方（右）？
前受金 負債
商品をまだ届けていない → 届ける義務がある → 負債！ 売上はサービス提供後。前払金は自分が払った場合（逆）。
貸方（右）
前受金は負債。負債が増える → 貸方（右）。同時に現金（資産）が借方（左）に入ります。
前払金と前受金は試験の超頻出ペア！ 「前払＝払った側＝資産」「前受＝もらった側＝負債」。主語が誰かを確認する癖をつけよう。
DRILL 6来月届く商品の代金として仕入先に10万円を先に振り込んだ
この「先に払ったお金」を記録する勘定科目は？
前払金は増えた。借方（左）？ 貸方（右）？
前払金 資産
先に払った → 商品をもらう権利がある → 資産！ 前受金は逆で「先にもらった＝義務＝負債」。仕入は商品が届いてから。
借方（左）
前払金は資産。資産が増える → 借方（左）。前受金（負債）と逆の側になるのがポイント！
DRILL 7建物の今年分の減価償却を計上する
「価値の目減り分」を費用として記録する勘定科目は？
減価償却費は発生した。借方（左）？ 貸方（右）？
減価償却費 費用
設備の価値が年々下がる分を費用化。修繕費は壊れた物の修理代。租税公課は税金関連。名前に「費」がつくので費用グループ。
借方（左）
減価償却費は費用。費用が増える → 借方（左）。相手科目は建物や備品（資産の減少）が貸方に入ります。
DRILL 8取引先が倒産して、売掛金20万円が回収できなくなった
この回収不能を記録する勘定科目は？
貸倒損失は発生した。借方（左）？ 貸方（右）？
貸倒損失 費用
回収不能 → 損失 → 費用として計上。売掛金は相手科目（貸方で消す）。「損失」とつく科目は基本的に費用グループ。
借方（左）
貸倒損失は費用。費用が増える → 借方（左）。同時に売掛金（資産）が貸方で減少します。
DRILL 9従業員の給料から源泉所得税5万円を天引きした
天引きして一時的に預かっている税金を記録する勘定科目は？
預り金は増えた。借方（左）？ 貸方（右）？
預り金 負債
従業員の税金を代わりに預かっている → 税務署に納める義務がある → 負債。租税公課は自社が負担する税金。未払金とは性質が違います。
貸方（右）
預り金は負債。負債が増える → 貸方（右）。給料（費用）の借方と、預り金（負債）の貸方がセットです。
DRILL 10株主が会社に500万円を出資した
この出資金を記録する勘定科目は？
資本金は増えた。借方（左）？ 貸方（右）？
資本金 純資産
株主からの出資 → 返す義務なし → 純資産。借入金は「返す必要あり」で負債。利益剰余金は「稼いで貯めた」もので出資とは別。
貸方（右）
資本金は純資産。純資産が増える → 貸方（右）。同時に普通預金（資産）が借方で増えます。

判断の黄金ルール
  • 本業か否か → 売掛金/買掛金 vs 未収入金/未払金
  • 自分が払ったか/もらったか → 前払金（資産） vs 前受金（負債）
  • 貸したか/借りたか → 貸付金（資産） vs 借入金（負債）
  • 権利か義務か → 権利＝資産（借方）、義務＝負債（貸方）

Q1ラーメン屋の開業資金
鈴木さんは自己資金300万円でラーメン屋を始めました。この300万円を記録する勘定科目として正しいものは？
正解：C. 資本金（純資産）

Q2ネット通販の「ツケ払い」
アパレルのネットショップが、仕入先から商品を50万円分仕入れ、来月末払いにしました。この「後で払う義務」を表す科目は？
正解：A. 買掛金（負債）
本業の商品の仕入れを後払いにしたら「買掛金」。Bの未払金は本業以外の後払い（例：パソコン購入）。Cの売掛金は売った側の未回収金。Dの前払金は先にお金を払って商品を待つケース。「本業の仕入＝買掛金」と覚えましょう。
Q3町工場の新しい機械
町工場が新型の旋盤（工作機械）を150万円で購入し、現金で支払いました。この旋盤を記録する勘定科目の分類は？
正解：D. 資産（備品 or 機械装置）
長期間使うものは「資産」として記録します。「買ったから費用」と思いがちですが、何年も使える設備は費用ではなく資産。毎年「減価償却費」という費用で少しずつ費用化していきます。これが費用と資産を間違えやすいポイントです。
Q4カフェの家賃
カフェのオーナーが今月分の家賃20万円を銀行振込で支払いました。この仕訳で「支払家賃」はどちら側に記入する？
正解：B. 借方（左）
支払家賃は費用。費用は増えたとき借方（左）に書きます。「借方＝資産・費用が増える側」がルール。同時に普通預金（資産）が減るので貸方に記入します。
Q5美容室の予約金
人気美容室が、結婚式の特別ヘアメイクの予約として、お客様から3万円を先に受け取りました。この3万円の勘定科目は？
正解：D. 前受金（負債）
まだサービスを提供していないのに先にもらったお金は「前受金」。サービスを届ける義務があるので負債です。Aの売上はサービス提供後に計上。Bの前払金は自分が先に払った場合。Cの売掛金はサービス後の未回収金です。
Q6配送業者のトラック売却
配送会社が古いトラック（帳簿価額50万円）を70万円で売却し、代金は来月入金予定。この「来月もらう70万円」の科目は？
正解：B. 未収入金（資産）
配送会社の本業はトラックを売ることではなく「配送」。本業以外の取引で後から受け取るお金は「未収入金」。Aの売掛金は本業の売上の未回収。なお、差額20万円（70万−50万）は「固定資産売却益」として収益に計上します。
Q7IT企業の銀行借入
IT企業が開発資金として銀行から500万円を借り入れ、普通預金に入金されました。この仕訳の貸方に来る科目は？
正解：C. 借入金（負債）
銀行から借りたお金は「借入金」（負債）。負債が増えると貸方。借方には普通預金（資産の増加）が入ります。Bの資本金は株主の出資。Dの貸付金は自分が他人に貸した場合なので逆です。
Q8飲食店の月末決算
飲食店で使用している冷蔵庫（取得原価120万円、耐用年数6年、残存価額ゼロ、定額法）の1年分の減価償却費はいくら？
正解：A. 20万円
定額法：120万円 ÷ 6年 ＝ 20万円。Bの12万は120÷10で耐用年数を間違えたケース。Cの60万は120÷2で半分にしてしまったケース。「取得原価 ÷ 耐用年数」がシンプルな公式です。

CASE 1武田薬品 × シャイヤー ― 6.8兆円の大型買収と「のれん」

2019年、武田薬品工業はアイルランドの製薬大手シャイヤーを約6兆8,000億円で買収しました。日本企業による海外M&Aとして過去最大級です。

買収後、2020年にはグループ内の再編として子会社「シャイアー・ジャパン」を吸収合併。武田薬品工業が存続会社、シャイアー・ジャパンが消滅会社となりました。

のれん（資産）：買収価格が相手の純資産を上回った差額。シャイヤーのブランド力・技術力・パイプラインに対する評価額です。武田薬品のB/Sには約4兆円の「のれん」が計上されました。

なぜ「のれん」は資産？
→ 目に見えないけど将来利益を生むと期待されるもの。日本基準では毎年のれん償却費（費用）として少しずつ費用化します。

関連する主な勘定科目：
  • のれん 資産 ← 買収プレミアム
  • のれん償却費 費用 ← 毎年の費用化
  • 資本金・資本剰余金 純資産 ← 株式交換で増加
  • 社債・借入金 負債 ← 買収資金の一部
のれんが大きすぎると、減損テスト（価値が下がっていないかチェック）で一気に「のれん減損損失（費用）」が発生するリスクがあります。東芝のウェスチングハウス買収でのれん減損が巨額に上り、経営危機に発展した事例は記憶に新しいところです。
「のれん」は料理店でいう「老舗の暖簾（のれん）の価値」。お店の建物や食材だけでなく、ブランド力や常連客という目に見えない価値にお金を払っているのです。試験では「のれんは資産！」と即答できるようにしておきましょう。
CASE 2ソフトバンクグループ ― 巨額の社債発行と支払利息

ソフトバンクグループは積極的な投資のために社債を大量に発行することで有名です。有利子負債は約10兆円規模に上ります。

社債を発行すると、投資家からお金が入り、その代わり定期的に利息を支払い、満期には元本を返す義務が生じます。

社債発行時：
→ 普通預金（資産）が増える ／ 社債（負債）が増える

利息支払い時：
→ 支払利息（費用）が発生 ／ 普通預金（資産）が減る

満期償還時：
→ 社債（負債）が減る ／ 普通預金（資産）が減る

関連する主な勘定科目：
  • 社債 負債 ← 借金の証書
  • 支払利息 費用 ← 利息コスト
  • 普通預金 資産 ← 入出金
中小企業の場合は社債より「借入金」が一般的。でも構造は同じです。お金を借りる → 負債が増える → 利息を払う → 費用が発生。
CASE 3街のパン屋さん ― キッチンカー事業を始める

個人でパン屋を営む佐藤さんは、イベント出店用にキッチンカーを200万円で購入。うち100万円は自己資金、残り100万円は日本政策金融公庫から借入れました。

初月のイベント売上は現金で25万円。キッチンカーの燃料代（ガソリン）は2万円、イベント出店料5万円は来月払い。

キッチンカー購入時：
  • 車両運搬具 200万 資産 ← キッチンカーを取得
  • 普通預金 100万 資産 ← 自己資金の支出（貸方で減少）
  • 借入金 100万 負債 ← 公庫からの借入

イベント売上と経費：
  • 現金 25万 ／ 売上 25万 ← 本業の売上
  • 旅費交通費(燃料代) 2万 ／ 現金 2万 ← 費用の発生
  • 支払手数料 5万 ／ 未払金 5万 ← 出店料は本業の仕入ではないので「未払金」
出店料を「買掛金」にしないのがポイント！買掛金は「本業の仕入（パンの材料など）」の後払いだけ。出店料は本業の商品仕入ではないので「未払金」を使います。
CASE 4ファーストリテイリング（ユニクロ）― 店舗の減価償却と広告宣伝費

ユニクロは世界中に店舗を展開しています。新店舗を出すたびに建物の内装工事費（建物附属設備）や備品（什器・マネキンなど）が資産に計上されます。

これらは購入時に一括で費用にするのではなく、耐用年数にわたって減価償却していきます。また、テレビCMやSNS広告には年間数百億円の広告宣伝費を使います。

「資産」になるもの vs 「費用」になるもの ― この違いが重要！

資産になる（何年も使える）

建物附属設備 資産

備品（什器） 資産

→ 毎年「減価償却費」で費用化

すぐ費用になる（効果が短い）

広告宣伝費 費用

水道光熱費 費用

→ 使った期に全額費用

「長く使えるもの＝資産 → 徐々に費用化」「使ったらすぐ終わり＝即費用」。この判断基準が減価償却を理解するカギです。
試験の引っかけ注意！「150万円の機械を買った → 費用？」と聞かれたらNO！資産です。費用になるのは毎年の減価償却費だけ。「買った＝費用」と反射的に答えないよう注意しましょう。
CASE 5居酒屋チェーン ― コロナ禍での貸倒損失

ある居酒屋チェーンでは、法人向けの宴会プランを売掛金で受けていました。しかし2020年のコロナ禍で取引先の中小企業が複数倒産。売掛金300万円が回収不能になりました。

回収不能が確定したとき：
  • 貸倒損失 300万 費用 ← 回収できない損失
  • 売掛金 300万 資産 ← 消滅（貸方で減少）

「貸倒引当金」で事前に備えていた場合：
  • 決算時に「将来回収できないかも」と見積って引当金を計上しておく
  • 貸倒引当金繰入額（費用）／ 貸倒引当金（資産のマイナス）
  • → 実際に回収不能になったら：貸倒引当金 ／ 売掛金
引当金を設定しておくと、実際に貸倒れたときの損益へのインパクトを平準化できます。中小企業でも売掛金が多い業種は貸倒引当金の計上が重要です。
