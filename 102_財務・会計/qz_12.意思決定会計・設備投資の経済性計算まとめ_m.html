📝 お題：たこ焼き屋台を買うべき？

屋台の値段：100万円（投資額）

毎年の稼ぎ：40万円（CF）× 3年間

割引率：10%（お金を借りるコスト）

1 毎年のCFを「今の価値」に直す

1年目：40万 ÷ 1.1 ＝ 36.4万

2年目：40万 ÷ 1.1² ＝ 40万 ÷ 1.21 ＝ 33.1万

3年目：40万 ÷ 1.1³ ＝ 40万 ÷ 1.331 ＝ 30.1万

↑ 遠い将来ほど「今の価値」が小さくなるのがポイント！

2 合計する

36.4 + 33.1 + 30.1 ＝ 99.6万円

3 投資額を引く ＝ NPV

NPV = 99.6万 − 100万 = −0.4万円
NPVが −0.4万円（マイナス）→ 投資しない方がいい！
40万×3年＝120万で一見お得に見えるが、お金の時間価値を考えるとギリギリ損。これが「現在価値に直す意味」！
👨

もし毎年の稼ぎが45万円だったら？ 現在価値合計は約111.9万で NPV＝+11.9万。こうなれば投資GO！ たった5万円の差で結論が変わる。だから計算が大事なんだよ。

⭐ 覚えるべき4つの手法

設備投資の判断方法は4つ！
手法ひとことで判断基準
NPV法
(正味現在価値法)		将来のお金を今の価値になおして比べるNPV ≥ 0 なら投資OK
IRR法
(内部収益率法)		投資の利回りを求めるIRR ≥ 資本コスト なら投資OK

(PI法)		1円あたり何円もどってくるかPI ≥ 1 なら投資OK

💵 NPV（正味現在価値）って？

🧑
受験生
「現在価値になおす」ってどういうこと？ あと「CF」って何？
👨
先生
CF＝キャッシュ・フロー（実際に出入りするお金のこと）だよ。そして今の100万円と、5年後の100万円は同じ価値じゃないよね。今の100万円を銀行に預ければ増えるから、今の方が価値が高い。将来のお金を「今だったらいくら？」に換算するのが現在価値だよ。
💡

会社がお金を調達するコストのこと。銀行から借りれば利息、株主から集めれば期待リターン。「お金を借りる利息のようなもの」と思えばOK！
NPV = 将来CFの現在価値の合計 − 投資額

📅 毎年CF発生売上−費用
→
🔢 現在価値に割引÷(1+r)n
→
✅ NPV判断≥0 なら投資GO!

📊 貨幣の時間価値

⏰ なぜ「今の1万円」と「1年後の1万円」は違う？

今の1万円を年利5%で運用すれば、1年後には10,500円。逆に1年後の10,500円の「今の価値」は1万円。これが現在価値（PV）の考え方です。

現在価値 = 将来の金額 ÷ (1 + r)n

r = 割引率（資本コスト）、n = 年数

🏃 例：マラソン大会の賞金 「来年もらえる賞金100万円」は、今すぐもらえる100万円より価値が低い。なぜなら今もらえば1年間運用して増やせるから。この「今の価値に直す」のが現在価値の考え方。

🔄 年金現価係数

毎年同額のCFが入ってくる場合、いちいち1年ずつ割り引かなくてもまとめて計算できる便利な係数。

年金現価係数 = {1 − (1+r)−n} ÷ r

毎年CFが同じなら → CF × 年金現価係数 で現在価値の合計が一発で出る！

👨
先生
年金現価係数の式は覚えなくてOK！試験では必ず「係数表」が与えられるよ。表から数字を読んで掛け算するだけ。ECサイトの「送料込み価格を自動計算してくれるカート」みたいなもの！

💰 4つの投資評価手法（詳細）

① NPV法（正味現在価値法）

NPV = ∑ CFt ÷ (1+r)t − 初期投資額
  • NPV ≥ 0 → 投資する
  • 複数案があるとき → NPVが最大の案を選ぶ
  • 貨幣の時間価値を考慮した最も理論的な手法

🏫 例：学校の太陽光パネル 設置費500万円、年間の電気代節約30万円。20年分の節約を現在価値に直して合計し、500万円を超えるか？がNPV法の判断。

② IRR法（内部収益率法）

NPV = 0 となる割引率 = IRR
  • IRR ≥ 資本コスト（ハードルレート） → 投資する
  • NPV法と結論が一致することが多いが、投資規模がちがう案の比較には不向き
  • 試験では補間法（内挿法）で求めることが多い

🎯 スポーツで例えると：IRRは「選手の打率」のようなもの。打率（IRR）が合格ライン（資本コスト）を超えていればレギュラー入り（投資採用）！

③ 回収期間法

回収期間 = 投資額 ÷ 毎年のCF
  • 回収期間が目標年数以内 → 投資する
  • かんたんでわかりやすいが、回収後のCFを無視する欠点あり
  • 貨幣の時間価値を考慮しない（割引回収期間法なら考慮する）

🍣 例：回転寿司の食洗機 100万円の業務用食洗機を導入。バイト代が年50万円減るなら回収期間＝2年。「2年で元がとれる」の判断がシンプル！

④ 収益性指数法（PI法）

PI = 将来CFの現在価値合計 ÷ 初期投資額
  • PI ≥ 1 → 投資する
  • 投資額1円あたりのリターンを示す → 投資規模が異なる案の比較に便利

🛒 例：ECサイトの広告 A案（広告費50万で売上80万相当、PI=1.6）とB案（広告費200万で売上280万相当、PI=1.4）。予算が限られるならPI＝コスパが高いA案を優先！

4手法の比較マトリクス
手法時間価値指標の種類弱点
NPV法✅ 考慮金額（絶対額）規模の異なる案の比較に注意
IRR法✅ 考慮%（利回り）規模が違う案の比較に不向き
PI法✅ 考慮倍率（相対値）単独では規模がわからない
回収期間法❌ 無視年数回収後のCFを無視
💡
NPV vs PI の使い分け
NPV法＝「どっちが儲かる金額が大きいか？」（大盛りラーメンvs普通盛り）
PI法＝「1円あたりの効率がいいのは？」（コスパで比較）
予算に限りがあって複数案を組み合わせたい場合はPI法が便利！
手法の比較 − 試験頻出！
• 貨幣の時間価値を考慮 → NPV法、IRR法、PI法
• 考慮しない → 回収期間法（ただし割引回収期間法は考慮する）
• 最も理論的 → NPV法

💵 キャッシュ・フローの計算

🏭 設備投資のCF計算

設備投資では「会計上の利益」ではなくキャッシュ・フローで考えます。

税引後CF = 税引後営業利益 + 減価償却費
= 営業CF × (1−税率) + 減価償却費 × 税率

🍳 例：料理教室の業務用オーブン 売上増300万、費用増100万、減価償却50万、税率30%の場合。税引後CF＝(300−100−50)×0.7＋50＝105+50＝155万円。「利益に減価償却を足し戻す」のがポイント。

💡
なぜ減価償却費を足す？
減価償却費は「帳簿上の費用」であって、実際にお金が出ていかない。でも費用として計上するから税金がその分安くなる（タックスシールド）。利益に足し戻してCFに変換するのがポイント！

📝 CF計算で忘れがちなポイント

  • 初期投資額：0年目にマイナスのCF
  • 残存価額（売却額）：最終年にプラスのCF（売却益に税金がかかる場合あり）
  • 運転資本の増減：投資時にマイナス、回収時にプラス
💡
運転資本って？
コンビニで例えると、新商品を仕入れるために在庫を増やす＝現金が減る。これが運転資本の増加（CF的にはマイナス）。閉店時に在庫を売り切れば現金が戻る（プラス）。

原価の種類意味意思決定で
差額原価代替案間で異なる原価考慮する

🔄 取替投資

🔧 取替投資とは

古い設備を新しい設備に置きかえるかどうかの意思決定。ポイントは「差額CF」で比較すること。

  • 新設備のCF − 旧設備のCF ＝ 差額CF
  • 新設備の投資額 − 旧設備の売却額 ＝ 差額投資額
  • 差額CFの現在価値 ≥ 差額投資額 → 取替える

🚗 例：配送業者のトラック 燃費の悪い旧トラック（売れば50万）を、燃費のよい新トラック（300万）に替えるか？ 差額投資250万と、毎年浮く燃料費（差額CF）の現在価値を比べて判断する。

取替投資の判断フロー
🏭 旧設備を売却売却額を得る
→
💸 差額投資額新−旧売却額
→
📊 差額CF計算毎年の差額
→
✅ NPVで判断≥0なら取替え
取替投資の注意点
旧設備の帳簿価額と売却額がちがうと、売却損益が発生して税金に影響する。この税効果も忘れずにCFに反映！
👨
先生
売却損益の税効果は「中古車の売却」と同じ。帳簿100万の車を60万で売ると40万の損失。この損が他の利益と相殺されて税金が安くなる（40万×税率＝節税額）。この節税分は手元に残るお金が増えるのでCFにプラス！
基本事項まとめ
• NPV法が最も理論的、回収期間法は時間価値を考慮しない
• CF計算 = 税引後利益 + 減価償却費（タックスシールド重要）
• サンクコストは無視、機会原価は考慮
• 取替投資は「差額」で考える

🏪 ケース1：パン屋さんの新オーブン導入

🧑
パン屋の店長
新しいオーブンを500万円で導入したい。毎年150万円の売上増が見込めるんだけど、買うべき？ 耐用年数5年、残存価額ゼロ、税率30%、資本コスト8%で教えてほしい。
👨
先生
NPV法でシミュレーションしてみよう！
ケース1：CFタイムライン
0年目−500万
(投資)
→
1年目+135万
→
2年目+135万
→
...毎年同額
→
5年目+135万

1 減価償却費を計算

500万円 ÷ 5年 = 100万円/年（定額法）

2 毎年の税引後CFを計算

営業CF増加 = 150万円（売上増 − 追加費用はなしと仮定）

税引後CF = 150万 × (1−0.3) + 100万 × 0.3

= 105万 + 30万 = 135万円/年

↑ 30万円が減価償却の節税効果（タックスシールド）！

3 現在価値に換算（年金現価係数を使用）

資本コスト8%・5年の年金現価係数 = 3.993（試験で与えられる係数表から読む）

CFの現在価値合計 = 135万 × 3.993 = 約539万円

4 NPVを計算

NPV = 539万 − 500万 = +39万円
NPV = +39万円 ≥ 0 → 投資すべき！
新オーブン導入で、資本コストを上回る約39万円分の価値を生みだせる。店長、GOサインです！
💡
身近に言い換えると
自動販売機の設置と同じ。設置費（投資額）を払って、毎月の売上（CF）がチャリンチャリン入ってくる。その合計の「今の価値」が設置費を超えるなら、設置する価値あり！

🔧 ケース2：町工場の機械取替え

🧑
町工場の社長
今の旧機械（帳簿価額200万円、残り耐用年数4年、残存価額0）を使い続けるか、新機械（800万円、耐用年数4年、残存0）に買い替えるか迷ってる。旧機械は今なら150万円で売れる。新機械にすると年間のコスト削減が250万円。税率30%、資本コスト10%。

1 差額投資額

新機械800万 − 旧機械売却150万 = 650万円

旧機械の売却損 = 帳簿200万 − 売却150万 = 50万円（損失）

売却損の節税 = 50万 × 30% = 15万円 → 実質投資額 = 650万 − 15万 = 635万円

💡
売却損の節税って？
損が出ると「今年は赤字が増えた」→ その分税金が安くなる → 手元にお金が残る。だから実質的な投資額が減るのです。映画で例えると、ハズレ映画のチケットを金券ショップで安く売った損失が、確定申告で少し戻ってくるイメージ！

2 差額減価償却費

新機械の減価償却 = 800万 ÷ 4年 = 200万/年

旧機械の減価償却 = 200万 ÷ 4年 = 50万/年

差額 = 200万 − 50万 = 150万円/年

3 差額税引後CF

コスト削減250万 × (1−0.3) + 差額減価償却150万 × 0.3

= 175万 + 45万 = 220万円/年

4 NPV計算

10%・4年の年金現価係数 = 3.170

CFの現在価値 = 220万 × 3.170 = 約697万円

NPV = 697万 − 635万 = +62万円
NPV = +62万円 ≥ 0 → 買い替えるべき！
社長、新機械への取替えで62万円分の価値創造ができます。古い機械を使い続けるよりお得です！
💡
身近に言い換えると
古いエアコンを新型に買い替えるか迷うのと同じ。古いエアコンは電気代が月2万円、新型なら月8千円。「新型の本体代−古いエアコンの下取り額」と「毎月浮く電気代」を比べて判断する！

⚠️ ケース3：サンクコストのワナ

🧑
IT企業の部長
新システム開発に既に300万円つぎこんだ。完成にはあと200万円必要だけど、完成しても効果は400万円の見込み。「300万円もったいないからやめられない」んだけど...
⚠️
要注意！
すでに使った300万円はサンクコスト（埋没原価）。どちらを選んでも戻ってこないので無視！

正しい判断：
• 続ける → 追加200万で400万の効果 → 差額+200万
• やめる → 追加0円で効果0円 → 差額0円
→ 続けるべき！（追加200万で400万のリターン）
「もったいない」で判断するのは危険！
サンクコストに引きずられて、本当は中止すべきプロジェクトを続けてしまう＝コンコルドの誤謬。意思決定では「これから」のコストと効果だけを比べること！
⚠️
逆パターン：中止すべきケース
もしケース3で「追加200万円かけても効果が150万円しかない」なら？ 300万円使ったことは忘れて、追加200万 > 効果150万 → 差額−50万で中止が正解。「既に300万使ったのに…」と考えるのがサンクコストの罠！

  • B. 投資すべきでない（300万円は高すぎる）
  • C. NPVだけでは判断できない

NPV ≥ 0 なら投資すべき！+50万円ということは、資本コストを上回る50万円分の価値を生みだせるということ。たけしさん、製麺機を買いましょう！

基本事項
美容室オーナーのゆきさん。店の改装に既に100万円使ったが、工事が中途半端で追加50万円が必要。完成すれば売上増加の効果は80万円。この「既に使った100万円」は意思決定でどう扱う？
  • A. 100万円＋50万円の合計150万円と80万円を比較する
  • B. 100万円を回収できるまで工事を続けるべき
  • C. 100万円は埋没原価なので無視し、追加50万円と効果80万円で判断
  • D. 100万円は機会原価として計上する
✅ 正解：C
既に支払った100万円はサンクコスト（埋没原価）。どちらを選んでも戻らないので無視！追加50万で80万の効果 → 差額+30万で続行が正解。
3
基本事項
次のうち、貨幣の時間価値を考慮しない投資評価手法はどれか。
  • A. 正味現在価値法（NPV法）
  • B. 回収期間法
  • C. 内部収益率法（IRR法）
  • D. 収益性指数法（PI法）
✅ 正解：B
回収期間法は「何年で元がとれるか」だけを見るので、貨幣の時間価値を考慮しない。NPV法・IRR法・PI法はいずれも割引計算を行うので時間価値を考慮する。

シミュレーション
花屋のまことさんが配送用バンを200万円で購入検討中。毎年の税引後CF60万円、耐用年数5年。資本コスト10%、5年の年金現価係数は3.791。NPVはいくら？
  • A. +100.0万円
  • B. −72.5万円
  • C. +60.0万円
  • D. +27.5万円
✅ 正解：D（+27.5万円）
CFの現在価値 = 60万 × 3.791 = 227.5万円
NPV = 227.5万 − 200万 = +27.5万円。プラスなので投資OK！

基本事項
居酒屋オーナーが新店舗と既存店リニューアルで迷っている。新店舗にすると既存店で得られるはずの年間利益300万円を失う。この300万円は何と呼ばれる？
  • A. 機会原価（機会コスト）
  • B. 埋没原価（サンクコスト）
  • C. 差額原価
  • D. 限界原価
✅ 正解：A
他の選択肢を選んでいたら得られたはずの利益 ＝ 機会原価。新店舗を選ぶことで失う既存店の利益300万円がまさにこれ。意思決定では考慮すべき！

入門
減価償却費が年100万円、税率30%のとき、減価償却による節税効果（タックスシールド）はいくら？
  • A. 100万円
  • B. 70万円
  • C. 30万円
  • D. 0円（減価償却は節税にならない）
✅ 正解：C（30万円）
タックスシールド = 減価償却費 × 税率 = 100万 × 0.3 = 30万円。減価償却費は実際にお金が出ないのに費用になるので、その分だけ税金が安くなる！

シミュレーション

  • A. 400万円
  • B. 300万円
  • C. 500万円
  • D. 340万円
✅ 正解：D（340万円）
差額投資額 = 新トラック400万 − 旧トラック売却60万 = 340万円。旧設備を売って得られるお金を差し引くのがポイント！

基本事項
ある設備投資案のIRR（内部収益率）が12%、会社の資本コストが10%のとき、この投資案をどう判断すべきか。
  • A. IRRが高すぎるので棄却
  • B. IRR ≥ 資本コストなので採用
  • C. 判断できない（NPV法も必要）
  • D. IRR − 資本コスト = 2% は低すぎるので棄却
✅ 正解：B
IRR法では、IRR ≥ 資本コスト（ハードルレート）なら投資すべき。12% ≥ 10% なので採用！IRRは「この投資で実現できる利回り」なので、資本コストを超えていれば価値を生み出せる。

☕ 実例1：スターバックスの日本進出（NPV法の実践）

🇺🇸 → 🇯🇵 1996年 スターバックス日本進出

スターバックスが日本に1号店を出すとき、膨大な初期投資が必要でした。店舗改装費、人材育成、ブランド確立のためのマーケティング費用…。

当時のスタバが考えたことを意思決定会計で読み解くと：

  • 初期投資：店舗開設費用、サザビーとの合弁設立コスト
  • 将来CF：日本のコーヒー市場の成長性、1杯300〜500円の高単価モデル
  • NPV分析：人口密度が高い日本では来客数が見込め、将来CFの現在価値が初期投資を大きく上回ると判断

→ 結果：大成功。日本はスタバにとって米国外で最も成功した市場の一つに。NPV的に大幅プラスの投資判断だったことがわかります。

💡
学びのポイント
「初期投資は大きいけど、将来のCFがそれを上回るか？」がNPV法の本質。スタバは日本市場の将来性を正確に読み、投資に踏み切りました。
⚠️
試験との接点
試験では「初期投資○万円、毎年のCF○万円、資本コスト○%」のようにデータが与えられ、NPVを計算させる問題が定番。スタバの例と同じ思考プロセス！

🏭 実例2：トヨタのEV工場投資（取替投資の発想）

🚗 トヨタのEVシフトと設備投資

トヨタは長年ハイブリッド車で成功してきましたが、世界的なEVシフトの流れを受けて、既存のエンジン工場をEV用バッテリー工場に転換する巨額投資を決定しました。

これはまさに取替投資の判断：

  • 旧設備：エンジン製造ライン（まだ使えるが、将来の需要は先細り）
  • 新設備：EV用バッテリー製造ライン（初期投資は莫大だが将来CFが見込める）
  • 差額CF：EV市場の成長による売上増 − エンジン車の売上減
  • サンクコスト：既存のエンジン技術開発に費やした費用 → これは埋没原価。「もったいない」ではなく「これからどうなるか」で判断

→ 過去の投資額に引きずられず、将来のCFで判断する。まさに「サンクコストを無視する」実例です。

⚠️
試験との接点
取替投資の問題では「旧設備の帳簿価額」「売却額」「新設備の価格」が与えられ、差額投資額と差額CFでNPVを求めさせるパターンが頻出！

📱 実例3：ソフトバンクのARM買収（IRRと機会原価）

💰 2016年 ソフトバンクがARMを約3.3兆円で買収

孫正義氏は「IoT時代にはあらゆるモノにチップが載る。ARMの設計はほぼすべてのスマホに使われている」と将来を見据え、約3.3兆円という巨額買収を実行。

  • IRRの視点：IoTデバイス爆増により、ARMのライセンス収入は年々増加 → 投資のIRRがソフトバンクの資本コストを大きく上回ると判断
  • 機会原価：3.3兆円を別の投資に使っていたら得られたリターン。孫氏はARMが最もリターンが高いと判断

→ 2023年にARMはNASDAQに上場し、時価総額は一時10兆円超え。IRRで見れば大成功の投資判断でした。

💡
学びのポイント
巨額投資の裏には必ず「IRRが資本コストを上回る」という計算がある。同時に「このお金を他に使ったら？」という機会原価の比較もしている。試験では小さな数字で出ますが、考え方はソフトバンクの経営判断と同じです！

🎬 実例4：コンコルドの悲劇（サンクコストの失敗例）

✈️ コンコルド − 超音速旅客機の開発

英仏が共同開発した超音速旅客機コンコルド。開発途中で「採算が合わない」とわかったにもかかわらず、「これまでに莫大な開発費を使ってしまったから」という理由でプロジェクトを中止できませんでした。

  • 投じた開発費：数千億円規模（＝サンクコスト）
  • 冷静な判断：追加投資のCFと将来収益を比較すべきだった
  • 実際：完成しても乗客数が限定的で赤字運航。2003年に退役

→ 「もったいない」が合理的判断を狂わせた典型例。ここから「コンコルドの誤謬（サンクコスト・ファラシー）」という言葉が生まれました。

コンコルドの教訓
どれだけお金を使っていても、過去のコスト（サンクコスト）は意思決定に含めてはいけない。判断材料は「これからのコスト」と「これからの収益」だけ！
👨
試験との接点
「埋没原価（サンクコスト）を意思決定に含めるべきか」という正誤問題が定番。答えは常に「含めない」。「コンコルドの誤謬」「サンクコスト・ファラシー」という用語も要チェック！

🍰 実例5：街のケーキ屋さん（回収期間法）

🎂 個人経営のケーキ屋「シュクレ」の設備判断

オーナーパティシエの田中さんは、新しいオーブン（200万円）を導入するか迷っていました。

  • 現在の売上：月50万円
  • 新オーブンで生産量1.5倍 → 売上増加分：月25万円（年300万円）
  • 追加の材料費・光熱費：年150万円
  • 純増CF：年150万円

回収期間 = 200万 ÷ 150万 = 約1.3年

→ 田中さん「1年ちょっとで元がとれるなら！」と即決。小規模事業者にとって回収期間法はわかりやすく、実務でよく使われます。

👨
先生
中小企業の現場では「何年で元がとれる？」が一番ピンとくる。ただし回収期間法は回収後の収益を無視する点に注意。NPV法と組み合わせるのがベストだよ！
実例から覚える意思決定会計のポイント
• NPV法 → スタバ日本進出（将来CFが投資を上回るか）
• 取替投資 → トヨタEVシフト（差額CFで判断）
• IRR・機会原価 → ソフトバンクARM買収（利回りと他の選択肢）
• サンクコスト → コンコルドの悲劇（過去の費用で判断してはダメ）
• 回収期間法 → 街のケーキ屋さん（わかりやすいが限界もある）

方法判断基準
NPV（正味現在価値）NPV＞0なら採用
IRR（内部収益率）IRR＞資本コストなら採用
回収期間法目標年数以内に回収できるか
現在価値＝将来CF÷（1＋割引率）^年数
NPV＝Σ（CF×複利現価係数）−投資額
✏️ 穴埋め中級｜キャッシュフロー
営業CF＝税引後利益＋減価償却費
タックスシールド＝減価償却費×税率（節税効果）
投資の終了時CF＝処分価値×（1−税率）＋帳簿価額×税率
機会原価：他で得られたはずの利益
サンクコスト：過去に既に支出したコスト → 意思決定に含めない
✏️ 穴埋め上級｜複合判断
NPVとIRRが矛盾するときはNPVを優先（規模を考慮）
取替投資の判断：差額CFでNPVを計算
耐用年数の異なる投資の比較：年金等価額（AE）法を使う
IRRの限界：CFが途中でマイナスになると複数解が生じうる
✏️ 穴埋め通常｜全範囲
NPV採用条件：NPV＞0
IRR採用条件：IRR＞資本コスト
営業CF＝税引後利益＋減価償却費
サンクコスト・機会原価のうち、意思決定に含めるのは機会原価
NPVとIRRの矛盾時に優先するのはNPV
財務・会計 意思決定会計・設備投資の経済性計算 学習まとめ
