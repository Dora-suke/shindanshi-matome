🔢 シナリオ別の期待収益率計算

📐
数学担当
基礎版では「加重平均」で求めたけど、試験ではシナリオ（好景気/普通/不景気）ごとに確率と収益率が与えられることが多い。この計算ができないと始まらないよ。

📝 計算例：シナリオ別の期待収益率

シナリオ確率資産Aの収益率資産Bの収益率
好景気30%20%-5%
普通50%8%10%
不景気20%-10%25%

1 各シナリオの「確率 x 収益率」を計算

A: 0.3x20% + 0.5x8% + 0.2x(-10%) = 6% + 4% + (-2%) = 8%

B: 0.3x(-5%) + 0.5x10% + 0.2x25% = -1.5% + 5% + 5% = 8.5%

2 結果を確認

E(R_A) = 8%　　E(R_B) = 8.5%
💡
ポイント
AとBでは期待収益率はほぼ同じ。でもAは好景気で+20%・不景気で-10%とブレが大きい。Bは不景気で+25%とAと逆に動く。この「逆に動く」が分散効果のカギ。

🔗 共分散 — 相関係数の正体

🧑
受験生
「共分散」ってなに？相関係数と何が違うの？
📐
数学担当
共分散は「2つの資産が一緒にブレる度合い」を数値化したもの。相関係数は共分散を「扱いやすい -1〜+1 の範囲」に変換したものだよ。
Cov(A,B) = Σ [ 確率_i x (R_Ai - E(R_A)) x (R_Bi - E(R_B)) ]
ρ_AB = Cov(A,B) ÷ (σ_A x σ_B)

共分散 ÷（Aのリスク x Bのリスク）= 相関係数

📝 共分散の計算（上の例の続き）

1 各シナリオで「期待値からの差」を出す

シナリオ確率A - E(A)B - E(B)確率 x 差A x 差B
好景気0.320-8 = +12-5-8.5 = -13.50.3 x 12 x (-13.5) = -48.6
普通0.58-8 = 010-8.5 = +1.50.5 x 0 x 1.5 = 0
不景気0.2-10-8 = -1825-8.5 = +16.50.2 x (-18) x 16.5 = -59.4

2 合計する

Cov(A,B) = -48.6 + 0 + (-59.4) = -108

3 結果を読む

共分散がマイナス = AとBは逆方向に動く傾向。組み合わせると分散効果大！

🏠 台所のシンクに共分散の計算表が浮いている
朝起きて台所に行ったら、シンクの水面に3行の計算表がプカプカ浮いていた。
各行には「差A x 差B」とマジックで書いてある。
全部すくい上げて合計したら、それが共分散だった。
— 「差 x 差の合計」= 共分散。変な光景だけど忘れない！

📊 2資産ポートフォリオのリスク計算

👨
先生
試験ではこの計算が毎年のように出る。公式を覚えるだけでなく、実際に手を動かして「どの数字をどこに入れるか」をマスターしよう。

📝 例題：A60%・B40%で組んだポートフォリオのリスク

σ_A = 12%、σ_B = 9%、ρ_AB = -0.5

1 公式に代入

σp² = (0.6)²(12)² + (0.4)²(9)² + 2(0.6)(0.4)(12)(9)(-0.5)

2 各項を計算

第1項: 0.36 x 144 = 51.84

第2項: 0.16 x 81 = 12.96

第3項: 2 x 0.6 x 0.4 x 12 x 9 x (-0.5) = -25.92

3 合計して平方根

σp² = 51.84 + 12.96 + (-25.92) = 38.88 　→　 σp = √38.88 ≈ 6.24%
💡
ポイント
Aのリスク12%、Bのリスク9%。単純な加重平均なら 0.6x12+0.4x9 = 10.8%。でもρ=-0.5のおかげで6.24%まで下がった。これが分散効果の威力！
試験のコツ：第3項の符号に注意。ρがマイナスなら第3項もマイナスになり、リスクが小さくなる。第3項だけ計算して符号を確認すれば、選択肢を絞り込める！

💱 CAPM計算の完全攻略

E(Ri) = Rf + βi x (E(Rm) - Rf)

📝 例題：3銘柄のCAPM計算

Rf = 2%、E(Rm) = 8%（リスクプレミアム = 6%）

銘柄β計算要求収益率
X社（安定企業）0.52% + 0.5 x 6%5%
Y社（成長企業）1.22% + 1.2 x 6%9.2%
Z社（ハイリスク）2.02% + 2.0 x 6%14%
💼
実務家
実務では「この事業の割引率をいくつにするか」でCAPMを使う。新規事業の収益性を評価するとき、その業界平均のβを使ってCAPMで割引率を算出するのが定番。

📉 SML（証券市場線）を読む

SML（証券市場線）のイメージ

縦軸：期待収益率 E(R)　　横軸：β（ベータ）

β=0Rf = 2%
（安全資産）
↗
β=1E(Rm) = 8%
（市場平均）
↗
β=214%
（ハイリスク）

この直線がSML。線の上なら割安、下なら割高。

⭐ α（アルファ）で割安・割高を判断

α = 実際の期待収益率 - CAPMの要求収益率

📝 例題

Y社（β=1.2）の実際の期待収益率が12%のとき

1 CAPMの要求収益率: 2% + 1.2 x 6% = 9.2%

2 α = 12% - 9.2% = +2.8%

3 αがプラス → SMLの上にある → 割安（買い）

α > 0（プラス）

SMLの上
リスクの割にリターンが高い
= 割安（お買い得）

VS

α < 0（マイナス）

SMLの下
リスクの割にリターンが低い
= 割高（避けるべき）

🚿 シャワー室でアルファベットの「α」が浮かんでいる
シャワーを浴びたら、蒸気の中にαの文字が浮かんでいた。
αが上に浮かんでいたら「お買い得（割安）」、
下に沈んでいたら「高すぎ（割高）」。
— 「αが浮いたらBuy、沈んだらBye!」
試験の超頻出パターン：「CAPMで算出した期待収益率は○%である。実際の期待収益率は○%である。この株式は割安か割高か。」→ αを計算するだけ！

📏 シャープレシオ

💼
実務家
投資信託を比較するとき、プロが最初に見る数字がシャープレシオ。「リスク1単位あたり、どれだけ超過リターンを稼いだか」を表すよ。
シャープレシオ = (E(Rp) - Rf) ÷ σp

（ポートフォリオの期待収益率 - 安全資産利回り）÷ ポートフォリオの標準偏差

📝 例題：2つのファンドを比較

Rf = 1%

ファンドE(R)σ計算シャープレシオ
ファンドA10%15%(10-1)÷150.60
ファンドB7%8%(7-1)÷80.75

ファンドAの方がリターンは高いが、リスクあたりの効率ではファンドBが優秀（0.75 > 0.60）。

💡
身近な例え
ラーメン屋のコスパ比較。A店は1,500円で超美味い。B店は800円でかなり美味い。1円あたりの美味しさ（= シャープレシオ）はB店の方が上。リターンだけでなく「効率」で比べるのがプロの視点！
引っかけ注意！「リターンが高い方が良いファンド」はバツ。シャープレシオで比較すべき。試験で「最も効率的なポートフォリオを選べ」と聞かれたらシャープレシオが最大のものを選ぶ！

🔀 トービンの分離定理

🧑
受験生
トービンの分離定理って難しそうな名前だけど、何が「分離」されるの？
👨
先生
実は簡単。「最適なリスク資産の組み合わせ」と「リスクをどれくらいとるか」の2つの判断が分離できるってこと。
トービンの分離定理のイメージ
判断1リスク資産の
最適な組み合わせ
（全員同じ = M）
+
判断2安全資産との配分
（人によって違う）
=
各自の
ポートフォリオCML上の1点

🔑 つまり...

  • リスク資産だけで見た「最強の組み合わせ」は全員共通（市場ポートフォリオM）
  • 慎重な人は「M + 安全資産多め」、攻めたい人は「M + 安全資産少なめ（借入れ）」
  • この2つの判断が独立して決められるのが「分離」の意味

📊 CML vs SML — 混同しない！

CMLとSMLの比較
CML（資本市場線）SML（証券市場線）
横軸σ（標準偏差 = 総リスク）β（ベータ = システマティックリスク）
対象効率的ポートフォリオのみすべての証券・ポートフォリオ
使い方最適な資産配分を決める個別銘柄の割安・割高を判定
傾きシャープレシオマーケットリスクプレミアム
🎢 遊園地の受付に「CML」と「SML」の看板
遊園地の入口に2つの受付がある。
CML受付は「σ（身長）を測って乗れるアトラクションを決める」（= 総リスクで判断）。
SML受付は「β（怖がり度）だけ聞いて全員チェックする」（= 市場リスクだけで全銘柄を判定）。
— CMLは「身長制限（σ）」、SMLは「怖がり度（β）」！
過去問ポイント：「CMLの横軸はβである」→ バツ！CMLはσ。「SMLの横軸はσである」→ バツ！SMLはβ。軸の入れ替えが最頻出引っかけ。

✏️ 本番レベル演習

1
期待収益率
好景気(40%)で+15%、不景気(60%)で-5%の資産の期待収益率は？
  • 5%
  • 3%
  • 4%
  • 6%
✅ 正解：3%
0.4 x 15% + 0.6 x (-5%) = 6% + (-3%) = 3%
2
加重平均
資産A(期待収益率12%)に30%、資産B(期待収益率6%)に70%投資した場合のポートフォリオの期待収益率は？
  • 9.0%
  • 8.4%
  • 7.2%
  • 7.8%
✅ 正解：7.8%
0.3 x 12% + 0.7 x 6% = 3.6% + 4.2% = 7.8%
3
CAPM
Rf=1%、E(Rm)=7%、β=1.5のとき、CAPMによる期待収益率は？
  • 10%
  • 11.5%
  • 9.5%
  • 10.5%
✅ 正解：10%
1% + 1.5 x (7% - 1%) = 1% + 1.5 x 6% = 1% + 9% = 10%
リスクプレミアム = E(Rm) - Rf = 6%。これにβをかけてRfを足す。
4
シャープレシオ
期待収益率10%、標準偏差20%、無リスク利子率2%のポートフォリオのシャープレシオは？
  • 0.50
  • 0.40
  • 0.45
  • 0.35
✅ 正解：0.40
(10% - 2%) ÷ 20% = 8% ÷ 20% = 0.40
5
α判定
CAPM上の要求収益率が8%の銘柄の実際の期待収益率が11%だった場合、この銘柄は？
  • 割高である
  • 適正価格である
  • 割安である
  • 判断できない
✅ 正解：割安である
α = 11% - 8% = +3%。αがプラス = SMLの上 = 割安。リスクに見合う以上のリターンが期待できる！
6
CML vs SML
CML（資本市場線）の横軸に使われる指標はどれか。
  • 標準偏差（σ）
  • ベータ（β）
  • 共分散
  • 分散
✅ 正解：標準偏差（σ）
CML = 横軸σ（総リスク）。SML = 横軸β（システマティックリスク）。入れ替え問題の定番！
7
共分散
2つの資産の共分散が負（マイナス）の場合、これらを組み合わせると何が期待できるか。
  • 期待収益率の上昇
  • ベータ値の低下
  • シャープレシオの一致
  • ポートフォリオのリスク低減
✅ 正解：ポートフォリオのリスク低減
共分散がマイナス = 逆方向に動く。リスク公式の第3項がマイナスになり、ポートフォリオ全体のリスクが下がる。
8
トービン
トービンの分離定理で「全投資家に共通」とされるのは何か。
  • 安全資産の配分比率
  • リスク資産の最適な組み合わせ（市場ポートフォリオ）
  • 投資家のリスク許容度
  • ベータ値
✅ 正解：リスク資産の最適な組み合わせ（市場ポートフォリオ）
分離定理のポイント：リスク資産の最適な組み合わせ（M）は全員同じ。違うのは「安全資産との配分」だけ。
9
リスク計算
σ_A=10%、σ_B=10%、ρ=-1のとき、A50%・B50%のポートフォリオのリスク(σp)は？
  • 0%
  • 5%
  • 10%
  • 7.07%
✅ 正解：0%
σp² = 0.25x100 + 0.25x100 + 2x0.5x0.5x10x10x(-1) = 25 + 25 - 50 = 0。ρ=-1かつ同じ比率・同じσなら理論上リスクゼロ！
10
CAPM応用
Rf=2%、リスクプレミアム=5%。ある事業のβ=0.8のとき、この事業の投資判断に使う割引率は？
  • 4.0%
  • 5.6%
  • 6.0%
  • 6.5%
✅ 正解：6.0%
2% + 0.8 x 5% = 2% + 4% = 6.0%。CAPMで算出した要求収益率をNPV計算の割引率として使う。

⚠️ 引っかけパターン一覧

❌ パターン1：リスクの加重平均

❌ よくある間違い✅ 正しい理解
ポートフォリオのリスクは各資産のリスクの加重平均であるリスクは加重平均より小さくなる（ρ<1のとき）

期待収益率は加重平均だが、リスクは違う。この「非対称」が最大の引っかけ。

❌ パターン2：分散投資で全リスク消滅

❌ よくある間違い✅ 正しい理解
分散投資ですべてのリスクを除去できる除去できるのはアンシステマティック・リスクのみ

システマティック・リスク（市場全体）はどんなに分散しても消えない。

❌ パターン3：CMLとSMLの軸入れ替え

❌ よくある間違い✅ 正しい理解
CMLの横軸はβであるCMLの横軸はσ（標準偏差）
SMLの横軸はσであるSMLの横軸はβ（ベータ）

❌ パターン4：β=0の解釈

❌ よくある間違い✅ 正しい理解
β=0の資産はリスクがないβ=0は市場と連動しないだけ。個別リスクはある

βが測るのはシステマティック・リスクだけ。アンシステマティック・リスクはβには反映されない。

❌ パターン5：シャープレシオの分母

❌ よくある間違い✅ 正しい理解
シャープレシオの分母はβである分母はσ（標準偏差）

βを分母にするのは「トレイナーレシオ」。名前が似ているので注意。

❌ パターン6：CAPM計算ミス

❌ よくある間違い✅ 正しい理解
E(Ri) = β x E(Rm)E(Ri) = Rf + β x (E(Rm) - Rf)

Rfを引き忘れ & Rfを足し忘れ のダブルミス。「Rfを引いてからβをかけ、最後にRfを足す」の手順を死守。

❌ パターン7：相関係数の範囲

❌ よくある間違い✅ 正しい理解
相関係数は0〜+1の範囲相関係数は-1〜+1の範囲

マイナスがあるからこそ分散効果が最大化する。「0〜1」と出題されたらバツ。

❌ パターン8：効率的フロンティアの上下

❌ よくある間違い✅ 正しい理解
効率的フロンティアの下側の曲線も効率的である効率的なのは上側の曲線のみ

同じリスクでリターンが低い下側は非効率。上側だけが「効率的フロンティア」。
