🔢 「加重平均」ってなに？

🧑
新人くん
「加重平均」って難しそう...普通の平均と何が違うの？
👨
先生
普通の平均は「全部同じ重さ」。加重平均は「重要度に差をつけて平均する」ってこと。学校のテストで例えてみよう！
普通の平均 vs 加重平均

普通の平均

国語 80点 + 数学 60点

(80 + 60) ÷ 2 = 70点

2科目を同じ重さで計算

加重平均

国語 80点(配点3) + 数学 60点(配点7)

80x0.3 + 60x0.7 = 66点

数学の方が「重い」ので結果が変わる！

🔤 ギリシャ文字はこわくない！

覚える記号はたった3つ！
σ

あだ名：「ブレブレくん」
ρ
ロー = 相関係数（連動度）
あだ名：「なかよし度」
β
ベータ = 市場との連動度
あだ名：「ゆれゆれ度」

🏫 日常に置き換えると...

記号日常の例え値が大きいと
σ（シグマ）テストの点数のバラつき毎回の点数がバラバラ
ρ（ロー）友達との趣味の一致度好みがまったく同じ
β（ベータ）ジェットコースターの激しさ激しく揺れる（ハイリスク）
⚠️
試験の注意
試験では記号が必ず出る。「σ（シグマ）= 標準偏差 = リスク」「ρ（ロー）= 相関係数」の対応を暗記しておくのが最低ライン！

📮 ポストの中に3匹の猫
玄関のポスト（= ポートフォリオ）を開けたら、中で

β（ベータ猫）がジェットコースターの模型にしがみついていた（= 市場との連動度）。
— 変な光景だけど、3匹の名前と特徴はもう忘れない！

🎯 リターンとリスク

💰 リターン（期待収益率）

「平均でどれくらい儲かるか」
= 各シナリオの収益率 x 確率 の合計

例: 好景気50%で+20%、不景気50%で-4%
→ 0.5x20% + 0.5x(-4%) = 8%

VS

📉 リスク（標準偏差）

「リターンがどれくらいブレるか」
= ブレ幅が大きい ＝ リスクが高い

定期預金はブレ小 → 低リスク
株式はブレ大 → 高リスク

🎬 実例：映画の興行収入。大作映画は大ヒットor大コケ（高リスク）、ドキュメンタリーは安定した小収入（低リスク）。

💡
身近な例え
リターン = お弁当屋さんの「1日の平均売上」
リスク = 日によって売上がどれくらいバラつくか
毎日安定して10万円売れるお店は低リスク。5万〜20万円とバラつくお店は高リスク。
⚠️
試験の大原則
ハイリスク・ハイリターン：リスクが高い資産ほど、見返りとして高いリターンが期待される。逆にローリスクならローリターン。この関係は試験で大前提として問われるよ！
リスクとリターンの関係
🏦 定期預金低リスク・低リターン
→
📄 債券中リスク・中リターン
→
📈 株式高リスク・高リターン

👨

🔑 入門で覚える3つのキーワード

キーワード意味
期待収益率平均でどれくらい儲かるか（リターン）
標準偏差リターンのブレ幅＝リスクの大きさ
相関係数2つの資産の値動きがどれくらい連動するか（-1〜+1）。低いほど分散効果が大きい
試験のツボ：ポートフォリオ理論は「リスクを下げたいなら相関係数が低い資産を組み合わせる」がすべての基本。これだけ覚えれば、応用問題も解ける！
📊 相関係数（ρ）が変わると、リスクがこんなに変わる！

条件：σ_A=12%・σ_B=9%、投資比率 A=60%・B=40%。ρがマイナスになるほどリスクが大幅に下がる！

📊 期待収益率の計算

👨
先生
ポートフォリオの期待収益率は、各資産の期待収益率を投資比率で加重平均するだけ！シンプルだよ。
E(Rp) = w₁ x E(R₁) + w₂ x E(R₂)

w = 投資比率、E(R) = 各資産の期待収益率

📝 計算例

資産A（期待収益率10%）に60%、資産B（期待収益率4%）に40% 投資する場合：

E(Rp) = 0.6 x 10% + 0.4 x 4% = 6% + 1.6% = 7.6%

🍔 実例：ハンバーガー店で「肉60%・パン40%」のバーガーの総カロリーを計算するのと同じ。各素材のカロリーを配分比で加重平均するだけ！

📉 ポートフォリオのリスク（標準偏差）

🧑
受験生
リターンは加重平均でかんたんだけど、リスクは同じじゃないの？
👨
先生
ここがポートフォリオ理論の核心！リスクは単純な加重平均にはならない。相関係数によって分散効果が生まれるから、たいてい加重平均よりリスクは小さくなるんだ。
σp² = w₁²σ₁² + w₂²σ₂² + 2w₁w₂σ₁σ₂ρ₁₂

σ = 標準偏差、ρ = 相関係数

公式の3つのパーツを分解
第1項Aだけのリスク
+
第2項Bだけのリスク
+
第3項AとBの連動分
← ρで変わる！
💡
覚え方のコツ
料理に例えると、第1項=塩の辛さ、第2項=砂糖の甘さ、第3項=混ぜたときの味変。相関係数ρが「味変の強さ」を決める。ρ=-1なら塩と砂糖が打ち消し合ってマイルドに！
試験最頻出ポイント！
この公式の第3項「2w₁w₂σ₁σ₂ρ」がカギ。
• ρ = +1 の場合 → リスクは単純な加重平均（分散効果なし）
• ρ = -1 の場合 → リスクを最小（ゼロ）にできる可能性あり
• ρ が低いほど → 分散効果が大きい

🔗 相関係数をビジュアルで

相関係数と値動きのイメージ
相関係数値動き分散効果身近な例
ρ = +1まったく同じ方向なしトヨタ株とホンダ株
ρ = 0無関係あり自動車株と食品株
ρ = -1まったく逆方向最大傘屋とアイス屋

⚽ 実例：サッカーチームで「攻撃的FW」と「守備的MF」は逆の役割（相関-1に近い）。両方いればチームが安定する。FWばかり（相関+1）だと守備崩壊！

🛁 風呂場に相撲取りと菊の花
お相撲さん（= 相関）が低い天井（= 低い）の風呂場で頭をぶつけ、
驚いて干していた洗濯物をバラバラに散らかした（= 分散）ら、
なぜか浴室中に菊の花（= 効く）の香りが広がった。
— 「相撲が低い天井で散らかして菊の香り」= 相関が低いほど分散が効く！

🎯 効率的フロンティアとCAPM

📈 効率的フロンティア

同じリスクの中で最大のリターンが得られる組み合わせの集合。グラフ上では左上の曲線になる。

= 「これ以上リスクを減らすにはリターンを諦めるしかない」という最適ライン

💡
身近な例え
回転寿司の「満足度（リターン）」と「お会計（リスク）」のバランス。効率的フロンティア＝同じ予算で最も満足できるメニューの組み合わせ。それより上はお金が足りない、下は損してる。

💱 CAPM（資本資産価格モデル）

E(Ri) = Rf + βi x (E(Rm) - Rf)

Rf = リスクフリーレート（安全資産の利回り。例：国債の利回り0.5〜2%程度）
β = ベータ値（市場全体に対する感応度）
E(Rm) - Rf = マーケット・リスクプレミアム（市場全体のリターン - 安全資産）

⚠️
試験の引っかけ
CAPM計算はRf + β x (Rm - Rf) の代入だけ！よくある引っかけは「β x Rm」と間違えること。Rfを引いてからかけるのを忘れずに！
💡
CAPMの身近な例え
遊園地のジェットコースター（= 市場全体）に比べて、β=1.5の株は「1.5倍激しく揺れる乗り物」。揺れが激しい分、スリルの報酬（= リターン）も大きい。β=0.5なら半分しか揺れないメリーゴーラウンド。安定だけどリターンも控えめ。

🛡 システマティック・リスク

市場全体のリスク。分散投資では消せない。

例：景気後退、金利変動、戦争

🎲 アンシステマティック・リスク

個別企業のリスク。分散投資で消せる。

例：不祥事、特定製品の失敗

🏫 実例：学校のテスト。全教科まとめて中止（台風=システマティック）vs 数学だけ追試（自分の体調不良=アンシステマティック）。他の教科を頑張れば補える！

リスクの分類ツリー
📊 投資の総リスク
⬇
システマティック✖ 分散で消せない
⬇
アンシステマティック○ 分散で消せる
💡
身近な例え
学校で例えると、システマティック=台風で学校ごと休校（全員に影響、避けられない）。アンシステマティック=自分だけ風邪で欠席（個人の問題、予防接種=分散で対策可能）。
試験頻出ポイント
• 「分散投資で除去できるのはアンシステマティック・リスクのみ」
• CAPMで評価されるのはシステマティック・リスク（β）だけ
• β=1 は市場平均と同じ動き
📈 効率的フロンティアとCML（資本市場線）

CMLはRf（安全資産）と市場ポートフォリオMを結ぶ直線。効率的フロンティアの上側のみが「最適な組み合わせ」。

📊 ベータ（β）を小学生でもわかるひと言で

「クラス全体のテストの点が1点上がったとき、この人は何点上がるか」を表す数字。
クラス平均が1点上がったとき、Aくんは2点上がった → Aくんのベータ（β）＝ 2
クラス平均が1点上がったとき、Bさんは0.5点しか動かなかった → Bさんのベータ（β）＝ 0.5

※「クラス全体」＝「株式市場全体」に置き換えれば投資の話になる

📝 式の意味 ― 「一緒に動く度合い ÷ 市場の揺れ幅」

🧑
新人くん
式で書くと「ベータ（β）＝共分散（Cov）÷市場分散（Var）」ってなってますが…
👨
先生
難しく見えるけど、意味をひと言で言うとこうなる：

共分散（Cov）＝「この株と市場が、どれだけ一緒に動くかを数字にしたもの」
市場分散（Var）＝「市場全体がどれだけ揺れるかを数字にしたもの」

つまり式の意味は：
「一緒に動く度合い ÷ 市場の揺れ幅」＝ 市場への乗っかり具合＝ベータ（β）！
ベータ（β） ＝ 一緒に動く度合い（共分散, Cov）市場全体の揺れ幅（市場分散, Var）

「市場と同じくらい一緒に動く（Cov↑）ほど、β は大きくなる」

🧮 計算で問われるパターン（2種類）

パターンA：数字をそのまま割り算する（H21・H26型）　← 3〜4年に1回だけ

ベータ（β） = 共分散（Cov）市場分散（Var）

例題）数字を代入してそのまま割るだけ

問題に書いてある数字計算意味
共分散（Cov） = 0.006
市場分散（Var） = 0.004		β = 0.006 ÷ 0.004 = 1.5市場の1.5倍動く株

パターンB：「儲け」の式からベータ（β）を逆算する（H29型） ← こちらが重要！

CAPMという式（基本タブ参照）をベータ（β）について解き直すだけ：

もらえる儲けの割合（期待収益率）＝
　　「確実にもらえる分（銀行預金の利息など）」
　　＋ ベータ（β）×「リスクを取った分の上乗せ（市場リスクプレミアム）」

↓ この式をベータ（β）について解き直すと

ベータ（β） = 儲けの割合 − 確実にもらえる分（Rf）リスクを取った分の上乗せ（市場リスクプレミアム）
例題）次の情報からベータ（β）を求めよ
・儲けの割合（期待収益率）= 8%
・確実にもらえる分（Rf）= 2%　※銀行預金の利息のようなもの
・市場リスクプレミアム = 4%　※リスクを取った分のボーナス

ステップ① 「儲け − 確実にもらえる分」を計算する
　8% − 2% ＝ 6%

ステップ② 「市場リスクプレミアム」で割る
　6% ÷ 4% ＝ 1.5

答え：ベータ（β）＝ 1.5
→ この株は市場全体の1.5倍動く（やや値動きが激しい株）

⚠ 試験での注意点（3つ）

注意①「割り算の下（分母）」は市場全体の揺れ幅。個別の株の揺れ幅を入れない。
ベータ（β）の式の下（分母）に入るのは「市場全体の揺れ幅（市場分散, Var）」。
「この株単独の揺れ幅」を間違えて入れるミスが多い。
→ 分母はいつも「市場全体」の数字！
注意②「共分散（Cov）」は別の形でも書ける。
共分散（Cov）= この株の揺れ幅 × 市場の揺れ幅 × 相関係数（ρ）
→ 「揺れ幅」と「どれだけ一緒に動くかの数（相関係数）」が与えられたときに使う変換。
注意③ ベータ（β）は「株をたくさん持っても消えないリスク」の大きさを表す。
たとえば100種類の株を持っていても、日本全体で不景気になれば全部一緒に下がる。
そのリスクの大きさをベータ（β）で測る。
「1つの会社だけの失敗」とは違い、分けて持っても消せない。

👨

相関係数（ρ）の値とリスク低減効果まとめ
相関係数（ρ）意味リスク低減の効果お店の例え

★★★ 最頻出：銀行預金と株を組み合わせるパターン（H23・H24・H30・R02型）

🧑
新人くん
「銀行預金と株を組み合わせる」とどうして式が簡単になるんですか？
👨
先生
銀行預金は「絶対にブレない」よね？つまり揺れ幅（σ）＝ 0。
さっきの式①＋②＋③で「銀行預金の揺れ幅＝0」を代入すると、②と③がゼロになって消える。
残るのは①だけ！「株への投資比率 × 株の揺れ幅」のかけ算1回で終わり！
式が消える過程を全部見せる
まず「株Bが銀行預金」なので → 銀行預金の揺れ幅（σ）＝ 0 を代入する
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
① 株Aだけのリスク（w₁²σ₁²）
　= 株Aへの投資比率² × 株Aの揺れ幅²
　→ そのまま残る ✅

② 株Bだけのリスク（w₂²σ₂²）
　= 銀行預金への投資比率² × 0²
　= 何かの数 × 0 ＝ 0（消える！）❌

③ 連動部分（2w₁w₂σ₁σ₂ρ）
　= 2 × 比率A × 比率B × 揺れ幅A × 0 × 相関係数
　= 何かの数 × 0 ＝ 0（消える！）❌
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
残った①の√を取ると：
全体の揺れ幅（σp）＝ 株への投資比率（w） × 株の揺れ幅（σ）
例題）株に60%・銀行預金に40%投資。株の揺れ幅（σ）＝ 20%
ステップ① 「株への投資比率（w）」を確認する
　→ 60% ＝ 0.6
ステップ② 「株の揺れ幅（σ）」を確認する
　→ 20%
ステップ③ かけ算する（これだけ！）
　→ 0.6 × 20% ＝ 12%
ステップ④ 答えの確認
　→ 全体のポートフォリオの揺れ幅（リスク）＝ 12%

★★ 共分散（Cov）と相関係数（ρ）を計算するパターン（H28問15-1型）

🧑
新人くん
このパターンはどんな問題ですか？
👨
先生
「景気が良い・普通・悪い」の3シナリオで株AとBのリターンが表で与えられて、
「2つの株がどれだけ一緒に動くか（共分散）」や「相関係数（ρ）」を求める問題。
3ステップで進めれば必ず解ける！
【ステップ1】各株の「平均のもうけ（期待収益率）」を求める
やること：「確率 × もうけの割合」を全シナリオ分足す
例）景気良い（確率30%・もうけ20%）＋ 普通（確率40%・もうけ10%）＋ 景気悪い（確率30%・もうけ0%）
→ 0.3×20% ＋ 0.4×10% ＋ 0.3×0% ＝ 6% ＋ 4% ＋ 0% ＝ 10%
※ 株AとBで同じ手順を繰り返す

【ステップ2】共分散（Cov）を求める
やること：各シナリオで「AのブレとBのブレを掛けて、確率を掛けて」全部足す
「ブレ」＝「実際のもうけ − ステップ1で求めた平均のもうけ」
共分散（Cov）＝ Σ（確率 × Aのブレ × Bのブレ）
↑「ブレが同じ方向なら正、逆方向なら負になる」←これが傘屋の話と同じ！

【ステップ3】相関係数（ρ）を求める
やること：共分散（Cov）を「両方の揺れ幅の積」で割る
相関係数（ρ）＝ 共分散（Cov）株Aの揺れ幅（σA）× 株Bの揺れ幅（σB）
※ 揺れ幅（σ）＝ √（各株の分散）で求める

△ フルの公式計算（R04型）：難問。深追い不要

2つの株の揺れ幅（σ）・投資比率（w）・相関係数（ρ）が全部与えられ、公式①＋②＋③に全部代入して計算後に√を取る問題。
出題は稀。上の★★★と★★をしっかり取れれば十分。

⚠ ステップ④ 最後に「落とし穴」を確認する（注意点3つ）

👨
先生
ここまで来たら、あとは「試験でミスしやすい落とし穴」を覚えるだけ。
計算の流れはわかったのに、これで点を落とすのはもったいない！
落とし穴①「分散（σp²）」と「揺れ幅・標準偏差（σp）」を混同する
公式①＋②＋③で求まるのは「分散（揺れ幅の2乗）」。
「標準偏差（揺れ幅）を求めよ」と言われたら、最後に√を取らないといけない。
→ 問題文をよく読んで、「分散」か「標準偏差」かを必ず確認！
落とし穴②「共分散（Cov）」と「相関係数（ρ）」は全然違う数
この2つは別物だが、変換できる：
共分散（Cov）＝ 揺れ幅A × 揺れ幅B × 相関係数（ρ）
→ つまり：相関係数（ρ）＝ 共分散（Cov）揺れ幅A（σ₁）× 揺れ幅B（σ₂）
「共分散と揺れ幅が与えられたら相関係数を計算できる」と覚えておく。
落とし穴③ 銀行預金との組み合わせ問題は「かけ算1回」に気づかず迷子になる
銀行預金の揺れ幅（σ）＝ 0 を代入すれば式がほぼ消える。
「全体の揺れ幅（σp）＝ 株への投資比率（w）× 株の揺れ幅（σ）」の1行で答えが出る。
H23・H24・H30・R02型すべてこの形。複雑に考えすぎない！

基本
資産A（期待収益率8%）に70%、資産B（期待収益率2%）に30%投資した場合、ポートフォリオ全体の期待収益率は？
  • 6.2%

  • 7.0%
  • 4.6%

0.7 x 8% + 0.3 x 2% = 5.6% + 0.6% = 6.2%。ポートフォリオの期待収益率は各資産の加重平均！

相関係数
海水浴場の売店（夏に儲かる）とスキー場のレストラン（冬に儲かる）。この2つの事業の相関係数に最も近いのは？
  • +1
  • +0.5
  • 0
  • -1に近い
✅ 正解：-1に近い
夏に儲かる事業と冬に儲かる事業は逆の値動き。まさに傘屋とアイス屋の関係と同じで、相関係数は-1に近い。組み合わせると年間通して安定する！

分散投資
分散投資によって除去できるリスクはどれ？
  • システマティック・リスク
  • アンシステマティック・リスク
  • マーケット・リスク
  • すべてのリスク
✅ 正解：アンシステマティック・リスク
個別企業のリスク（不祥事、製品の失敗など）は分散で除去可能。一方、市場全体のリスク（景気後退など）= システマティック・リスクは分散しても消せない！

CAPM
あなたの会社のベータ値は1.2。リスクフリーレートが2%、マーケットリスクプレミアムが5%のとき、CAPMによる要求収益率は？
  • 8.0%
  • 7.0%
  • 8.5%
  • 6.0%
✅ 正解：8.0%
E(Ri) = Rf + β x (Rm - Rf) = 2% + 1.2 x 5% = 2% + 6% = 8.0%。ベータが1より大きいので、市場平均より高いリターンが要求される！

効率的フロンティア
効率的フロンティア上のポートフォリオの特徴として正しいものは？
  • リスクがゼロのポートフォリオである
  • リターンが最大のポートフォリオである
  • すべての投資家が選ぶべきポートフォリオである
  • 同じリスクの中でリターンが最大の組み合わせである
✅ 正解：同じリスクの中でリターンが最大の組み合わせである
効率的フロンティアは「同じリスク水準で最もリターンが高い」（= 同じリターンで最もリスクが低い）ポートフォリオの集合。リスクゼロやリターン最大とは限らない。

応用
地元のパン屋さんが、カフェ併設とネット通販の2つの新事業を検討中。ポートフォリオ理論の観点から最もよいアドバイスは？
  • リターンが最も高い事業だけに集中投資すべき
  • 両方やるとコストが倍になるのでどちらか一方にすべき
  • 売上の相関が低い事業を組み合わせるとリスクが下がる
  • 新事業はリスクが高いのでどちらもやめるべき
✅ 正解：売上の相関が低い事業を組み合わせるとリスクが下がる
カフェ（立地・天候に依存）とネット通販（立地に依存しない）は相関が低い可能性が高い。ポートフォリオ理論では、相関の低い資産を組み合わせることでリスクを低減できる！
8
CAPM
β（ベータ値）が1.0の株式について正しいものは？
  • 市場全体と同じ動きをする
  • 市場より2倍激しく動く
  • リスクフリーレートと同じリターンになる
  • 分散投資の効果がない
✅ 正解：市場全体と同じ動きをする
β=1.0は「市場平均と同じ感応度」。市場が10%上がればこの株も約10%上がり、10%下がれば約10%下がる。β>1なら市場より激しく、β<1なら市場より穏やか。

🏦 GPIF（年金積立金管理運用独立行政法人）

🌏 世界最大級の機関投資家による分散投資

私たちの年金を運用しているGPIF。運用資産は約200兆円超（2024年時点）。世界最大級の機関投資家です。

📊 GPIFの基本ポートフォリオ（2020年〜）

資産クラス配分比率特徴
国内債券25%低リスク・低リターン
外国債券25%為替リスクあり・やや高リターン
国内株式25%中リスク・中リターン
外国株式25%高リスク・高リターン
💡
ポートフォリオ理論との関連
4つの資産クラスに均等に分散 = 相関の低い資産を組み合わせてリスクを抑制。国内・海外、債券・株式で値動きが異なるからこそ分散効果が働く。まさにポートフォリオ理論の教科書的な実践例！

GPIFは2001年の運用開始以来、年平均約3.9%（2024年3月末時点）のリターンを達成。「分散投資は長期で力を発揮する」ことを証明しています。

👨
先生
定期預金の利率が0.01%だった時代と比べると、年3.9%は約390倍の差。分散投資の威力がよくわかるね。コンビニで例えると、おにぎりだけ売る店 vs おにぎり・飲料・デザート・日用品を売る店の安定感の違い！

📱 ソフトバンクグループのビジョン・ファンド

🚀 集中投資のリスクを体現した事例

孫正義氏率いるソフトバンクグループは、2017年に約10兆円規模のビジョン・ファンドを設立。AI・テクノロジー企業に集中投資しました。

✅ 成功例

  • Armの上場で巨額の含み益
  • Coupang（韓国EC大手）の成長

❌ 失敗例

  • WeWork：上場延期→経営危機で大損
  • 複数のスタートアップが破綻
⚠️
ポートフォリオ理論の教訓
テクノロジー企業ばかりに投資 = 相関の高い資産に集中。業界全体が不調になると一気に損失が膨らむ。アンシステマティック・リスクは分散できても、テクノロジーセクター全体のリスク（システマティック・リスクに近い）は残るという好例。

わかりやすく言うと
学校で例えると「友達全員サッカー部」なら、雨が降ったら全員練習できない。サッカー部・吹奏楽部・将棋部に分かれていれば、雨でも誰かは活動できる。これが業種分散の考え方！

🏪 イオンの事業ポートフォリオ

🛒 小売業の分散戦略

イオンは総合スーパー（GMS）だけでなく、金融・不動産・ヘルス&ウェルネスなど多角的に事業展開しています。

📊 イオンの事業セグメント

セグメント内容景気感応度
GMS（総合スーパー）食品・衣料・日用品低（生活必需品）
ディベロッパーイオンモール運営中
金融イオン銀行・イオンカード中
ヘルス&ウェルネスドラッグストア（ウエルシア）低

🏫 実例：学校の部活動が「運動部だけ」vs「運動部+文化部+帰宅部」。雨天で運動部が活動停止しても、文化部は平常運転。事業セグメントの分散と同じ！

💡
ポートフォリオ理論で読み解く
食品スーパーは不景気でも安定（低β）。金融事業は景気に連動しやすい（高β）。βの異なる事業を組み合わせることで、グループ全体のリスクを抑制。中小企業でも「本業＋副業」で同じ発想が使える！

👨‍🍳 中小企業の身近な成功事例

🍣 地方の寿司チェーン「くら寿司」の多角化

回転寿司チェーンのくら寿司は、国内だけでなく海外（アメリカ・台湾）にも積極展開。国内の少子高齢化リスクを、成長する海外市場で分散しています。

👨
診断士の視点
国内市場と海外市場の景気は完全には連動しない（相関が低い）。為替リスクは増えるものの、売上全体のブレ幅（リスク）を抑える効果がある。まさに「地理的分散」というポートフォリオ戦略！
