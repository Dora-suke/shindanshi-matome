# 学習データ同期メモ

## 評定〜push までの流れ

1. ブラウザで `11.トランザクション管理まとめ_学習システム版.html` を開く
2. ◯/△/✗ を押して評定（localStorage に保存される）
3. ノート末尾の「📥 エクスポート」で learning-data.json をダウンロード
4. ダウンロードしたJSONを `106_経営情報システム/` 直下に上書き保存
5. Claudeに「push して」と頼む → Claude が git add/commit/push を実行

## 復元・別端末で開く時

- ページロード時に同ディレクトリの learning-data.json を自動 fetch
- localStorage が空なら取り込み、データありなら新しい方を採用
- 強制上書きは「📤 インポート」ボタンから手動可能

## 注意

- localStorage は GitHub には行かない（ブラウザ内のみ）
- 月1回以上はエクスポート→pushしてバックアップ推奨
- iPhone等別端末で評定するとローカルは別管理になる（pull忘れ注意）

## 関連ファイル

- `11.トランザクション管理まとめ_学習システム版.html` — 試作版（Phase1実装済み）
- `learning-data.json` — 同期データ本体（このフォルダ直下に置く）
