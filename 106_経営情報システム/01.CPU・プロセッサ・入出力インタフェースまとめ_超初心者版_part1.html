🏭 コンピュータの5大装置（教科書最重要の枠組み）

💡
教科書より
コンピュータのハードウェアには 5種類の装置 があります。これを5大装置と呼びます。
入力装置・出力装置・記憶装置・演算装置・制御装置 の5つ。

🧠 CPU（Central Processing Unit：中央処理装置）

💡
教科書より
CPU はコンピュータの頭脳にあたる部分で、制御機能と演算機能から構成されます（中央演算処理装置とも呼ばれます）。
CPUは記憶装置から命令（プログラム）を読み込み、その命令を実行します。命令には色々な種類がありますが、単純な命令を繰り返すことで複雑な処理を実現します。

📋 レジスタ（教科書頻出：H22-2／H18-1）

📝 レジスタ＝CPU内の超高速・超小容量メモリ

CPUでデータの演算を行うときは、データを一時的に置いておく場所が必要です。これがレジスタ。CPUの中にある、非常に高速で小さいメモリです。

処理の流れ：

  1. 主記憶装置（メモリ）からレジスタにデータを取ってくる
  2. 演算装置で計算する
  3. 結果をレジスタに書き込む
  4. 結果を主記憶装置に書き戻す

🔢 レジスタのビット数

現在のパソコンは32ビットや64ビットのレジスタを持つCPUが多い。32ビットなら2進数で32桁を同時に演算できる、ということ。

レジスタの容量が大きいほど、一度に計算できるデータ量が多くなる→高速。

補足：8ビットで英数字1文字、32ビットで英数字4文字分。レジスタは非常に小容量。

📝 【1次】試験ポイント：H22-2、H18-1 でレジスタが出題。「CPU内の高速・小容量メモリ」「演算用にデータを一時的に置く場所」が答えられればOK。

レジスタ＝CPU内の作業机超高速・超小容量
👨‍🏫
先生からひとことレジスタって聞き慣れないよね。「CPUが計算するとき、目の前に一時的に置く小さなメモ用紙」みたいなもの。机（主記憶）から書類を取ってきて、メモ用紙（レジスタ）の上で計算して、また机に戻す、って繰り返してる！

🎵 クロック周波数（教科書頻出：H29-1／H24-3／H22-2）

📊 クロック周波数＝CPUの動作リズム

CPUは一定のリズムで演算を処理しています。このリズムをクロック周波数と呼びます（メトロノームのように一定のリズムを刻む）。

クロック周波数が大きいほど、CPUは高速に動作します。だからクロック周波数は CPUの性能の基準 として使われます。

単位はヘルツ（Hz）＝1秒間に実行される演算の数。
例：1GHz＝1秒間に10億回の演算（1G = 10の9乗）。

クロック＝メトロノーム動作リズム1GHz＝1秒に10億回性能の基準

📝 【1次】試験ポイント：クロック周波数は3年以上の頻度で出題（H29-1、H24-3、H22-2）。「CPUの動作リズム」「単位はヘルツ（Hz）」「大きいほど高速」が押さえどころ。

👨‍🏫
先生からひとこと「3.5GHzのCPU」って書いてあったら、「1秒間に35億回も演算できるんだ！」って思おう。それくらい速いから、私たちが「クリックしたらすぐ動く」体験ができるわけ！

🖼️ GPU・GPGPU（教科書補足）

📷 GPU（グラフィックボード）

GPU（Graphics Processing Unit：画像処理装置）

ビデオカードにおいて画像処理を行う半導体チップ。最大の特徴は並列処理能力に優れ、複数のタスクを一度にこなせること。単純な計算を並列処理するのが得意。

CPUもGPUもどちらも処理装置ですが、CPUの目的は汎用的な処理、GPUの目的は高速な画像処理。

GPGPU（General-Purpose computing on GPU）

本来画像処理用のGPUを画像処理以外の汎用計算にも利用する技術。並列処理を活かした高速計算が特徴で、画像・動画編集、ゲーム、AI処理など大規模データ・計算量の多いタスクで効果を発揮。

CPU＝汎用万能型GPU＝画像処理特化並列処理GPGPU＝GPUの汎用計算利用AI・動画

🔌 入出力装置とインタフェース

💡
教科書より
入出力装置には、キーボード、マウス、ディスプレイ、プリンタなど様々なものがあります。こういった周辺機器とコンピュータをつなぐ仕組みをインタフェースと呼びます。
インタフェースの種類によってコネクタのピン数などが決まっており、機器同士を接続するには双方が同じインタフェースに対応している必要があります。

📱 入出力装置の種類（教科書頻出：R6-1／R1-1／H29-3 ほか）

🖼️ イメージスキャナ

画像や文書をデジタル画像としてデータ化するための機器。一般にはスキャナと呼ばれる。

読み取った画像は点の集まりとして表現され、その細かさはdpi（dot per inch：1インチあたりの点の数）で表される。dpiが高いほど解像度が高い。

  • フラットベッドスキャナ：ガラス面に置いて、光を当てて読み取る
  • ハンディスキャナ：機器を手で持ちながらスキャン

📱 タッチパネル（3方式・教科書最重要）

ディスプレイを指やペンで直接触れて操作する入力装置。スマホ・タブレットの代表的な入力方法。
3方式があり、それぞれ特徴が異なる：

方式仕組み特徴・用途
静電容量方式指先の微弱な電流を感知軽く触れるだけで操作可、マルチタッチ対応。スマホで主流
赤外線方式赤外線で位置特定画面に直接触れずに操作可、複数点同時入力可。耐久性高、ATM・券売機
抵抗膜方式（感圧式）画面の圧力を検出指・ペン・手袋でもOK。複数点認識は不可
静電容量＝電流スマホ赤外線＝光ATM・券売機抵抗膜＝圧力手袋でもOK

📝 【1次】試験ポイント：タッチパネル3方式は頻出。「スマホは静電容量／券売機は赤外線／手袋OKは抵抗膜」のセット暗記。

👨‍🏫
先生からひとこと3つの方式、混同しがち！ 「スマホ＝静電容量」「冬手袋で操作したいなら抵抗膜」「銀行ATMやコンビニ券売機＝赤外線」って自分の体験で結びつけよう。覚えやすいよ！

🖨️ プリンタ

  • レーザープリンタ：レーザー光でドラム（感光体）にトナーを付着→紙に押し付けて印刷
  • インクジェットプリンタ：インクの細かい粒子を紙に吹き付けて印刷

プリンタの印字精度もdpiで表現される（dpi が高いほど解像度が高い）。

🔀 インタフェースの分類：シリアル vs パラレル

💡
教科書より
インタフェースは大きく シリアル伝送 と パラレル伝送 に分類されます。

🔗 シリアル伝送

1本の信号線でつなぐ。データを順次転送する。

代表例：USB、Bluetooth

🔗🔗 パラレル伝送

複数の信号線でつなぐ。データを並列に転送。一度に送れるデータ量が多い。

代表例：SCSI（旧式）

シリアル＝1本ずつ順次USB・Bluetoothパラレル＝複数本同時SCSI（旧式）
👨‍🏫
先生からひとこと「複数本の方が速そう」って思うよね？ でも現代の主流はシリアル（USB）！ 理由は、複数本だと線同士で干渉して逆に速度が頭打ちになっちゃうから。シンプルに1本を高速化する方が今は速い、ってわけ。

🔌 USB（教科書頻出：R4-1／R3-1／R2-1 ほか）

📷 USB Type-A／B／C

📝 USB（Universal Serial Bus：汎用シリアルバス）

現在最も広く使われているインタフェースの1つ。マウスやキーボード、プリンタ、外付けHDD、USBメモリなど幅広い周辺機器の接続に利用。

主な特長：

  • ホットプラグに対応：電源を入れたままでも機器を接続できる
  • USBハブを使えば、最大127台までの周辺機器を接続可能
  • 上位互換性：異なる世代の機器でも接続可（ただし通信速度は古い規格に合わせる）

🚀 USBの規格と転送速度（教科書3区分）

規格名最大転送速度概要
USB 2.0約 480Mbps現在も多くの機器で利用
USB 3.x最大 20GbpsUSB2.0の約10倍以上の速度
USB4 Version 2.0最大 80Gbps高速データ転送や映像出力にも対応
USB2.0＝480Mbps定番USB3.x＝最大20Gbps10倍以上USB4 V2.0＝最大80Gbps映像も

📝 【1次】試験ポイント：教科書では3区分。USB2.0=480Mbps、USB3.x=最大20Gbps、USB4 V2.0=最大80Gbpsを押さえる。

🔌 USBコネクタの形状（Type-A／B／C）

  • Type-A：従来のパソコンに多い長方形端子。上下の向きが決まっている
  • Type-B：プリンタやスキャナなど周辺機器でよく使う端子
  • Type-C：近年主流の小型コネクタ。上下どちらの向きでも差し込めるリバーシブル構造。スマホ・ノートPCで広く採用

⚡ USB Power Delivery（USB PD）

USB Type-Cで採用されている電力供給規格。従来のUSBより高出力で、最大240ワットまで電力供給可能。

これにより、スマホだけでなくノートパソコンの充電やモニタへの電源供給にも使える。

※ ただし、すべてのType-CポートがPD対応というわけではない。

Type-C＝上下OKリバーシブルUSB PD＝最大240WノートPCも充電

📶 Bluetooth（教科書頻出：R2-1／H25-1）

📶 Bluetooth（ブルートゥース）

ケーブルを使わずにデータをやり取りできる近距離無線通信の規格。マウス、キーボード、イヤホン、スマホなどの機器同士を、10メートル程度の範囲で接続できる。

2.4GHz帯の電波を利用。この周波数帯は電子レンジなどでも使われるため電波干渉が起きやすい。そこで 周波数ホッピング（通信中に周波数を短い間隔で切り替える技術）で干渉を避け安定通信を実現。

通信速度はUSBより遅いが、低消費電力でモバイル機器・ワイヤレス周辺機器を中心に幅広く利用。

Bluetooth＝10m近距離無線2.4GHz帯＋周波数ホッピング干渉対策

CPU内にある、演算用のデータを一時的に置く高速・小容量メモリはどれか。
  • A. 主記憶装置（メモリ）
  • B. レジスタ
  • C. ハードディスク
  • D. SSD
正解B。レジスタ＝CPU内の超高速・超小容量メモリ。32／64ビットが主流。
3
スマートフォンで主流のタッチパネル方式はどれか。
  • A. 抵抗膜方式
  • B. 静電容量方式
  • C. 赤外線方式
  • D. 超音波方式
正解B。静電容量方式＝指の電流を感知。マルチタッチ対応。赤外線はATM・券売機、抵抗膜は手袋でも操作可。
4
USBの特徴として誤っているものはどれか。
  • A. ホットプラグに対応している
  • B. USBハブで複数機器を接続できる
  • C. 異なる世代の機器は接続できない
  • D. シリアル伝送方式である
正解C（誤り）。USBは上位互換性があり、異なる世代でも接続可（速度は古い規格に合わせる）。
5
USB Type-Cで採用される、最大240Wまでの電力供給規格はどれか。
  • A. USB Hub
  • B. USB Power Delivery（USB PD）
  • C. USB Boost
  • D. USB Charge
正解B。USB PD＝Type-Cの給電規格。最大240Wまで対応。ノートPCの充電も可能。

① 5大装置
・5大装置：入力／出力／記憶／演算／制御

② CPU
・CPU内の高速メモリ＝レジスタ（32／64ビット）
・CPU動作リズム＝クロック周波数（単位Hz）。1GHz＝10億回/秒
・画像処理特化＝GPU／GPUを汎用計算に＝GPGPU
③ 入出力装置
・スキャナ・プリンタの解像度単位＝dpi
・タッチパネル3方式：静電容量（スマホ）／赤外線（ATM）／抵抗膜（手袋OK）
・プリンタ2種：レーザー（トナー）／インクジェット（インク粒子）
④ インタフェース
・1本順次＝シリアル／複数本並列＝パラレル／現代主流＝シリアル

・リバーシブル＝Type-C／給電規格＝USB PD（最大240W）
