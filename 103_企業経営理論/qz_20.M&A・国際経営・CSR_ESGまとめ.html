🧪 クイズ：M&A手法
1M&A手法
事業譲渡に関する記述として、最も適切なものはどれか。
  • A. 会社全体の法人格を取得するため、簿外債務も引き継ぐ
  • B. 特定の事業だけを買収できるため、簿外債務リスクを低く抑えられる
  • C. 株式取得と同様に、対象会社の全ての権利義務を包括的に承継する
  • D. 買収後は対象会社を子会社として連結財務諸表に取り込む
✅ 正解：B。事業譲渡は「事業」を買うため、引き継ぐ権利義務は契約で明示したものだけ。簿外債務リスクが低い点が最大のメリット。
❌ A・C・Dは株式取得の説明。事業譲渡と株式取得の違いを明確に区別しよう。
✏️ 穴埋めチェック：M&A手法
🏢 M&A手法の基本
1. 会社全体の株式を取得して子会社化する手法を株式取得という。
2. 特定の事業だけを買う手法を事業譲渡という。
3. 経営陣が自社を買収することをMBOという。
4. 買収価格と純資産の差額をのれんという。
5. 日本基準でのれんは20年以内に償却する。

1のれん

  • A. 日本基準は20年以内に定期償却。IFRSは定期償却せず減損テストのみ
  • B. 日本基準・IFRSともに10年以内に定期償却する

3. LBOとは借入を多用して買収し、買収後のキャッシュフローで返済する手法。

優位性内容例
O（所有優位）
Ownership		自社固有の強み（技術・ブランド等）特許技術、強いブランド力
L（立地優位）
Location		進出先の魅力（コスト・市場等）安い労働力、大きな市場
I（内部化優位）
Internalization		市場取引より自社内部でやる方が得技術流出リスクの回避
3つ揃う場合 → 海外直接投資（FDI）が最適。揃わない場合 → 輸出・ライセンスにとどまる。
1次OLIの各英語の意味と日本語のセットは頻出。「I＝内部化（技術流出防止）」を特に注意。
💡 かんたん解説

OLI理論は「海外に直接投資するには3つの条件が揃わないといけない」という理論です。O＝自社の強み、L＝進出先の魅力、I＝技術が流出しないよう自社内でやる理由、の3つが揃って初めてFDIが最善策になります。

アップサラモデル（段階的国際化）

  • 企業は学習しながら段階的に国際化する
  • 心理的距離が近い国（言語・文化が似た国）から先に進出

2. 心理的距離の近い国から段階的に国際化するプロセス理論＝アップサラモデル
3. OEM → ODM → OBM の発展パターン
🌱 ③ ステークホルダー・ガバナンス
1. フリードマンの主張＝株主利益の最大化が企業唯一の責任
2. フリーマンの主張＝ステークホルダー全体への責任
3. 機関投資家が企業経営に積極関与する行動原則＝スチュワードシップ・コード
✏️ 穴埋め③（上級）
🏢 ① M&A横断整理
1. 吸収合併では消滅会社の法人格が消え、存続会社が引き継ぐ
2. LBOは借入を多用して買収し、買収後のキャッシュフローで返済
3. 事業譲渡と株式取得の違い：事業譲渡は特定の事業だけ買う
4. JVは新しい法人が生まれる。アライアンスは法人をつくらない
🌏 ② 国際経営横断整理
1. グローバル統合高・現地適応低の戦略＝グローバル戦略
2. 各国で独自製品・現地ニーズ完全対応＝マルチドメスティック戦略
3. 本国のやり方を海外移転・知識活用型＝インターナショナル戦略
4. FDIが最適なのはOLIの3つ全てが揃うとき
🌿 ③ CSR・ESG横断整理
1. ESGのE/S/G＝環境/社会/統治
2. 指名委員会等設置会社の3委員会＝指名/報酬/監査の各委員会
3. SDGsは2030年までの17の国際目標
4. CGコードは社外取締役原則2名以上を推奨
📝 穴埋め（全問）
🏢 M&A基本用語
1. 会社全体の株式を取得して子会社化する手法＝株式取得
2. 特定の事業だけを買う手法＝事業譲渡
3. 経営陣が自社を買収することをMBOという
4. 買収価格と純資産の差額をのれんという
5. 日本基準でのれんは20年以内に償却
🏰 買収防衛策と会計
1. 持株比率を希薄化する防衛策＝ポイズンピル
2. 友好的な第三者に買収を依頼する防衛策＝ホワイトナイト
3. 優良資産を売却して買収の魅力を下げる防衛策＝クラウンジュエル
4. 重要事項に拒否権を持つ特別株式＝黄金株
5. IFRSでのれんは償却なし（減損テストのみ）
💰 M&A横断整理
1. 吸収合併では消滅会社の法人格が消え、存続会社が引き継ぐ
2. LBOは借入を多用して買収し、買収後のキャッシュフローで返済
3. JVは新しい法人が生まれる。アライアンスは法人をつくらない
🌏 国際経営基本・中級
1. OLI理論のO＝所有優位、L＝立地優位、I＝内部化優位
2. 知識流出リスクが最高の進出形態＝ライセンシング
3. 心理的距離の近い国から段階的に国際化するプロセス理論＝アップサラモデル
4. OEM → ODM → OBM の発展パターン
🎯 国際経営横断整理（4戦略）
1. グローバル統合高・現地適応低＝グローバル戦略
2. グローバル統合低・現地適応高＝マルチドメスティック戦略
3. グローバル統合低・現地適応低＝インターナショナル戦略
4. グローバル統合高・現地適応高＝トランスナショナル戦略（理想形・最難関）
🌱 CSR基本・ガバナンス
1. CSR4層の最も基盤となる責任＝経済的責任
2. コンプライアンス＝CSR4層の法的責任に相当
3. CSVの提唱者＝ポーター＆クラマー
4. 指名委員会等設置会社の3委員会＝指名・報酬・監査委員会
5. 機関投資家が企業経営に積極関与する行動原則＝スチュワードシップ・コード
🌿 ESG・CSV・SDGs横断整理
1. ESGのE/S/G＝環境/社会/統治
2. SDGsは2030年までの17の国際目標
3. CGコードは社外取締役原則2名以上を推奨
4. フリードマン＝株主利益最大化 / フリーマン＝ステークホルダー全体への責任
