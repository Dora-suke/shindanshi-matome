🤝 関係性マーケティングの全体像

👨
先生
関係性マーケティングはリレーションシップ・マーケティングとも言うよ。従来の「売ったら終わり」から「売ってからが本番」への転換がポイント！

🛒 従来型マーケティング

取引ごとに完結
トランザクション型
新規獲得がメイン

VS

🤝 関係性マーケティング

長期的な関係を重視
リレーションシップ型
既存顧客の維持がメイン

概念内容覚え方
CRM
Customer Relationship Management		顧客との関係を管理する仕組み。購買履歴・問い合わせ履歴などをデータベースで一元管理し、個々の顧客に最適なアプローチをするお客様カルテ
LTV
Life Time Value		1人の顧客が生涯にわたってもたらす利益の合計。LTVが高い顧客ほど大切にすべき生涯おつきあい金額
RFM分析Recency（最終購買日）、Frequency（購買頻度）、Monetary（購買金額）の3指標で顧客をグループ分けする手法いつ・何回・いくら
FSP
Frequent Shoppers Program		購買頻度や金額に応じて特典を提供するプログラム。ポイントカードが代表例ポイントカード
ワン・トゥ・ワン顧客1人1人の嗜好やニーズに合わせた個別対応のマーケティングあなただけ特別

📊 1：5の法則と5：25の法則

  • 1：5の法則：新規顧客の獲得コストは既存顧客維持の5倍
  • 5：25の法則：顧客離れを5%改善すれば、利益が25%改善する

→ だから既存顧客を大切にする関係性マーケティングが重要！

💻 デジタルマーケティングの3段階分類

👨
先生
デジタルマーケティングは手法がたくさんあるけど、目的別に整理すると覚えやすいよ！「集客」「接客」「再来」の3段階で見てみよう。
デジタルマーケティング手法の分類
🔍 集客する（見つけてもらう）
SEO / リスティング広告（SEM） / SNS広告 / アフィリエイト広告 / コンテンツマーケティング
⬇️
🏠 接客する（体験してもらう）
LPO（ランディングページ最適化） / EFO（入力フォーム最適化） / レコメンデーション / チャットボット
⬇️
🔄 再来してもらう（関係を続ける）
リターゲティング広告 / メールマーケティング / MA（マーケティングオートメーション） / SNS運用

✍️ 基本チェック（穴埋め）

基本のキホン
・顧客関係を管理する仕組み＝CRM／生涯にもたらす利益合計＝LTV
・最終購買日・頻度・金額の3指標で分類＝RFM分析／購買頻度に応じた特典プログラム＝FSP
・新規獲得は維持の5倍＝1：5の法則／離脱5%改善で利益25%改善＝5：25の法則
・3段階分類：集客→接客→再来

⚠️ 関係性マーケ：引っかけ注意ポイント

⚠️
引っかけ注意
FSPとCRMを混同させる問題が出る！FSPは「特典プログラム」、CRMは「顧客情報を管理する仕組み全体」。FSPはCRMの一部と考えよう。
用語内容覚え方
カスタマー・エクイティ企業が持つ全顧客のLTVの合計。企業の顧客資産としての価値お客全員のLTV合計
パーミッション・マーケティング顧客から事前に許可を得てから情報を届ける手法。メルマガ登録など許可をもらってから

💻 デジタルマーケティング手法の詳細

手法内容
SEO
Search Engine Optimization		検索エンジンで自然検索の上位に表示されるよう、サイトの構造や内容を最適化すること。費用はかからないが時間がかかる
リスティング広告
（検索連動型広告）		検索結果ページの上部に表示される有料広告。クリックごとに課金される（PPC）
アフィリエイト他のサイトやブロガーに商品を紹介してもらい、成果に応じて報酬を支払う
リターゲティング広告一度Webサイトを訪問した人に対し、別サイトでも広告を表示して再訪問を促す
コンテンツマーケティング役立つ記事・動画などのコンテンツを提供して顧客を引き寄せる
バイラルマーケティング口コミがウイルスのように広がることを狙う。「バズる」状態
インフルエンサーマーケティングSNSで影響力のある人に商品を紹介してもらう

📊 その他の重要キーワード

🔍 ビッグデータ

大量・多種・高速なデータ。3Vで特徴づけられる：Volume（量）、Variety（多様性）、Velocity（速度）

🤖 MA

マーケティングオートメーション。見込み客に自動でメール配信やスコアリングを行い、営業につなげる

📧 オプトインとオプトアウト

オプトイン＝「送っていいですよ」と同意
オプトアウト＝「もう送らないで」と拒否
特定電子メール法では原則オプトイン規制（同意なしのメール送信は違法）

🎯 ロングテール

ECでは販売量が少ないニッチ商品の合計が大きな売上を生む現象。棚の制約がないネットならでは

💬 相談1：常連客が減ってきた老舗パン屋

🍞 相談内容

「創業30年の町のパン屋です。最近、常連さんが高齢化して来店が減りました。新しいお客さんも増えません。何かできることはありますか？」

👨
診断士の回答
まずはCRMの発想が大事です。常連さんの情報を整理しましょう！

✅ Step1：既存顧客を大切にする（関係性マーケティング）

  • ポイントカード（FSP）を導入 → 来店頻度を可視化
  • 常連さんの購買パターンを分析（RFM分析） → 最近来なくなった人にDMを送る
  • 誕生日に特別クーポンを送る → ワン・トゥ・ワン・マーケティング

✅ Step2：新規顧客を呼び込む（デジタルマーケティング）

  • Instagramで焼きたてパンの写真を投稿 → SNSマーケティング
  • Googleマイビジネスに登録 → ローカルSEO対策
  • 初回限定のネットクーポン配布 → O2Oでオンラインから来店誘導
診断のポイント
関係性マーケティング（既存客維持）とデジタルマーケティング（新規集客）は両輪！
中小企業ではまずコストの低いSNSから始めるのが現実的。

💬 相談2：ECサイトの売上が伸びない家具メーカー

🪑 相談内容

「地方の家具メーカーです。ECサイトを3年前に作りましたが、アクセスも売上も伸びません。大手ECモールに出店すべきでしょうか？」

👨
診断士の回答
ECモール出店の前に、今のサイトの課題を整理しましょう。集客・接客・再来のどこが弱いか見てみます。

🔍 課題の診断

段階課題対策
集客検索しても出てこないSEO対策＋「地方 家具 オーダーメイド」などのロングテールキーワード
接客サイトが見づらいLPO（ランディングページ改善）、商品レビュー掲載
再来一度きりのお客が多いメールマーケティング（パーミッション制）、リターゲティング広告

💡 さらに踏み込んだアドバイス

  • コンテンツマーケティング：「家具の手入れ方法」「部屋のレイアウト術」などのブログ記事を発信 → 検索流入を増やす
  • ショールームと連携したオムニチャネル：ネットで見て、実物はショールームで確認して、ネットで注文
  • 購入者にレビュー投稿を依頼 → 口コミ効果（バイラルマーケティング）

💬 相談3：リピーターを増やしたい美容サロン

💇 相談内容

「個人経営の美容サロンです。新規のお客さんは来るのですが、2回目以降のリピート率が20%しかありません。何が足りないでしょうか？」

📊 LTVで考える

美容サロンの場合、1回あたり5,000円 × 月1回 × 3年間通えば LTV = 18万円。リピート率を20%→40%に上げるだけで、売上は大幅アップ！

LTV = 平均単価 × 購買頻度 × 継続期間

✅ リピート率UPの施策

  • 来店後にLINE公式アカウントでフォローメッセージ → CRM的な顧客管理
  • 来店3回目でスペシャルトリートメント無料 → FSP的な仕組み
  • お客さんの髪質・好みをデータベース化 → ワン・トゥ・ワン・マーケティング
  • 「次回予約割引」で次回来店を確約 → 顧客維持
  • 施術のビフォーアフターをInstagram投稿（お客の許可あり）→ パーミッション＋SNS集客

☕ CRM／FSP：スターバックスの「Starbucks Rewards」

☕ スターバックスはなぜあんなにリピーターが多い？

CRM FSP ワン・トゥ・ワン

スターバックスの公式アプリ「Starbucks Rewards」は、購入ごとにスターが貯まり、一定数でドリンクやフードと交換できるFSPの代表例です。

しかし単なるポイントカードではありません。アプリは顧客の購買履歴を記録し、よく注文するドリンクをトップに表示したり、好みに合わせたパーソナライズされたクーポンを配信します。これはまさにCRM（顧客関係管理）とワン・トゥ・ワン・マーケティングの実践です。

さらに、モバイルオーダー（アプリで事前注文して店舗で受け取り）はO2Oでもあり、アプリ・店舗・ECを統合したオムニチャネル戦略でもあります。

💡
ここがポイント
スタバが成功しているのは、FSPで来店頻度を上げつつ、CRMで顧客ごとに最適化しているから。ただポイントを貯めるだけでなく、「あなたのことを分かっている」体験を作っている！

📦 オムニチャネル：無印良品の「MUJI passport」

🏠 ネットと店舗の壁を消した無印良品

オムニチャネル CRM O2O

無印良品のアプリ「MUJI passport」は、店舗での買い物もネットでの買い物も1つのIDで統合管理。来店するだけでマイル（チェックインマイル）が貯まり、商品レビューを書いてもマイルがもらえます。

アプリで店舗の在庫確認ができ、ネットで購入した商品を店舗で受け取ることもできます。これにより「ネットで下見→店舗で実物確認→ネットで注文→近くの店舗で受取」といったチャネルを横断した購買体験が自然にできるようになりました。

これがまさにオムニチャネル。O2Oが「ネット→店舗」の一方通行だったのに対し、オムニチャネルはどのチャネルからでも同じ体験を提供します。

📱 SNS×バイラル：シャープのTwitter（X）運用

💬 大企業なのに親しみやすいSNS運用

バイラルマーケティング SNSマーケティング 関係性構築

シャープの公式X（旧Twitter）は、「中の人」がまるで友人のようにフランクに投稿することで話題になりました。自社製品の宣伝だけでなく、日常のつぶやきやユーモアのある返信でフォロワーとの「関係性」を構築。

面白い投稿がリツイートで拡散（バイラル効果）し、テレビやネットニュースでも取り上げられ、広告費をかけずに大きな認知度向上を実現しました。

これはバイラルマーケティングの好事例です。「ウイルスのように自然に広がる」口コミの力を、SNSを通じて最大限に活用しています。

💡
中小企業への応用ヒント
シャープの事例から学べるのは、広告予算がなくてもSNSで勝負できるということ。中小企業でも「中の人」の個性を出した運用で、大手に負けないファンを作れます。ただしパーミッションの発想は忘れずに！

🎮 ロングテール：Amazonのビジネスモデル

📚 売れ筋じゃない商品が利益を生む

ロングテール レコメンデーション CRM

書店では売れ筋の本しか置けません（棚の広さに限界がある）。しかしAmazonは倉庫とネットを使うことで、何百万冊もの本を在庫できます。

売上の分布をグラフにすると、ベストセラーは左端の「頭」で、ニッチな本は右に長く伸びる「しっぽ（テール）」になります。このテールの部分の合計売上が、ベストセラーの合計を上回るのがロングテール現象です。

さらにAmazonは「この商品を買った人は～」のレコメンデーション機能で、ニッチ商品同士をつなげて購買を促進。まさにデジタル技術がロングテールを可能にしています。

💳 デジタル×関係性の融合：楽天経済圏

🛒 すべてのサービスがポイントでつながる

FSP オムニチャネル LTV最大化 ビッグデータ

楽天は、ECモール・銀行・証券・モバイル・トラベル・保険など70以上のサービスを「楽天ポイント」で統合。これは日本最大級のFSPであり、オムニチャネル戦略でもあります。

ユーザーが楽天市場で買い物→楽天カードで決済→楽天モバイルで通信→楽天トラベルで旅行...とすべてのサービスを楽天に集約するほどポイント還元率が上がる（SPU）仕組みです。

これにより楽天は顧客のLTV（顧客生涯価値）を最大化。同時に、全サービスの利用データをビッグデータとして活用し、より精度の高いレコメンデーションやマーケティングを実現しています。

👨
先生
楽天の例は関係性マーケティングとデジタルマーケティングの究極的な融合。FSP（ポイント）で継続利用を促し、CRM（データ活用）でパーソナライズし、オムニチャネル（全サービス統合）で離脱を防ぐ。試験で「LTVを高める施策」を問われたら楽天を思い出そう！

🔍 コンテンツマーケティング：土屋鞄製造所

👜 「売らない」発信で売れるブランドへ

コンテンツマーケティング バイラル 関係性構築

土屋鞄製造所は、Instagramで商品の直接的な宣伝ではなく、革の質感や職人の手仕事の美しさを伝える写真を投稿。「製品を売る」のではなく「ものづくりのストーリー」を発信するコンテンツマーケティングを実践しています。

この結果、ブランドへの共感・愛着が生まれ、ファンが自発的にSNSで紹介（バイラル効果）。中小企業でもストーリーを語る力があれば大手に負けないという好事例です。

実例から学ぶ3つのポイント
① 関係性マーケティングとデジタルマーケティングは別々ではなく融合して使うもの
② 中小企業でもコストの低いSNS・コンテンツマーケティングから始められる
③ LTVの最大化が最終ゴール。そのためにCRM・FSP・デジタルツールを組み合わせる
