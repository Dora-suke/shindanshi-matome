# 模試・作問メソッド 汎用設計図（中小企業診断士 1次 全7科目共通）

**目的**：既存の模試／過去問（=第1回相当）を分析し、弱点を補正した「第2回模試」を設計・作問するための、科目に依存しない標準手順。
経営法務で実施した「分析 → 弱点抽出 → 第2回ブループリント → 作問」をテンプレート化したもの。
作成日: 2026-06-23

---

## 0. 全体フロー（5ステップ）

```
STEP1  棚卸し      既存模試/過去問を「出題一覧」に分解
   ↓
STEP2  辛口評価     本試験傾向と5観点で突き合わせ、弱点を言語化
   ↓
STEP3  穴の抽出     カバー漏れテーマを頻出度マップで洗い出し（重複ゼロ化）
   ↓
STEP4  設計図作成   第2回ブループリント（難度補正・形式補正・比率調整）
   ↓
STEP5  作問・解説   問題本体＋選択肢＋解説（比較表添付）を生成
```

各STEPのアウトプットは次STEPの入力。STEP4までで「設計図」、STEP5で「成果物」。

---

## STEP1. 棚卸し（既存模試を分解する）

既存の問題を、以下の列を持つ **出題一覧表** に必ず再構築する（スタディング模試の「出題一覧」と同じ粒度）。

| 列 | 内容 |
|---|---|
| 問・設問 | 番号 |
| テーマ | 論点名 |
| 出題の狙い | 何を確認する問題か |
| 形式 | 後述の形式タグ |
| 重要度 | A/B/C |
| 難易度 | A(易)/B/C(難) |
| 配点 | |
| 類似過去問 | 本試験の対応年度 |

→ これにより「どのテーマを・どの難度で・どの形式で」既に使ったかが一目で分かり、第2回の重複を防げる。

---

## STEP2. 辛口評価（本試験傾向との突き合わせ）

下記 **共通5観点** で評価し、各観点を「◎/○/△/×」で採点して根拠を1〜2行で書く。

| # | 評価観点 | チェック内容 |
|---|---|---|
| 1 | **形式の再現度** | 本試験の難化形式（組合せ・個数・長文事例・計算）を再現できているか。単純択一に偏っていないか |
| 2 | **難易度分布** | 易問に偏っていないか。本番の体感難度・足切りリスクを反映しているか |
| 3 | **網羅性** | 頻出テーマに穴がないか。「傾向と対策」に明記なのに未出題の論点はないか |
| 4 | **時事・改正対応** | 直近の法改正・制度変更・統計更新を取り込めているか |
| 5 | **正確性・解説品質** | 正答に誤りがないか。解説に根拠（条文/公式/出典）と比較表があるか |

→ 経営法務での結論例：「形式×・難度△・網羅△・時事◎・正確◎」。**×と△が次の設計で潰す対象**。

---

## STEP3. 穴の抽出（カバー漏れテーマ）

1. 科目の**頻出テーマ全リスト**を用意（過去5〜10年の本試験から）
2. STEP1の棚卸し表と突き合わせ、**未出題テーマ**をマーク
3. 各未出題テーマに「本試験頻出度（◎/○/△）」を付け、優先順位化
4. **第1回とテーマ重複ゼロ**を原則に、第2回の出題候補を確定

> アウトプット = 「分野 × テーマ × 頻出度 × 推奨形式 × 推奨難易度」の候補表

---

## STEP4. 第2回ブループリント作成

経営法務の設計図と同じ章立てで作る。必須要素：

1. **設計コンセプト**：STEP2の弱点（×△）と補正方針を1対1で対応させた表
2. **形式配分の目標**（数値）：単純択一を減らし、難化形式を1/4程度入れる
3. **難易度分布の目標**（数値）：A/B/Cの問題数を明示し、第1回より難化
4. **分野別比率**：本試験の構成比に合わせる
5. **問題別ブループリント表**：問・テーマ・形式・狙い／仕込む論点・重要度・難易度
6. **解説に付ける比較表の指定**
7. **作問の優先順位TOP5**

---

## STEP5. 作問・解説

- 問題文 → 選択肢（誤答は「もっともらしい誤り」を仕込む）→ 解説（根拠付き）→ 比較表
- 解説には必ず **根拠（条文番号／公式／統計出典）** と **得点アップのポイント** と **テキスト該当箇所** を付す
- 最後に **正答検証**：全設問の正答に誤りがないかを独立にチェック

---

## ■ 形式タグ（全科目共通の語彙）

| タグ | 内容 | 難化度 |
|---|---|---|
| 択一 | 最も適切/不適切なものを1つ | 低 |
| ★組合せ-正誤 | a〜eの記述から正/誤の組合せをア〜オで選ぶ | 高 |
| ★個数 | 正しいものはいくつあるか（0〜n個） | 高 |
| 空欄組合せ | 空欄A・Bの語句/数値の組合せ | 中 |
| ★計算 | 数値を算出して選ぶ（財務・経済・運営） | 中〜高 |
| ★長文事例 | 会話文/ケース → 設問複数 | 中〜高 |
| ★図表読解 | グラフ・図・統計表を読む | 中 |
| 英文 | 英文読解（経営法務のみ） | 高 |

## ■ 科目タイプ別の調整（ここが汎用化の肝）

7科目を3タイプに分類し、STEP2・4の重点を変える。

### タイプ① 計算科目：財務・会計／経済学・経済政策
- **形式の重点**：★計算・★図表読解（グラフ）を主軸に。単純知識択一は減らす
- **難度補正**：計算の手数（ステップ数）と桁・端数処理で難度を作る
- **網羅の穴になりやすい論点**：
  - 財務 = 連結・税効果・デリバティブ・企業価値評価・CVP・意思決定会計
  - 経済 = ゲーム理論・国際貿易・情報の非対称性・IS-LM/AD-AS の応用
- **時事**：会計基準改正、マクロ統計の更新
- **比較表**：公式一覧・グラフのシフト要因マトリクス

### タイプ② 理論・事例科目：企業経営理論／運営管理
- **形式の重点**：★長文事例・★組合せ-正誤。学者名/モデル名のひっかけ
- **難度補正**：類似概念の識別（例：各種リーダーシップ理論の異同）、長文の読み取り
- **網羅の穴**：
  - 企業経営理論 = 組織論の動機づけ理論・人的資源管理・労働関連法規・マーケの最新概念
  - 運営管理 = 生産管理の計算（生産性・在庫・PERT）、店舗の法規（都市計画法等）、IE手法
- **時事**：DX・サステナビリティ等の新概念、労働法改正
- **比較表**：理論の提唱者×主張、手法の適用場面マトリクス

### タイプ③ 暗記・横断科目：経営法務／経営情報システム／中小企業経営・政策
- **形式の重点**：★組合せ-正誤・★個数・★長文事例（法務）。横断比較問題
- **難度補正**：制度の細かい数字・例外・近年改正、横断的な異同
- **網羅の穴**：
  - 法務 = （別ファイルの第2回設計図参照）
  - 情報 = 最新技術トレンド（AI/クラウド/セキュリティ）、統計手法、システム開発・管理
  - 中小・政策 = **最新版の中小企業白書／施策（毎年更新必須）**、統計数値
- **時事**：★最重要。情報＝技術トレンド、政策＝白書・施策の年度更新が得点源かつ落とし穴
- **比較表**：制度・施策の要件マトリクス、横断比較表

---

## ■ 品質基準（数値目標・全科目共通）

| 指標 | 基準 |
|---|---|
| 難化形式の比率 | 全設問の **約1/4以上**（★タグの問題） |
| 難易度分布 | 易問に偏らず、難（C）を **2〜3割**確保。第1回より難化 |
| 第1回との重複 | テーマ重複 **ゼロ**（2回でフルカバー） |
| 分野別比率 | 本試験の構成比に **±1問**で一致 |
| 正答検証 | 全設問を独立チェックし誤り **0** |
| 解説 | 全問に根拠（条文/公式/出典）＋比較表＋得点ポイント |

---

## ■ 使い方（クイックスタート）

1. 対象科目の既存模試PDF（問題＋解説）を読み込む
2. STEP1 → 出題一覧表を作る
3. STEP2 → 5観点で辛口採点
4. STEP3 → 頻出テーマと突き合わせ、穴を抽出
5. STEP4 → 科目タイプ別調整を反映した設計図を作る（経営法務の設計図がサンプル）
6. STEP5 → 優先TOP5から作問
