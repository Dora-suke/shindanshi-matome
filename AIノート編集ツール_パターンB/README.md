# AIノート編集 — パターンB サンプル（Gemini 無料版）

ローカルサーバーが **① AIを呼ぶ（APIキーを持つ）** と **② ファイルに保存** の両方を担当する最小サンプル。
左にノートのプレビュー、右にAIチャット。指示でHTMLを修正し、「保存」で実ファイルに書き戻す（`.bak`バックアップ付き）。
AIは **Google Gemini の無料枠** を使用（クレジットカード登録不要）。

## 必要なもの
- Python 3（標準ライブラリのみ。pip install 不要）
- Gemini の APIキー（無料）

## STEP 1｜無料APIキーを取る
1. ブラウザで **https://aistudio.google.com** を開く（Googleアカウントでログイン）
2. 左上 **「Get API key」** → **「Create API key」**
3. `AIza...` をコピー（**支払い登録は不要**）

## STEP 2｜起動
ターミナルで（このセッションなら頭に `!` を付けて実行してもOK）：

```
export GEMINI_API_KEY="AIza...あなたのキー"
python3 "/Users/nori3/Desktop/まとめ/AIノート編集ツール_パターンB/server.py"
```

`APIキー : 設定済み ✅` と出れば成功。ブラウザで **http://127.0.0.1:8780** を開く。

## STEP 3｜使う
1. 起動時に `sample_note.html` を自動読込（上のパス欄で別ノートに変更可）
2. 右下の欄に指示を入力 →「送信」（⌘+Enterでも送信）
   - 例：「この表に『審査の有無』の列を足して」「覚え方をもっとやさしく」
3. 左のプレビューが書き換わる（まだファイルは未変更）
4. 「💾 保存」で実ファイルへ書込（同じ場所に `*.bak` を作成）

## STEP 4｜終了
ターミナルで **Ctrl+C**。

## 仕組み（パターンB）
```
ブラウザ ──→ このサーバー ──→ Gemini API（無料枠）
  保存  ←──   .htmlに書込  ←──   修正後HTML
```
- APIキーはサーバー側だけが持つ（ブラウザに出さない）
- 書込先は安全のため「まとめ/」フォルダ配下のみに制限

## 設定（server.py 冒頭）
- `MODEL`             … 既定 `gemini-2.0-flash`（無料）。`gemini-2.5-flash` 等に変更可
- `MAX_OUTPUT_TOKENS` … 既定 8192。大きいノートはここが壁
- `PORT`             … 既定 8780

## Gemini 無料枠の注意
- **回数制限あり**（1分・1日あたりのリクエスト数）。連打すると一時的に弾かれることがある
- 無料枠は**入力データが学習に使われ得る**。個人の学習ノートなら問題ないが、機密は載せない
- キーは無料だが、aistudio へのログイン（Googleアカウント）は必要

## このサンプルの割り切り（本番化の検討ポイント）
- **HTML全文をやり取り**するため、巨大ノート（数百KB）は出力上限で途中で切れる。
  → 本番では「変更箇所だけ差分編集」へ。
- 取り消し（Undo）は `.bak` 1世代のみ。
- RAG（全ノート横断の知識庫検索）は未実装。Bならサーバー側に追加可能。
- qz-memo / qz-hide を含むノートも編集できるが、AIが構造を壊さない保証はないので
  保存前にプレビューで必ず確認を。
