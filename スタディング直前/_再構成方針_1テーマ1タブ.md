## 3. 共通実装ルール（全ファイル必須）

### テンプレート
正本 **`経営法務/qz_31.直前_知的財産法まとめ_m.html`**（CSS/JS/タブ/クラス）と、最新実装例 **`直前対策講座－経営情報システム_冊子/qz_38.直前_経営情報システムまとめ_m.html`** を踏襲。`スタディング直前/CLAUDE.md` の F-1〜F-6 を全実装する。

### 構成（各タブ＝1テーマ）
```
[ステッパー(自ファイルのタブのみ)] → 要点チェック穴埋め(fill-card複数) → 論点チェック○×(ronten-card) → ズバリ予想問題(yoso-wrap)
```

### 絶対要件（過去に差し戻された3点）
1. **解説パネルは全fill-cardに付ける**：explain-panel数＝explain-btn数＝fill-card数。要所のみ不可。
2. **各パネルに「図解」＋「参照リンク」**：図解は qz_31/qz_38 のインラインCSS図（2列比較/矢印フロー/公式ボックス/表）。参照リンクは `.ex-label` 内に `<a target="_blank">`。リンク先は担当科目の詳細ノートfolderを `grep -l "キーワード" ../../FOLDER/*.html` で特定。該当なしは同フォルダ冊子PDF（`直前対策講座_◯◯_講義テキスト.pdf`）にフォールバックし「📄 冊子：講義テキスト（テーマ名）」表記。**生成後 grep でリンク先実在を検証（欠落0）**。
3. **F-6**：`ansRonten` の答え取得を `querySelector('.correct,.wrong')` にする（×答えの誤判定対策）。

### LSキー・採番
- 自ファイルの全 `qzNN_` を割当表の番号で統一（CSSクラス `.qzNN-done` 等／JS LSキー `qzNN_done_v1`/`qzNN_pin_v1`/`qzNN_rhide_v1`/`qzNN_sechide_`／IIFE内 `var LS='qzNN_done_v1'` ほか）。
- yoso正答は `_ansY(id,label,正解)` をベースに各設問へ正解を埋め込む（qz_38方式）。

### 末尾
`</body>` 直前に **hide-buttons v4 正本**を貼る（qz_31/qz_38 の該当ブロックをそのまま流用。`.tab-content` 自動検出・冪等・`__QZ_HIDE_V4__` マーカー）。

### 内容ソースの扱い
- **論点○×・予想問題の正答は必ず解答解説PDFで確定**。講義テキストの抽出は要点穴埋め用。
- **OCR反転に注意**（既知：情報「シリアル/パラレル」、運営「内的/外的参照価格」「MTTR保守性」等）。冊子の○×答えが事実と矛盾する場合は解答解説の解説本文で確認し、**事実に合わせて正しい設問・答えに整える**。
- lift元HTML（現行1ファイル版）には既に著述済みのfill-card/ronten/yosoがある。**該当テーマ分を取り出し、1テーマ＝1タブに割り直して再配置**（マージタブを分割）。本文は作り直さず流用・微修正。

### 暗記科目の量（情報=106/中小=107）
1テーマ＝1タブで薄くなりすぎないよう、各タブ 論点○×は**最低6問**・予想1問を確保（角度を変えて反復）。

## 4. 検証（各ファイル生成後に必ず実行）
```bash
python3 -c "
c=open('ファイル名.html',encoding='utf-8').read()
print('div balance:', c.count('<div')-c.count('</div>'))                       # =0
print('fill/btn/panel:', c.count('class=\"fill-card\"'), c.count('class=\"explain-btn\"'), c.count('class=\"explain-panel\"'))  # 3つ一致
print('F-6:', '.correct,.wrong' in c)                                           # True
print('resetAll pinned:', ':not(.pinned)' in c)                                 # True
print('tab-content:', c.count('class=\"tab-content'))                           # =タブ数
print('LSキー他番号混入:', [k for k in ['qz31','qz33','qz34','qz35','qz38'] if k in c and k[:4]!='qzNN'])  # 自番号以外が無いこと
"
# 参照リンク実在
grep -oE 'href="\.\./\.\./[^"]+"' ファイル名.html | sed 's/href="//;s/"//' | while read p; do [ -f "$p" ] && echo "OK $p" || echo "MISSING $p"; done
```
合格条件：div=0／fill-card=explain-btn=explain-panel／F-6=True／参照リンク欠落0。
