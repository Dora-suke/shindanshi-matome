🔧 生産関数とは — 投入と産出の対応ルール

🧒
学習者
生産関数って、結局なにを表してるの？
🧑‍🏫
先生
「投入したモノ（労働Lと資本K）を入れたら、製品Qがどれだけ出るか」という技術的な対応関係です。式で書くと Q＝f(L, K)。料理でいえば「材料の量→できる料理の量」のレシピ。

生産関数の基本

Q = f(L, K) L＝労働投入量／K＝資本投入量／Q＝産出量
  • 短期：資本K は固定（工場・機械はすぐ変えられない）→ 動かせるのは L だけ
  • 長期：L・K の両方を動かせる → 等産出量曲線・等費用線の世界（タブ②③）
短期はKが固まる短期＝Kは固定／長期＝両方可変

1次📝 短期／長期の区別がそのまま費用論（固定費用の有無）に直結。この区別を最初に押さえると後がラク。

📈 総生産物・限界生産物・平均生産物（TP・MP・AP）

短期（K固定）で、L を増やすと Q がどう動くかを3つの曲線で見る。

用語定義グラフ上の意味
総生産物 TP
（Total Product）		L を増やしたときの産出量 Q そのもの最初は逓増→やがて逓減のS字
限界生産物 MP
（Marginal Product）		L を1単位増やしたときの Q の増加分TP曲線の傾き。TPの変曲点で MP が最大
平均生産物 AP
（Average Product）		労働1単位あたりの産出量 ＝ Q ÷ L原点からTPへ引いた直線の傾き
MPとAPの関係（頻出）：MP曲線は AP曲線の最高点を上から下へ貫く（MP＝AP のとき AP が最大）。MP＞AP なら AP は上昇中、MP＜AP なら AP は下降中。
📊 TP（上図）と MP・AP（下図）の位置関係

1次📝 R01-14・R02-16・H26-13。TP/MP/APのグラフを与え「変曲点でMP最大」「MP=APでAP最大」「MPが原点へ向かう」等の位置関係の正誤を判定させる。

1次⚠️ 「APが最大になるのはMP=0のとき」は誤り。AP最大はMP=APのとき。MP=0はTPが最大（頂点）になる点。

📉 限界生産物逓減の法則

限界生産物逓減（収穫逓減）の法則

資本 K を固定したまま労働 L だけを増やしていくと、ある点を超えると MPL（労働の限界生産物）がだんだん小さくなる。

イメージ：同じ厨房（K固定）に料理人（L）を増やすと、最初は効率UPだが、やがて狭くてぶつかり合い、1人増やしても増産分が減る。

Kそのまま人増やすと効きが減る限界生産物逓減＝K固定でLだけ↑

🔁 規模に関する収穫

こちらは全要素（L も K も）を同時に t 倍したときの話（長期）。限界生産物逓減（片方だけ）と混同注意。

収穫一定

t倍投入 → 産出もt倍

収穫逓増

t倍投入 → 産出はt倍超（規模の経済）

収穫逓減

t倍投入 → 産出はt倍未満

1次⚠️ 「限界生産物逓減」と「規模に関する収穫逓減」は別物。前者＝片方の要素だけ増やす（短期）、後者＝全要素を同時に増やす（長期）。引っかけ頻出。

🧮 コブ＝ダグラス型生産関数

コブ＝ダグラス型

Q = A Ka Lb A＝技術水準／a, b＝各要素の分配率
指数の和規模に関する収穫
a＋b ＝ 1収穫一定
a＋b ＞ 1収穫逓増
a＋b ＜ 1収穫逓減

a＝資本分配率、b＝労働分配率。微分すれば MPK＝aQ/K、MPL＝bQ/L とすぐ出る。

足して1ならそのままa+b=1→収穫一定／＞1逓増／＜1逓減

🗺️ 等産出量曲線とは

🔁 ①の生産関数 Q＝f(L,K) を「真上から見た等高線」。地図の等高線が同じ標高を結ぶように、これは同じ産出量 Q を結ぶ。

定義

同じ産出量 Q を生み出す (L, K) の組合せをすべて結んだ曲線。無差別曲線の「生産者版」。

📐 4つの性質

① 右下がり

L を増やせば K を減らせる（一方を増やせば他方を減らせる）

② 原点に凸

MRTS が逓減するから（後述）

③ 交わらない

1点で2つの産出量を同時に表せないため

④ 右上ほど高産出

原点から離れる＝投入量が多い＝産出量Qが大きい

🔻 MRTS（技術的限界代替率）

MRTS（Marginal Rate of Technical Substitution）

MRTS = MPL / MPK ＝等産出量曲線の傾き（の絶対値）

「L を1単位増やすとき、同じ Q を保つために K を何単位減らせるか」。L が増えるほど（右に行くほど）MRTS は小さくなる＝逓減。これが「原点に凸」の理由。

傾きはエムピー比MRTS＝MPL／MPK

1次📝 R05追試-14・R04-16(2)。MRTSの定義（＝MPL/MPK＝傾き）・逓減の意味・曲線の形状を問う。

1次⚠️ MRTS＝MPL/MPK の分子分母を逆（MPK/MPL）に書いた選択肢が頻出ひっかけ。「L を増やす側＝分子が MPL」と覚える。

🧩 特殊ケース

完全代替

L と K が自由に置き換え可能 → 等産出量曲線は直線（傾き一定）

完全補完（レオンチェフ）

決まった比率でしか使えない → L字型（直角）

💰 等費用線とは

等費用線

C = wL + rK w＝賃金率（労働の価格）／r＝資本のレンタル価格

総費用 C が一定になる (L, K) の組合せ。消費者理論の予算制約線の生産者版。

要素値
傾き（の絶対値）w / r（要素価格比）
横切片（L軸）C / w
縦切片（K軸）C / r
費用線の傾きはダブリュー割るアール等費用線の傾き＝w/r

🔀 シフトと回転（最重要・頻出）

費用Cの増加

傾きそのまま外側へ平行移動（より多く投入可能）

賃金率wの変化

横切片 C/wが動いて回転

レンタル価格rの変化

縦切片 C/rが動いて回転

1次⚠️ 「wが下がると等費用線は平行移動」は誤り。正しくは横切片C/wが外へ動く＝回転。平行移動するのは「Cが変わったとき」だけ。R01-15・H29-15・R04-16(1)の定番ひっかけ。

🌳 幹となる接続式 — この1枚で3論点が全部つながる

タブ①②③はバラバラに見えて、1本の幹でつながっている。ここだけ覚えれば芋づる式に思い出せる。

生産関数 Q = f(L, K)
│ 微分
▼
MPL（労働の限界生産物）, MPK（資本の限界生産物）
┌─────────┴─────────┐

3ステップで再現できる

🧬 展開・応用 — 隣接論点へのつながり

①労働需要曲線の導出

🔁 ①のMPLを使う。

企業の利潤最大化条件は 労働の限界生産物価値＝賃金。すなわち P・MPL＝w（⇔ MPL＝w/P＝実質賃金）。MPL は逓減するので、実質賃金が下がるほど雇用 L を増やす → 右下がりの労働需要曲線が導かれる。

②エッジワース・ボックス（生産の効率）

2財・2要素の経済で、両財の等産出量曲線が接する点の集合＝契約曲線。そこでは両財のMRTSが等しく、生産がパレート効率的（これ以上、一方の財を減らさずに他方を増やせない）。

③費用曲線への橋渡し

各産出量 Q について最適投入（接点）を求め、その最小費用をつなぐと総費用曲線 TCになる。ここから MC（限界費用）・AC（平均費用）・AVC（平均可変費用）へ展開 → 利潤最大化・供給曲線の論点に接続する。

＝つまり本テーマは「費用関数の上流」。ここを固めると費用論・供給曲線がスムーズ。

Q1. 等費用線について、正しいものはどれか。
  • 傾きはレンタル価格÷賃金率（r/w）である
  • 賃金率wが低下すると、等費用線は平行に外側へ移動する
  • 費用Cが増加すると、傾きを保ったまま外側へ平行移動する
  • 縦切片（K軸）は C/w である
正解：C。傾きは w/r（A誤）、wの低下は横切片C/wが動く＝回転で平行移動ではない（B誤）、縦切片はC/r（D誤）。費用Cの増加だけが平行移動。
Q2. 費用最小化（最適投入）の条件として正しいものはどれか。
  • MPL × w ＝ MPK × r
  • MPL / MPK ＝ w / r
  • MPL / w ＝ r / MPK
  • 等産出量曲線と等費用線が交わる点
正解：B。最適投入は接点で MRTS（＝MPL/MPK）＝w/r。これは MPL/w＝MPK/r とも書ける。「交わる点」ではなく「接する点」（D誤）。
Q3. 生産関数の限界生産物（MP）と平均生産物（AP）について正しいものはどれか。
  • MPがAPと等しくなる点でAPは最大になる
  • MPはAPが最大の点でゼロになる
  • APは総生産物曲線の傾きである
  • MPは原点から総生産物曲線へ引いた直線の傾きである
正解：A。MP＝APでAP最大。MP＝0はTP最大の点（B誤）。AP＝原点からの傾き／MP＝TPの傾き（C・D逆）。

  • a は労働分配率を表す
  • a＋b＝1 のとき規模に関する収穫は一定（一次同次）
正解：D。a+b=1で収穫一定、＞1で逓増、＜1で逓減（A・B誤）。aは資本分配率、bが労働分配率（C誤）。
