乗数効果・45度線分析

中小企業診断士 経済学・経済政策 マクロ経済学（財市場の分析）
📘 投資乗数／租税乗数／均衡国民所得／限界消費性向

🐣 「100億円の公共事業」って、本当に100億円分しか効かないの？

🙋
新人くん
ニュースで「政府が100億円の公共事業を行います」と聞くと、GDPは100億円増えるだけだと思っちゃう。でも実は、もっと大きく増えるって本当？
👨‍🏫
先生
その通り！ これが 「乗数効果」 という、マクロ経済学のいちばん面白いマジック。
100億円が 2倍・3倍・5倍 に膨らんでGDPを押し上げるんだ。鍵を握るのは 「もらったお金のうち何％を消費するか」＝限界消費性向。

🌊 100億円が膨らむイメージ（c=0.8の場合）

  • 政府が建設会社に 100億円 払う → 建設会社の所得 +100億円
  • 建設会社の従業員はそのうち 80億円（80%）を消費 → 商店の所得 +80億円
  • 商店はそのうち 64億円 を消費 → さらに別の所得に…
  • これを無限に繰り返すと合計 500億円 ＝ 元の 5倍 のGDP増！

この「5倍」の正体が 1/(1−c) ＝ 1/(1−0.8) ＝ 5。これを 政府支出乗数 という。

乗数＝1÷(1−c)＝バケツリレー消費が消費を呼ぶ
👨‍🏫
先生からひとこと 「無限に増えるなら何でも10倍・100倍になるじゃん？」って思うよね。実は、もらったお金の一部は貯金に回るので、波が少しずつ小さくなっていくんだ。貯金の割合が大きい（＝消費性向が小さい）ほど、波はすぐ消える。だから乗数は小さくなる。

📐 「45度線分析」って何のグラフ？

🙋
新人くん
「45度線分析」 って、グラフに45度の線を引いただけ？ なんでそれが大事なの？
👨‍🏫
先生
45度線は「需要＝供給（生産）」を表す線。横軸が国民所得Y、縦軸も金額。
そこに 総需要曲線（消費＋投資＋政府支出＋純輸出）を重ねて、45度線とぶつかる点が「均衡GDP」。需要に合わせて生産が決まる、というケインズ理論の図解だよ。

📘 シンプルな見方

  • 45度線：原点を通る斜め45度の線（縦＝横）。「作ったものが全部売れる」状態の線
  • 総需要曲線：傾きはc（限界消費性向）、縦軸切片はA＝C₀+I+G+X。45度より緩やか
  • 2本の交点が 均衡国民所得（均衡GDP）
📝 【1次】試験ポイント：R07-7-1・R04-6・H30-7・R02-5 など毎年のように出題。45度線の意味（生産＝所得＝需要）を理解しているかが鍵。

💰 減税より公共事業の方が「効く」って本当？

🙋
新人くん
不景気のとき、政府は「公共事業」と「減税」を選べるよね。同じ100億円なら、どっちがGDPを多く増やすの？
👨‍🏫
先生
公共事業の方が必ず大きい。なぜなら公共事業は「100億円→そのまま誰かの所得」になるけど、減税は最初に消費に回る分だけがスタート（いきなり80億円スタート）。1段階ロスするから乗数が小さくなる。
政策乗数の式c=0.8のとき
政府支出（公共事業）1/(1−c)5倍
減税（マイナス課税）c/(1−c)4倍
増税（差は同じ大きさで逆向き）−c/(1−c)−4倍
📝 【1次】試験ポイント：R07-7-2・R06-7・R03-5 出題。「政府支出乗数 ＞ 減税乗数」は鉄板。差は常に 1（＝1/(1−c) − c/(1−c)）。
公共事業はフル100、減税は0.8スタート1段階ロスがある
⭐ 今日のこれだけ覚えて（ゼロから）
  • ❶ 乗数効果：政府支出100億円が、消費の連鎖で 1/(1−c) 倍 のGDP増を生む
  • ❷ 45度線分析：45度線（生産=所得）と総需要曲線の交点が 均衡GDP
  • ❸ 政府支出乗数 ＞ 減税乗数。差はぴったり 1（均衡予算乗数定理）
📝 ゼロから穴埋め
① 政府支出が増えるとGDPが何倍にも増える効果を 乗数効果 という。
② 政府支出乗数の式は 1/(1−c)。c=0.8 なら値は 5。
③ 45度線と総需要曲線の交点で決まる所得を 均衡国民所得（均衡GDP）という。
④ 減税乗数 c/(1−c) は政府支出乗数より 小さい（c<1のため）。
⑤ 限界消費性向 c は 所得 が1単位増えたときの消費の増加分を表す。

🌱 ケインズ型消費関数

C ＝ C₀ ＋ cY C：消費 ／ C₀：基礎消費（独立消費）／ c：限界消費性向（0<c<1）／ Y：国民所得

📐 用語の整理

  • 基礎消費 C₀：所得が0でも生活のために必要な最低限の消費（食費・家賃など）
  • 限界消費性向 c：所得が1単位増えたとき、消費に回る割合。例：c=0.8 なら所得+1万円で消費+8000円
  • 限界貯蓄性向 (1−c)：所得が1単位増えたとき、貯蓄に回る割合。c+(1−c)=1
  • 平均消費性向 C/Y：所得のうち消費に回る割合。所得Yが大きいほど小さくなる
📝 【1次】試験ポイント：R06-5 ケインズ型消費関数の性質を直接出題。「所得が増えると平均消費性向は低下する」「限界消費性向＜平均消費性向」は頻出。 ⚠️ 【1次】引っかけ：「限界消費性向と平均消費性向は等しい」は誤り。基礎消費C₀があるため、平均>限界 となる。

💴 均衡国民所得の式

Y* ＝ 1/(1−c) × A A = C₀ + I + G + (EX−IM)　← 独立需要の合計

📐 導出（シンプル版）

  • 均衡条件：Y_D = Y_S → cY + A = Y
  • 移項：Y − cY = A → (1−c)Y = A
  • 両辺を(1−c)で割る：Y = A/(1−c)

1/(1−c) がまさに乗数。Aが1増えれば、Yはその 1/(1−c) 倍だけ増える。

均衡Y＝1/(1−c)×A独立需要を乗数倍したのが均衡GDP

  • ❷ 均衡国民所得 Y* = A/(1−c)。Aは独立需要（C₀+I+G+純輸出）

① ケインズ型消費関数 C = C₀ + cY。cは 限界消費性向。
② 限界消費性向は0と 1 の間の値を取る。
③ 45度線の傾きは 1、総需要曲線の傾きは c。
④ 均衡国民所得は Y* = A/(1−c)。Aは独立需要 C₀+I+G+純輸出。
⑤ 政府支出乗数は 1/(1−c)、租税乗数は −c/(1−c)。
⑥ ΔG=ΔTの均衡予算乗数定理によれば乗数は 1（cによらない）。

📘 投資乗数（政府支出乗数）の導出

🔁 ゼロからタブの「バケツリレー」を理論化。Iが1兆円増えると、Yは1/(1−c)倍だけ増える。

📐 導出ステップ

  1. 均衡式：Y = (1/(1−c))×(C₀+I+G+X)
  2. Iを ΔI 増やすと → ΔY = (1/(1−c))×ΔI
  3. 同様に G、純輸出も同じ係数 1/(1−c) が掛かる
  4. これが 投資乗数＝政府支出乗数＝1/(1−c)
ΔY ＝ (1/(1−c)) × ΔG 例：c=0.75、ΔG=10兆円 → ΔY=10×4=40兆円
📝 【1次】試験ポイント：R05-7・R04-5・H29-4-2・H28-8-1。計算問題は「c→1/(1−c)」を瞬時に変換できることが必須。c=0.6→2.5、0.75→4、0.8→5、0.9→10。 ⚠️ 【1次】引っかけ：「政府支出を100増やせばGDPが100増える」は誤り。乗数効果で 1/(1−c) 倍 増える。

📘 租税乗数の導出

📐 税金がある場合の消費関数

税金Tが課されると、可処分所得はY−T。よって消費は：

C = c(Y−T) + C₀ = cY − cT + C₀

これを総需要に代入し、均衡を求めると：

Y = (1/(1−c)) × (A − cT)

よって ΔT が1増える → ΔY = −c/(1−c)（減税ならΔTがマイナスなので、ΔYはプラス）

租税乗数 ＝ −c/(1−c) 1兆円減税（ΔT=−1）→ ΔY = −c/(1−c)×(−1) = c/(1−c)　正の効果
📝 【1次】試験ポイント：R07-7-2・R06-7・H28-8-1。「政府支出乗数 − 租税乗数の絶対値 ＝ 1」を覚えれば、暗記項目が1つ減る。

⚖️ 均衡予算乗数定理（ΔG=ΔT）

🎯 定理の中身

増税で得た税収を、そのまま政府支出に使う場合（ΔG=ΔT）：

ΔY ＝ (1/(1−c))×ΔG ＋ (−c/(1−c))×ΔT ＝ ((1−c)/(1−c))×ΔG ＝ 1×ΔG

つまり 乗数＝1（cの値によらず常に1）。

📝 【1次】試験ポイント：R03-5-1 で各種乗数の比較として出題。「均衡予算乗数の値はcに依存しない」が論点。 ⚠️ 【1次】引っかけ：「均衡予算乗数は0」は誤り（増税と支出が打ち消し合うわけではない）。必ず1。

🌍 開放経済の乗数（輸入を考慮）

📐 限界輸入性向 m を導入

所得が増えると輸入も増える（IM = mY）。総需要は：

Y_D = cY + I + G + EX − mY = (c−m)Y + (I+G+EX)

均衡：(c−m)Y + A = Y → Y = A/(1−c+m)

開放経済の乗数 ＝ 1/(1−c+m) 輸入漏出があるため、閉鎖経済より乗数は小さい
📝 【1次】試験ポイント：R01-5-1 出題。「開放経済の乗数 ＜ 閉鎖経済の乗数」（分母が大きいから）。例：c=0.8, m=0.2 → 1/(1−0.8+0.2)=1/0.4=2.5（閉鎖経済の5の半分）。

📉 デフレギャップ・インフレギャップ

デフレギャップ

完全雇用GDP（Y_F）>均衡GDP
需要不足→ 非自発的失業
対策：政府支出↑/減税

VS

インフレギャップ

完全雇用GDP（Y_F）<均衡GDP
需要超過→ 物価上昇
対策：政府支出↓/増税

📏 ギャップを埋めるのに必要な政府支出

ギャップ＝(Y_F−Y*)。これを乗数で割れば必要な ΔG が出る：

必要なΔG ＝ (Y_F−Y*) × (1−c)

例：ギャップ50兆円、c=0.8 → ΔG=50×0.2=10兆円でOK（乗数5倍効果）

📝 【1次】試験ポイント：R07-7-1・R04-6-2・R02-5・H30-7-2 頻出。ギャップは「総需要と45度線の縦の差」で測る（横軸の所得差ではない）。 ⚠️ 【1次】引っかけ：「デフレギャップ＝Y_F−Y*」と所得差で答えると誤り。正しくは Y_F 水準における総需要と総供給の縦の差。
⭐ 今日のこれだけ覚えて（基本）
  • ❶ 政府支出乗数 1/(1−c)、租税乗数 −c/(1−c)、差は1
  • ❷ 開放経済では 1/(1−c+m)。輸入漏出で乗数は小さくなる
  • ❸ デフレギャップ＝Y_F水準の縦の差。必要なΔG＝ギャップ×(1−c)
📝 基本タブ穴埋め
① 政府支出を ΔG 増やすと均衡GDPは ΔG/(1−c) だけ増える。
② T円減税すると均衡GDPは cT/(1−c) だけ増える。
③ ΔG=ΔTの均衡予算乗数定理ではGDPは ΔG だけ増える（乗数=1）。
④ 開放経済の乗数は 1/(1−c+m)。mは 限界輸入性向。
⑤ 完全雇用GDPが均衡GDPを上回る場合の差を デフレギャップ といい、非自発的失業 を生む。
⑥ 完全雇用GDPが均衡GDPを下回る場合の差を インフレギャップ といい、物価上昇 を引き起こす。

🔍 倹約のパラドックス（合成の誤謬）

🔁 個人の節約は美徳でも、経済全体では悲劇になるという有名なパラドックス。

🌀 ロジック

  1. 家計が「将来不安」で貯蓄を増やす（限界消費性向 c が低下、または基礎消費 C₀ が低下）
  2. 消費が減る → 総需要が減る → 均衡GDP（Y）が減少
  3. 所得が減ったので、結局 貯蓄額（S=(1−c)Y）も増えない、むしろ減少することも
  4. 個々人の合理的行動が、社会全体では悪い結果を生む（合成の誤謬）
📝 【1次】試験ポイント：R05追試-7-2 直接出題。「貯蓄意欲の高まりが均衡GDPを下げ、結果として貯蓄も増えない」というケインズ理論の含意。 ⚠️ 【1次】引っかけ：「貯蓄が増えれば投資も増えてGDPが増える」は古典派の主張。ケインズ理論では貯蓄増→消費減→GDP減。古典派とケインズ派の違いに注意。

🔍 恒常所得仮説（消費理論の発展）

📐 フリードマンの恒常所得仮説

  • 消費は 一時所得 ではなく、恒常所得（長期的に見込める平均所得） によって決まる
  • 一時的な所得増（ボーナス・一時金）は、貯蓄に回りやすく、消費に回る割合は小さい
  • 逆に「恒常的な減税」は消費を増やす効果が高い
📝 【1次】試験ポイント：R03-4 で一時金の経済効果として出題。「定額給付金のような一時的給付は乗数効果が小さい」という現代的論点。

🔍 貯蓄関数と投資＝貯蓄の均衡

S ＝ (1−c)Y − C₀ 貯蓄関数：傾き(1−c)＝限界貯蓄性向、Y軸切片 −C₀

📐 もう一つの均衡条件：S = I（民間部門のみ）

三面等価より Y=C+S（分配面）、Y=C+I（支出面、投資のみ）。両者を等しくおくと S=I。

これは「貯蓄＝投資」のグラフとしても、45度線分析と同等の均衡を示す。

政府を含めれば S = I + (G−T) + (EX−IM) となる（ISバランス式）。

📝 【1次】試験ポイント：R05追試-7-1 貯蓄-投資図と均衡GDPで出題。45度線分析と同じ均衡を別の切り口で表現できることを理解。

🔍 ビルトイン・スタビライザー（自動安定化装置）

🛡 制度として組み込まれた景気安定化装置

仕組み不景気時の作用
累進課税（所得税）所得減→税負担減→可処分所得の減少を緩和
法人税利益減→税負担減→企業の資金繰り支援
失業保険失業増→給付増→消費の急減を防止
社会保障所得減→給付増→生活水準の下支え

裁量的財政政策ではなく「自動的に」効くのがポイント。

🔍 計算問題の標準パターン

例題（H28-8-2 タイプ）

限界消費性向 c=0.75、政府支出を10兆円増やしたとき、均衡GDPはいくら増えるか？

解答：乗数=1/(1−0.75)=4 → ΔY=10×4=40兆円

例題（R04-5 タイプ）

c=0.8 のとき、減税1兆円と政府支出1兆円、それぞれの効果は？

解答：政府支出 1×5=5兆円、減税 1×4=4兆円、差は1兆円

⭐ 今日のこれだけ覚えて（詳細）
  • ❶ 倹約のパラドックス：貯蓄意欲↑→GDP↓→結局貯蓄も増えない（合成の誤謬）
  • ❷ 恒常所得仮説：一時所得は消費に回りにくい（ボーナス・一時金は乗数効果が小さい）
  • ❸ ビルトイン・スタビライザー：累進課税・失業保険などが自動で景気安定化
📝 詳細タブ穴埋め
① 個人の貯蓄が美徳でも社会全体では均衡GDPを下げる現象を 倹約のパラドックス（合成の誤謬）という。
② フリードマンの 恒常所得仮説 によれば、一時的な所得増は消費に回りにくい。
③ 貯蓄関数は S = (1−c)Y − C₀。傾きは限界貯蓄性向。
④ 累進課税や失業保険など、景気変動を自動で和らげる仕組みを ビルトイン・スタビライザー という。
⑤ c=0.75のときの政府支出乗数は 4、c=0.9のときは 10。

🎭 暗記用 語呂・フレーズ集

① 政府支出乗数 1/(1−c)

「いっぴき・いんちのc」1/(1−c)＝政府支出乗数

「1ぴき」＝分子の1。「1−c」＝c=0.8なら0.2、で「1割る0.2＝5」と即計算。

② 租税乗数 −c/(1−c)

「税金マイナスcぶんのc」租税乗数の符号はマイナス

分子に「c」が乗る＝政府支出乗数より「c倍」分小さい、というイメージ。

③ 均衡予算乗数定理 = 1

「ジー＝ティーなら、ぴったり1」ΔG=ΔTなら乗数=1（cによらず）

どんなcでも常に1。これは政府支出乗数 − 租税乗数の絶対値 = (1−c+c)/(1−c) ÷ ... と暗算しなくていい。

④ 開放経済の乗数 1/(1−c+m)

「輸入のmで漏れて、乗数小さく」分母が大きくなる→乗数縮小

⑤ デフレ・インフレギャップ

「不足デフレ／超過インフレ」需要不足＝デフレ、需要超過＝インフレ

デフレ＝需要が足りなくて失業／インフレ＝需要が多すぎて物価上昇。

⑥ 倹約のパラドックス

「みんなで節約、所得も貯蓄も消える」合成の誤謬

⑦ ケインズ型消費関数 C=C₀+cY

「コゼロたすシーワイ」基礎消費＋限界消費性向×所得

🎵 語呂・フレーズ穴埋め

📝 語呂穴埋め
① 政府支出乗数 ＝ 1/(1−c)
② 租税乗数 ＝ −c/(1−c)。符号は マイナス。
③ ΔG=ΔTの均衡予算乗数定理は 1（cによらない）。
④ 開放経済の乗数 ＝ 1/(1−c+m)。mは 限界輸入性向。
⑤ ケインズ型消費関数 C＝C₀＋cY。
⑥ 完全雇用GDPに対し均衡GDPが小さい→デフレギャップ。大きい→インフレギャップ。
⑦ c=0.8 のとき政府支出乗数＝5、租税乗数＝−4。
⑧ c=0.75 のとき政府支出乗数＝4、租税乗数＝−3。
⑨ 個人の貯蓄が社会全体ではGDPを下げる現象を 倹約のパラドックス という。
⑩ 累進課税・失業保険などの景気自動安定化装置を ビルトイン・スタビライザー という。

✏️ 穴埋め問題（初級・中級・上級）

📝 初級（用語の確認）
① ケインズ型消費関数：C = C₀ + cY。
② cは 限界消費性向、(1−c)は 限界貯蓄性向。
③ 45度線は「生産＝所得」を表す。
④ 45度線と総需要曲線の交点が 均衡国民所得。
⑤ 政府支出乗数 ＝ 1/(1−c)。
📝 中級（公式と性質）
① 租税乗数 ＝ −c/(1−c)、符号は マイナス。
② 均衡予算乗数定理：ΔG=ΔTのとき乗数 ＝ 1（cによらない）。
③ 開放経済の乗数 ＝ 1/(1−c+m)。mは限界 輸入性向。
④ 完全雇用GDPで需要不足の状態を デフレギャップ、需要超過を インフレギャップ という。
⑤ デフレギャップ解消には 政府支出増・減税 など総需要拡大政策。
📝 上級（計算と応用）
① c=0.8、政府支出を10兆円増→ ΔY = 50兆円。
② c=0.75、減税1兆円→ ΔY = 3兆円（=1×0.75/0.25）。
③ c=0.8、ΔG=ΔT=10兆円 → ΔY = 10兆円（均衡予算乗数定理）。
④ デフレギャップ50兆円、c=0.8 → 必要なΔG = 10兆円。
⑤ c=0.8、m=0.2 の開放経済の乗数 = 2.5（=1/(1−0.8+0.2)=1/0.4）。
⑥ 一時的所得は消費に回りにくいというのは 恒常所得仮説（フリードマン）。

🏢 事例で学ぶ乗数効果

📕 事例1：100億円の公共事業が地域に与える効果

A県が100億円の道路工事を発注。地域の限界消費性向 c=0.7 とする。

  • 第1段階：建設会社の所得 +100億円
  • 第2段階：従業員が70億円消費 → 商店に+70億円
  • 第3段階：商店主が49億円消費 → +49億円
  • … 無限級数の和 ＝ 100/(1−0.7) ＝ 約333億円のGDP増

気づき：地方に消費性向の高い地域があれば、同じ財政出動でも効果が大きい。

📕 事例2：定額給付金10万円（2009年・2020年）

国民1人10万円配布。総額 約12兆円。

  • 租税乗数 c/(1−c)。実証研究では実質的な c は 0.2〜0.3 と低かった（恒常所得仮説）
  • → 実際のGDP押し上げ効果は理論値の半分以下と試算された
  • 「一時的給付」は消費喚起効果が限定的、という現代経済学の知見

📕 事例3：消費税増税の影響（2014年5%→8%、2019年8%→10%）

租税乗数 −c/(1−c) によりGDPが減少。実際、増税後の四半期GDPは大幅マイナス。

対策として「軽減税率」「ポイント還元」などで消費の落ち込みを緩和（疑似的な減税）。

📕 事例4：コロナ禍の財政出動（2020年）

日本では補正予算 約57兆円超。乗数効果でGDP水準を維持しようとした。

ただし将来不安で c が低下、貯蓄率が急上昇 → 倹約のパラドックス が現実化。

📕 事例5：開放経済の輸入漏れ

給付金で家電を購入 → 海外メーカーの製品（輸入）→ 日本のGDPには寄与しない。

「Made in Japan」の購入が乗数効果を高める、という政策的含意（地域振興券・買い物券）。

⭐ 今日のこれだけ覚えて（事例）
  • ❶ 地方の公共事業＝消費性向が高ければ乗数効果も大きい
  • ❷ 一時給付金＝恒常所得仮説により乗数効果が低い（理論値より小）
  • ❸ 輸入漏れ＝開放経済では乗数 1/(1−c+m) で縮小する

1
限界消費性向が0.8のとき、政府支出を10兆円増加させた場合の均衡GDPの増加額は？（H28-8-2類題）
  • 10兆円
  • 25兆円
  • 50兆円
  • 80兆円
乗数 = 1/(1−0.8) = 5。ΔY = 10×5 = 50兆円。
2
限界消費性向が0.75のとき、5兆円減税の効果は？（R07-7-2類題）
  • 5兆円増加
  • 15兆円増加
  • 20兆円増加
  • 25兆円増加
租税乗数の絶対値 = c/(1−c) = 0.75/0.25 = 3。ΔY = 5×3 = 15兆円。
3
均衡予算乗数定理（ΔG=ΔT）に関する記述として正しいのは？（R03-5-1類題）
  • 乗数は0になる
  • 乗数は1で、限界消費性向に依存しない
  • 乗数は1/(1−c)になる
  • 乗数はcに依存して変動する
政府支出乗数1/(1−c) − 租税乗数c/(1−c) = (1−c)/(1−c) = 1。cの値によらず常に1。
4
デフレギャップに関する記述として正しいのは？（R02-5類題）
  • 完全雇用GDP水準において総需要が総供給を上回る部分
  • 完全雇用GDP水準において総需要が総供給を下回る部分で、非自発的失業を生む
  • 完全雇用GDPと均衡GDPの所得の差そのもの
  • 物価上昇を引き起こす要因となる
デフレギャップは Y_F水準での総需要と総供給の縦の差。需要不足→失業発生。所得差ではない点に注意。
5
開放経済（限界消費性向c=0.8、限界輸入性向m=0.2）における乗数は？（R01-5-1類題）
  • 5
  • 4
  • 2.5
  • 1.25
1/(1−c+m) = 1/(1−0.8+0.2) = 1/0.4 = 2.5。輸入漏出で閉鎖経済（5）の半分に。
6
倹約のパラドックスについて正しいのは？（R05追試-7-2類題）
  • 家計の貯蓄意欲が高まると投資が増えGDPが拡大する
  • 家計の貯蓄意欲が高まると消費が減り、均衡GDPが下がり、結果として貯蓄も増えない
  • 古典派経済学の代表的な理論である
  • 限界消費性向が大きいほど顕著に現れる
ケインズ理論の代表例。合成の誤謬の一種で、個人の合理性と全体の結果が乖離する。
7
ケインズ型消費関数 C=C₀+cY の性質として正しいのは？（R06-5類題）
  • 所得が増えると平均消費性向は一定
  • 限界消費性向と平均消費性向は等しい
  • 所得が増えると平均消費性向は低下し、常に平均>限界が成立する
  • 基礎消費C₀がない場合でも平均>限界が成立する
平均消費性向 C/Y = C₀/Y + c。Yが増えるとC₀/Yは0に近づき、平均消費性向はcに収束。常に 平均>限界（c）。
8
一時的な定額給付金の効果が理論値より小さくなる理由として最も適切なのは？（R03-4類題）
  • 乗数効果が政府支出より大きいから
  • 恒常所得仮説により、一時所得は消費に回りにくいから
  • 政府支出乗数より租税乗数が大きいから
  • 均衡予算乗数定理が働くから
フリードマンの 恒常所得仮説。一時所得は貯蓄に回りやすく、cが実効的に小さくなる。

💡 身近な実例で「乗数効果」を体感

🏗 ニューディール政策（米国・1933〜）

大恐慌対策としてフランクリン・ルーズベルトが大規模公共事業（テネシー川流域開発など）を実施。ケインズ理論の実証実験と評される。失業率は25%→15%程度まで改善。

🏛 日本の補正予算（毎年恒例）

景気対策として数兆〜数十兆円規模の補正予算が組まれる。1兆円の公共事業で c=0.6 なら2.5兆円のGDP押上効果と試算される。

🛍 地域振興券・プレミアム付商品券

地域内でしか使えないため 輸入漏れを防ぐ工夫。乗数効果を最大化する政策設計。

📉 リーマンショック後の財政出動（2008〜2009）

米国「アメリカ復興・再投資法」7870億ドル、日本「定額給付金」12兆円。乗数の理論を実際の政策で検証した時代。

🦠 コロナ禍の経済対策（2020〜2022）

日本：3度の補正予算で総額100兆円超。米国：CARES法2.2兆ドル。「乗数効果」「ヘリコプターマネー」が主要キーワードに。

💴 消費税の景気影響

2014年5→8%、2019年8→10%の増税後、いずれもGDPが大きく落ち込み（租税乗数効果）。経済学のテキストの予測と一致した実例。
