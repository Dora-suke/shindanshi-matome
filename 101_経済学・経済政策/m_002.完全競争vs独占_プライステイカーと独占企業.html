完全競争 vs 独占

中小企業診断士 経済学・経済政策 | プライステイカーと独占企業の違い
全4タブ構成 図解で理解する市場構造
一番の違い:「自分で価格を動かせるか」

完全競争市場の企業(プライステイカー)

市場が決めた価格を黙って受け入れるだけ。自分では価格を動かせず、ただ「受け入れる(take)」しかない立場。だから price-taker と呼ばれる。

独占企業

市場に自分しかいないので、「いくらで売るか」を自分で決められる。ただし「高くすれば客が減る」という需要曲線の制約を一人で背負う。

この一点の違いから、「収入の形」と「儲け方のルール」がガラリと変わります。
「プライステイカー」をもう少し丁寧に

完全競争市場にはライバル企業が星の数ほどいるため、1つの企業が「うちだけ生産量を増やそう!」と頑張っても、市場全体のバランス(価格)には全く影響を与えません。

企業は市場で決まった価格をただ受け入れるしかない「プライステイカー(価格受容者)」となるため、企業にとって「商品の価格は○○円で一定(固定)」という状態になります。

直感的なまとめ
完全競争 独占
価格を決めるのは市場自分
D・MR・Pの関係全部同じ1本の線MRがDより下にズレる
利潤最大化条件P = MCMR = MC、ただし P > MR
戦略のイメージコストが価格に追いつくまで作るわざと量を絞って高値で売る

完全競争の企業が「ひたすら追いかける」のに対して、
独占企業は「出し惜しみして上から値段を被せる」
── この2つの図を行き来すると、その違いがハッキリ見えてくるはずです。

収入(TR)の形を比べる
完全競争(プライステイカー)
収入は真っ直ぐな直線
価格は固定 → 売るほど比例して伸びる
独占企業
収入は山なりの曲線
売りすぎると値下げで収入が減る
🤔
左は「100円で何個でも売れる世界」なので、売った分だけ素直に収入が伸びるんですね。
💡
右は「11個目を売るには既存の10個も値下げしないといけない世界」。ある量を超えると追加販売の損のほうが大きくなり、収入が逆に減ります。
なぜ完全競争では「価格 = 限界収入」になるのか?

パンで理解するシミュレーション(1個100円で固定)

限界収入とは、「生産量(販売量)を1個増やしたときに、追加で増える売上」のことでした。価格が一定の世界で計算してみましょう。

販売量売上(TR)追加の売上(MR)
1個100円100円
2個200円100円
3個300円100円
4個400円100円
5個500円100円
結論: 価格が100円で固定されている以上、「1個追加で売った時の売上(限界収入)」は、常に「商品の価格(100円)」と全く同じ。だから完全競争市場では P = MR が成立する!

独占市場との違い(復習)

独占市場では、このルールが成り立ちませんでした。独占企業がたくさん商品を売るためには「自ら価格を下げていかなければならない」ため、安くなった価格で全ての商品を売る羽目になり、追加の売上(限界収入)は価格よりも激しく減っていく ── 限界収入曲線が需要曲線の2倍の傾きで下がるという現象が起きていました。

「価格を変えられない完全競争市場」と「価格を自分で変える独占市場」の違いとして押さえておくとバッチリ!
価格 P・限界収入 MR・限界費用 MC を1つの図で
ここがいちばん大事! 完全競争では D・MR・P が同じ水平線上で1本に重なるので、左のグラフでは1本の青い線が3つの役割を兼ねています。「MR の線がない」のではなく「P の線と完全に同じ場所にいる」ということです。
完全競争(プライステイカー)
P = MR = MC で生産
需要曲線が水平 → D・MR・Pが
1本の線にピッタリ重なる
この線とMCが交わる点で生産
独占企業
P > MR = MC で生産
①MR=MCで生産量Q*を決め
②そのQ*に対して需要曲線まで
真上に上がった点が価格P*
図の読み方まとめ

左(完全競争)の読み方

青い線1本が「需要曲線」「限界収入」「価格」の3役を兼ねています。

なぜ重なるか? 価格が市場で固定されているので、1個追加で売っても収入は必ずちょうどその価格分だけ増えるから。

つまり「あと1個売って増えるお金(MR)」と「価格(P)」が常に等しい。

利潤最大化条件: P = MC
シンプル! 価格とコストが等しくなるまで作る

右(独占)の読み方

需要曲線が右下がりなので、追加で1個売るには値下げが必要 → MR は需要曲線より下を、しかも倍の急さで下がっていく。

これが MR が D の下に分かれて見える理由です。

独占企業の意思決定は2段階:
  1. MR と MC が交わる点を見て、生産量 Q* を決める(儲けが最大になる量)
  2. その Q* から真上に上がって需要曲線にぶつかった高さを価格 P* にする
利潤最大化条件: MR = MC(ただし P > MR)
P* と MR=MC の縦の隙間 = マークアップ(超過利潤の正体)
これは何の話? 「完全競争市場で一番儲かるのは P = MC のとき」というルールを、数学的な言葉(グラフの線の傾き)で言い換えただけのものです。新しい話ではなく、同じことを別の角度から見ています。
① 「費用曲線の接線の傾き」の正体

資料には「限界費用は費用曲線の接線の傾きと等しくなります」と書かれています。つまり ──

費用曲線の接線の傾き = 限界費用(MC)
「追加で1個作るときにかかるコスト」のこと
イメージ: 費用曲線のある点に定規を当てたとき、その定規の角度(傾き)が、その生産量での「追加1個のコスト」を表している。生産量が増えるほどカーブが急になる = MCが上がっていく。
② 「収入曲線の傾き」の正体

完全競争市場では、1個売ろうが100個売ろうが価格はずっと同じ(例:100円)。

このとき売上のグラフ(収入曲線)を描くと、「1個売れば100円、2個売れば200円」という一定のペースで増えていく右上がりの直線になります。

収入曲線の傾き = 価格(P) = 限界収入(MR)
直線だから、どこでも傾き(変化のペース)は同じ = 常に価格と等しい
ポイント: 直線の傾きはどこを取っても一定。完全競争では価格が固定なので、収入曲線は直線になり、その傾きは常に「商品の価格」= 「限界収入」と同じになる。
③ なぜ2つの傾きが等しくなると「利潤最大」なのか?

難しい文を翻訳してみよう

①②で正体がわかったので、元の文を「翻訳」できます:

元の文:
「費用曲線の接線の傾き」が「収入曲線の傾き」に等しくなるとき利潤が最大化される
→
翻訳後:
「限界費用(追加のコスト)」が「限界収入(追加の売上 = 価格)」に等しくなるとき利潤が最大化される

追加の1個を作るべきかどうか?

企業の儲け(利潤) = 収入 - 費用 です。追加の1個を作ろうか迷ったとき ──

追加の売上 > 追加のコスト

→ 作れば作るほど利潤が増える
(まだ作った方がいい!)

追加の売上 < 追加のコスト

→ 作れば作るほど損をする
(作りすぎ!)

追加の売上 = 追加のコスト(MR = MC)になる瞬間
= これ以上利潤を増やせない「利潤最大化」のゴール地点

補足:数学的な理由

「グラフが最大値(頂点)となるときの接線の傾きは0になる(微分すると0)」という性質を使います。

利潤 = 収入 - 費用 を微分すると:
利潤の傾き = 収入の傾き - 費用の傾き = 0
式を整理すると:
収入の傾き(MR) = 費用の傾き(MC)
数学的にも、両者の傾きが等しくなる地点が利潤の最大値であることが証明されます。
