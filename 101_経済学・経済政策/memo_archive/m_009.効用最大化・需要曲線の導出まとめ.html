📘 予算制約と接点条件

🐣 「決まったお金で一番嬉しい組合せ」を選ぶ

🧒
学習者
持ってるお金は決まってる。何をどれだけ買えば一番うれしい？
🧑‍🏫
先生
これが 効用最大化問題。「予算の線」と「同じ嬉しさの線（無差別曲線）」が 接する場所 が答え。さらに価格を変えながら最適点を追うと 需要曲線 が出来上がる。

🐣 つかみ

  • 予算制約＝買える組合せの上限ライン。
  • 効用最大化＝予算ぴったり使って、無差別曲線の一番外側に届く点。
  • 需要曲線＝価格を動かしたときの最適な消費量の軌跡。
📊 予算制約の直観：「買える/買えないエリア」
📷 対応する手書きノート/教科書ページ：IMG_2393
予算M円で買えるX,Yの組合せは緑の三角形の中。線の上ぴったりが「予算使い切り」、線の外は「足りない」。

🌱 予算制約式の作り方

所得 M、価格 Px・Py のとき

X財をx個、Y財をy個買うとき、支出は Px·x + Py·y。これを所得Mに収める。

Px·x + Py·y = M y = -(Px/Py)·x + M/Py 傾き＝-(Px/Py)（価格比のマイナス）／y切片＝M/Py

導出：両辺をPyで割って整理。直線の式 y = ax+b の形にして傾きと切片が読み取れる。

「購入可能範囲で最も高い効用を得る」

予算制約線を 超えない範囲 で、効用が最も高くなる点が最適消費点。
予算ぴったり使い切るのが基本（残すと効用がもっと上がるはずだから）。

非飽和性の仮定：「もっと欲しい」気持ちが続く限り、お金を残しても効用は増えない。よって予算は使い切る。

📘 効用最大化の条件

📐 接点条件（最重要）

予算制約線 と 無差別曲線 の 接点 が最適消費点。
そこでは「予算線の傾き＝無差別曲線の傾き」が成り立つ。

MRS = MUx/MUy = Px/Py 限界代替率＝価格比

📘 需要曲線の導出（4ステップ）

  1. X財の価格 Px を 段階的に下げる（Px₁ → Px₂ → Px₃）
  2. そのたびに予算制約線が x切片側に 右へ拡大 する（y切片 M/Py は不変）
  3. 新しい予算制約と無差別曲線の接点が 最適消費点 となり右へシフト
  4. 価格と消費量の組合せ（Px, x）をプロット → 右下がりの需要曲線

予算制約の「シフト」と「回転」は区別！
所得が変わると 平行シフト、価格が変わると 回転（一方の切片だけ動く）。

🔍 数学的な解法（コブ＝ダグラス型）

U = X^a · Y^b 型の効用関数

限界効用：MUx = a·X^(a-1)·Y^b、MUy = b·X^a·Y^(b-1)
MRS = MUx/MUy = (a/b)·(Y/X)

最適：(a/b)·(Y/X) = Px/Py これと予算制約 Px·X + Py·Y = M を連立して解く

代表的な解の公式：a+b=1 なら X*=aM/Px、Y*=bM/Py。
→ 支出比率がパラメータ a, b に等しくなる（コブ＝ダグラスの著名な性質）。

🧮 計算例：U = X^0.5·Y^0.5、M=100、Px=10、Py=20

  1. MRS = (0.5/0.5)·(Y/X) = Y/X
  2. 最適条件：Y/X = 10/20 = 1/2 → Y = X/2
  3. 予算制約に代入：10·X + 20·(X/2) = 100 → 20X = 100 → X* = 5
  4. Y* = 5/2 = 2.5
  5. 支出比率：X支出 50円、Y支出 50円（半々＝指数比0.5:0.5に一致）

🔍 加重限界効用均等の法則

効用最大化条件 MUx/MUy = Px/Py を変形すると、

MUx/Px = MUy/Py 「1円当たりの限界効用が等しくなるように消費する」

X財に1円使うと得られる追加効用 と Y財に1円使うと得られる追加効用 が等しい状態。

🔍 価格効果による需要曲線の右下がり

価格↓ → 最適消費点が右へ → 消費量↑

価格効果＝代替効果＋所得効果（次の論点で詳しく）。
通常はこの2効果の合計で需要曲線は右下がりになる。

✍️ ノートの青文字・手書きメモ

IMG_2393：予算制約とMRSの定義

ノート：M = Px·X + Py·Y → Y = -(Px/Py)·X + M/Py
  • 青囲み式：M = Px·X + Py·Y → Y = -(Px/Py)·X + M/Py
  • 「予算制約」（赤縦書き）
  • MRS = Marginal Rate of Substitution
  • 傾き＝限界代替率＝MUx/MUy
  • 限界代替率逓減：|ΔY₂/ΔX₂| > |ΔY₁/ΔX₁|

IMG_2394：「購入可能範囲」のメモ

ノート：「購入可能範囲において最も高い効用を得る」最適消費点の図
  • 購入可能範囲において最も高い効用を得る（効用最大化の言語化）
  • 価格比＝(Px/Py)
  • 最適消費点（接点）：予算制約と無差別曲線（等高線）の接点

IMG_2408：価格効果と需要曲線

教科書：Px低下 → 予算制約線が右に拡大 → 最適消費点が右にシフト → 需要曲線
  • 「価格効果」「最適消費点が変化することを 価格効果 といいます」
  • 「予算制約線が右に拡大していく」
  • 需要曲線（赤波線で強調）
  • 青書込み：需要弾力：価格↑→消費↓
  • 「効用最大化条件のもと」（前提として明記）
🔑 これだけは押さえる
  • 予算制約：Px·x + Py·y = M（傾き-Px/Py）
  • 効用最大化条件：MRS = Px/Py（接点で成立）
  • 加重限界効用均等の法則：MUx/Px = MUy/Py
  • 需要曲線＝価格を変えたときの最適消費点の軌跡（右下がり）

✏️ 穴埋め演習

基礎レベル
① 予算制約式：Px·x + Py·y = M（所得）
② 予算制約線の傾きは -(Px/Py)
③ 効用最大化の条件：MRS = Px/Py
④ 同条件を変形した 加重限界効用均等の法則：MUx/Px = MUy/Py
⑤ X財の価格を下げると最適消費点は 右へシフト する。

Q2
X財の価格Pxが下落したとき、予算制約線はどう動くか。
  • 平行に右シフトする
  • y切片はそのままで、x切片が右に動く（回転）
  • y切片もx切片も上に動く
  • 変化しない
x切片はM/Pxで、Pxが下がると大きくなる。一方y切片M/Pyは不変。よって y軸を軸に右回転する。
