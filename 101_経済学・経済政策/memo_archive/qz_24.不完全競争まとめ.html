📘 独占企業の限界収入（MR）

需要関数 D(x) = -ax + P₀ のとき

  • 収入 R(x) = D(x)·x = -ax² + P₀·x
  • 限界収入 MR(x) = R'(x) = -2ax + P₀
MR は需要曲線 D の「2倍の傾き」 需要曲線：傾き-a／MR：傾き-2a（Y切片は同じ、X切片は半分）

🎯 独占の利潤最大化（MR=MC）

🔑 手順

  1. MR = MC を解き、最適生産量 Q_M を求める
  2. Q_M を 需要曲線 に代入し、独占価格 P_M を求める

→ 価格 P_M は MC より 高い（マークアップ＝独占の利益源）。

📊 独占の利潤最大化（D・MR・MC）
📖 独占グラフの読み方（5ステップ）
  • ① 需要曲線 D（赤）から MR（紫）を引く（傾き2倍）
  • ② MR と MC（緑）の交点で Q_M を決定（独占企業はここを選ぶ）
  • ③ Q_M を D 曲線へ上向き投影 → 独占価格 P_M を決定（MCより高い）
  • ④ 完全競争なら D=MC の交点（Q*, P*）＝より多い数量・低い価格
  • ⑤ Q_M < Q* の差で 死荷重△ が発生
①完全競争vs独占 → ②利潤最大化 → ③余剰・死荷重 → ④自然独占 → ⑤寡占

📉 独占による厚生損失

  • 完全競争：Q* で生産（P=MC）→ 総余剰最大
  • 独占：Q_M < Q* で生産（MR=MC）→ 取引量が減る
  • 減った取引分の余剰が 死荷重として失われる（誰の得にもならない）
📊 独占の余剰分解（消費者余剰・生産者余剰・死荷重）
📖 読み方
  • 赤△＝消費者余剰（独占下では完全競争より小さい）
  • 緑＝生産者余剰（独占企業の取り分。消費者余剰の一部を奪う）
  • 紫△＝死荷重＝Q_M〜Q* の取引が行われず失われた総余剰
①完全競争vs独占 → ②利潤最大化 → ③余剰・死荷重 → ④自然独占 → ⑤寡占

🏭 自然独占（費用逓減産業）

大きな固定費で平均費用が下がり続ける産業

電力・鉄道・ガス・水道など。固定費が巨大で、生産量を増やすほど 平均費用 AC が下がる。
→ 1社で大規模生産する方が安いので、自然と独占になる。

  • 長期的にも AC が逓減し、MC < AC が続く
  • P=MC（限界費用価格形成） は効率的だが、AC>MC なので 赤字→補助金が必要
  • P=AC（平均費用価格形成） は損益ゼロで操業継続可能（次善策）
📊 自然独占と価格規制（独占／P=AC／P=MC）
P=MC は社会的に効率的だが、自然独占では AC>MC のため赤字。そこで現実には P=AC（損益ゼロ） で規制したり、赤字分を補助金で補ったりする。
①完全競争vs独占 → ②利潤最大化 → ③余剰・死荷重 → ④自然独占 → ⑤寡占

📐 寡占：屈折需要曲線

🔑 価格が動きにくい理由（価格の下方硬直性）

  • 値上げ：他社は追随しない → 自社だけ高くなり客が逃げる（需要は弾力的＝なだらか）
  • 値下げ：他社も追随する → 客はあまり増えない（需要は非弾力的＝急）
  • 現行価格で需要曲線が 折れ曲がる（屈折） → MR に段差（不連続）が生じる
  • MC がこの段差の範囲で動いても 価格・数量は変わらない＝価格が硬直的
📊 屈折需要曲線とMRの段差

🍽 独占的競争

  • 多数の企業が 差別化された製品 で競争（飲食店・美容院など）
  • 差別化により各社は 右下がりの需要曲線 をもち、短期は超過利潤を得られる
  • 参入自由なので 長期では超過利潤がゼロ（新規参入で需要を奪われる）
