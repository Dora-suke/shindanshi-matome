🏭 生産関数 ― 入れたら出てくる

🧒
学習者
そもそも「いくつ作れるか」ってどう決まるの？
🧑‍🏫
先生
労働（L）や機械（K）を投入すると製品が出てくる関係を式にしたのが生産関数。費用はこの生産関数から決まるので、まずここが出発点です。

生産関数の基本

x = f(L, K) x＝生産量、L＝労働、K＝資本（機械・設備）

投入（人・機械）を入れると、製品が何個出てくるかを表す。短期では資本Kを固定とみなし、労働Lを増やして生産量xを調整する。

⚠️ 限界生産性逓減の法則（ていげん＝だんだん減る）

他の投入要素（K）を一定として、ある投入要素（L）を追加し続けると、追加1単位あたりの生産増加量（限界生産性）は次第に減少する。

📊 生産関数（総生産物 TP）と限界生産性の逓減
🏪 たとえ：料理で考えると、1人より2人の方が早い。でも同じ広さの厨房に5人、10人と増やすと、ぶつかり合って効率が落ちる。これが限界生産性の逓減。畑の面積（資本K）が一定のとき、肥料・人手（労働L）を増やしても収穫の伸びは鈍る。
試験ポイント：限界生産性が逓減するから、生産量を増やすほど「あと1個」を作る費用（限界費用 MC）は上昇する。生産関数の逓減 ⇄ MC曲線の右上がりは表裏一体。次タブの費用へつながる。

💰 会社の費用は2種類ある

🧒
学習者
会社の費用ってお給料、家賃、材料費とかあるけど、何が違うの？
🧑‍🏫
先生
大事なのは「生産量に応じて変わるか」。家賃のように生産量と関係なく毎月かかるのが 固定費（FC）、材料費のように作るほど増えるのが 変動費（VC）。

📦 費用の2分類

  • FC（固定費）＝生産量に関係なく一定（家賃・人件費・設備）
  • VC（変動費）＝生産量とともに増える（材料費・光熱費）
  • TC（総費用）＝ FC ＋ VC
📊 固定費 FC と 変動費 VC のイメージ
生産量を増やしても FC（固定費） は変わらず、VC（変動費） だけ増える。両方足したのが TC（総費用）。TC曲線は VC曲線を FC分だけ上に平行移動した形。

📘 6種類の費用と典型形

🔑 6つの費用

  • TC＝総費用、VC＝変動費、FC＝固定費（TC=VC+FC）
  • AC＝平均費用＝TC÷x、AVC＝平均可変費用＝VC÷x
  • MC＝限界費用＝「1つ増やすごとにかかる費用」＝dTC/dx

費用関数の典型形（3次関数 → U字AC）

TC = C(x) = ax³ + bx² + cx + d FC=d、VC=ax³+bx²+cx
MC = dTC/dx = TC'(x) TCを生産量xで微分した値＝接線の傾き
📊 U字型 AC・AVC・MC（MCはAC,AVCの最低点を貫く）
📖 グラフの読み方
  • 緑（MC）＝限界費用＝1単位追加で増えるコスト
  • 赤（AC）＝平均費用＝TC÷x。最低点でMCと交差＝損益分岐点
  • 橙（AVC）＝平均可変費用＝VC÷x。最低点でMCと交差＝操業停止点
  • MC曲線は必ず AC・AVC の最低点を下から貫く（数学的必然）

🎯 利潤最大化（P = MC）

利潤＝収入−費用

π(x) = R(x) − C(x) R(x)＝収入関数（完全競争では R(x)=P·x）、C(x)＝費用関数
完全競争企業は プライステイカー（価格を市場から受け入れる）。よって P は与件で、決めるのは生産量 x のみ。
📊 収入 R(x)・費用 C(x)・利潤 π(x) の関係

📐 利潤最大化の条件

1階条件：π'(x) = 0

R(x) = P·x なので R'(x) = P。
π'(x) = R'(x) − C'(x) = P − MC(x) = 0

P = MC(x) 価格＝限界費用 → 利潤最大の条件
📊 限界利潤の累積：P と MC の間の面積
📖 利潤の2つの読み方（同じ値を別の角度から）
  • 総額グラフ（上の R(x)・C(x) 図）：x* で R−C の縦差 ＝利潤
  • 面積グラフ（P と MC の図）：緑面積＝0〜x*の増分利潤の累積。x*でP=MCとなり追加1個の利潤が0
  • x*を超えると MC>P で1個ごとに損失（赤面積）。緑面積を最大化するのがx*

🚦 完全競争の供給曲線

📐 供給曲線＝MC曲線（AVC最小値以上の部分）

各価格Pに対して、企業は P=MC を満たす生産量を選ぶ。
→ MC曲線そのものが供給曲線になる。
ただし、P < AVC最小値 では操業停止（生産量0）。

🔍 価格水準で決まる4つの状態

P と AC・AVC の関係状態理由
P > AC最小値黒字（利潤プラス）1個あたり総費用を超える価格
P = AC最小値損益分岐利潤ゼロ（操業継続）
AVC最小 < P < AC最小赤字でも操業FCの一部を回収できる
P < AVC最小値操業停止VCすら回収不能
📊 価格Pの位置で「黒字／損益分岐／赤字操業／操業停止」が決まる
結論：供給曲線は AVC最小点より上のMC曲線。価格Pが上がるほど P=MC を満たすxが増える（右上がりの供給）。P₄のように AVC最小を下回ると供給は0になる。
