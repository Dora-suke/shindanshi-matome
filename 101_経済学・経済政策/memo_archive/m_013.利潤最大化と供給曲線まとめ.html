📘 完全競争企業の意思決定

🐣 「いくつ作れば一番儲かる？」

🧒
学習者
商品の価格は決まってる。じゃあ、いくつ作れば利益が最大？
🧑‍🏫
先生
「もう1個作ったときの売上(価格P) ＝ もう1個作るのにかかる費用(MC)」になる量。これが P = MC の利潤最大化条件。

🐣 つかみ

  • 利潤 π = 収入 R(x) − 費用 C(x)
  • 利潤最大の条件：P = MC(x)
  • 完全競争企業の供給曲線＝MC曲線（AVC最小値より上）
📊 「もう1個作ったときに儲かる？」の判断
P = MC（追加収入＝追加費用の生産量）
📷 対応する手書きノート/教科書ページ：IMG_2400

🌱 利潤の式

利潤＝収入−費用

π(x) = R(x) − C(x) R(x)＝収入関数（完全競争では R(x)=P·x）、C(x)＝費用関数
完全競争企業は プライステイカー（価格を市場から受け入れる）。よって P は与件で、決めるのは生産量 x のみ。
📊 収入 R(x)・費用 C(x)・利潤 π(x) の関係
📷 対応する手書きノート/教科書ページ：IMG_2404
📖 利潤＝収入−費用 の3曲線
  • 緑（R）＝収入：価格P × 生産量x の直線（プライステイカーなのでPは固定）
  • 赤（C）＝費用：固定費FCから始まる3次曲線
  • 紫（π）＝利潤＝R−C：上に凸の曲線。頂点が最適点 x*
  • x* では RとCの縦差が最大＝Rの傾き(=P)とCの傾き(=MC)が等しい点

📘 利潤最大化の数学的導出

1階条件：π'(x) = 0

R(x) = P·x なので R'(x) = P。
π'(x) = R'(x) − C'(x) = P − MC(x) = 0

P = MC(x) 限界収入＝限界費用 → 利潤最大の条件

📘 完全競争の供給曲線

📐 供給曲線＝MC曲線（AVC最小値以上の部分）

各価格Pに対して、企業は P=MC を満たす生産量を選ぶ。
→ MC曲線そのものが供給曲線になる。
ただし、P<AVC最小値 では操業停止（生産量0）。

📖 利潤の2つの読み方（同じ値を別の角度から）
  • ① 総額グラフ（入門タブの R(x)・C(x) 図）：x* で R−C の縦差（青バーの長さ） ＝利潤
    R(x*) = P·x*（緑線の高さ）、C(x*) ＝赤曲線の高さ。差がそのまま利潤総額。
  • ② 単位あたりグラフ（下の IMG_2405 のような MC/AC 図）：(P − AC) × x* ＝ 長方形の面積 ＝利潤
    STEP① P=MC で x* を決める／STEP② x* での AC 高さを読む／STEP③ 縦(P−AC) × 横 x* の長方形面積
  • どちらも同じ値： π = R − C = (P − AC)·x*。総額図では「線分の長さ」、単位あたり図では「長方形の面積」として現れる。
📷 対応する手書きノート/教科書ページ：IMG_2405（MC/AC 図と利潤長方形）
📊 限界利潤の累積：P と MC の間の面積（IMG_2400 の図）
📷 対応する手書きノート/教科書ページ：IMG_2400（青ハッチング＝利潤の増加、赤ハッチング＝過剰生産で減じる）
📖 グラフの読み方（IMG_2400 の図解と対応）
  • 緑色の面積 ＝ 0 から x* までの増分利潤の累積（P と MC の差を積分した値）。
    1個ごとに「売値 P − 1個追加するコスト MC」分だけ利潤が増えていき、その合計。
  • x* で P = MC となり、追加1個分の利潤が 0 になる ＝ これ以上作っても得しない最適点
  • x* を 超えて 生産すると MC > P。1個ごとに利潤がマイナスされる。
    赤色の面積 ＝ 過剰生産で削られる利潤（緑の累積から差し引かれる）
  • 結論：緑の面積を最大化するのが x*。それより少なくても多くても利潤は縮む。

🔍 利潤の図解

P と AC の関係状態理由
P > AC最小値黒字（利潤プラス）1個あたり総費用を超える価格
P = AC最小値損益分岐利潤ゼロ（操業継続）
AVC最小 < P < AC最小赤字でも操業FCの一部を回収できる
P < AVC最小値操業停止VCすら回収不能

🔍 利潤最大化の経済数学

具体例：C(x) = 3x² + 2x + 5、P = 14

  • MC(x) = 6x + 2
  • P = MC ⇒ 14 = 6x + 2 ⇒ x* = 2
  • 利潤 π = 14·2 − (3·4+2·2+5) = 28 − 21 = 7
📊 価格Pの位置で「黒字／損益分岐／赤字操業／操業停止」が決まる
📷 対応する手書きノート/教科書ページ：IMG_2401
📖 4ケースの判定ルール（価格Pの位置で決まる）
  • ①P > AC最小：黒字、長期も継続
  • ②P = AC最小：損益分岐（利潤=0）
  • ③AVC最小 < P < AC最小：赤字だが 固定費の一部回収できるので操業継続
  • ④P < AVC最小：VC すら回収不可 → 操業停止

💼 3つの重要点の「経営上の意味」

状態・点 収益（売上）でカバーできている範囲 経営上の意味（どんな状態？）
限界費用 MC 最小点
（MCの底）		 「追加の1個」を最も安く作れている状態（売上1個あたりの粗利が最も拡大する瞬間）。
※ただし、材料費などの直接的な費用は 一部のみ を賄えている状態。		 費用の増え方が最小になる点（生産効率MAX）。習熟効果などで効率が上がりきった状態。ここを過ぎると、設備や人手の限界で1個あたりの追加コストが増え始める。
操業停止点 P=AVC最小
（黄信号の境界）		 「材料費などの直接的な費用」だけを賄えている。家賃などの 固定費は全く賄えていない。 会社を動かす「最低限の条件」。これより売上が低いと、材料費すら一部しか賄えず、作れば作るほど赤字が膨らむため 即座に休業すべき。
損益分岐点 P=AC最小
（プラス転換ライン）		 「すべての費用（材料費＋家賃など）」を賄えている。
赤字も黒字もない トントンの状態。		 「利益が0」になる目標地点（固定費を賄える点）。ようやく固定費を全て回収できた瞬間。ここから先が本当の「儲け」 になり、経営が安定し始める境界線。
📊 価格水準で見る「費用カバーの段階」
一部のみ 賄える＝即休業すべき水準 ✓ 黒字ゾーン（利潤あり） ⚠ 赤字でも操業継続（FC一部回収） ✗ 即休業（VC すら回収不可） 📖 価格Pがどの帯にあるかで カバー範囲 と 取るべき行動 が決まる 緑帯：FC+VC全カバー → 黒字　　黄帯：VCのみ回収 → 赤字でも操業 → 短期はOK　　赤帯：VCも未満 → 即停止 📌 MC最小点(P_MC)は「効率が最も良い生産量」を示すが、収益が低すぎるため操業判断とは別軸の指標

  • 利潤＝収入−費用：π(x) = R(x) − C(x)
  • 利潤最大化条件：P = MC(x)（限界収入＝限界費用）

❓ ミニクイズ

Q1
完全競争企業の利潤最大化条件として正しいのはどれか。
  • P = AC
  • P = MC
  • MR = AC
  • MR > MC
完全競争では P = MR なので、利潤最大化条件 MR=MC は P=MC となる。
