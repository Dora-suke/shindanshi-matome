😊 効用（うれしさ）と限界効用

🧒
学習者
「効用」ってなに？
🧑‍🏫
先生
消費から得る満足度（うれしさ）のこと。最初は「うれしさ＝点数」と思ってOK。点数が同じなら「どっちでもいい＝無差別」です。
📊 ケーキを1個ずつ食べたときの「追加のうれしさ」（限界効用逓減）
1個目が一番うれしく、追加の喜びは少しずつ減っていく＝限界効用逓減（ていげん）の法則。

効用関数 U の例（U = √x）

消費量 x の関数で書くと、よく使うのが U = √x。x=1→U=1、x=4→U=2、x=9→U=3。
「増えるけど、増え方は段々鈍る」＝限界効用逓減の典型。

📊 効用関数 U=√x（凹関数）

限界効用 MU の意味

MU＝Marginal Utility。X財に関する限界効用は MUx。数学的には効用関数Uをxで微分した U'(x)＝U(x)曲線の接線の傾き。

MUx = dU/dx X財を1単位多く消費したときに追加で得るうれしさ
📊 効用Uと限界効用MU の対比
①効用 → ②無差別曲線 → ③限界代替率 → ④効用最大化 → ⑤特殊型 → ⑥需要導出 → ⑦所得/価格効果

📈 無差別曲線（同じ効用の組合せ）

X財とY財の組合せのうち、効用が同じになる点を結んだ線。山（効用の曲面）を上から見た等高線にあたる。

性質意味
右下がり（単調性）X財を増やすならY財を減らす必要がある
原点に対して凸偏った消費より、両方バランスよく消費する方が満足
交わらない効用水準が同時に2つになることはない
📊 無差別曲線（U₁ < U₂ < U₃）
右上に位置する無差別曲線ほど効用が高い。複数の曲線が交わらないのは、同じ点で効用が2つになるのは矛盾だから。
①効用 → ②無差別曲線 → ③限界代替率 → ④効用最大化 → ⑤特殊型 → ⑥需要導出 → ⑦所得/価格効果

🔄 限界代替率（MRS）

MRS = Marginal Rate of Substitution

無差別曲線上で、X財を1単位増やしたときに諦めるY財の量。数式では無差別曲線の接線の傾きの絶対値。

MRS = -ΔY/ΔX = MUx / MUy 限界代替率＝限界効用の比

📌 導出の直観：効用一定（dU=0）のとき MUx·dX + MUy·dY = 0 → -dY/dX = MUx/MUy。

📊 限界代替率＝接線の傾き
限界代替率逓減の法則：X財が増えるほどMRSは小さくなる。
|ΔY₁/ΔX₁| > |ΔY₂/ΔX₂|（X財が増えるほど、諦めるY財は少なくなる）
📊 限界代替率逓減（点①は急、点②は緩い）
①効用 → ②無差別曲線 → ③限界代替率 → ④効用最大化 → ⑤特殊型 → ⑥需要導出 → ⑦所得/価格効果

💸 予算制約線

所得 M、価格 Px・Py のとき

X財をx個、Y財をy個買うとき支出は Px·x + Py·y。これを所得Mに収める。

Px·x + Py·y = M y = -(Px/Py)·x + M/Py 傾き＝-(Px/Py)（価格比のマイナス）／y切片＝M/Py
📊 予算制約線の構造（傾きと切片）

🎯 効用最大化の条件

📐 接点条件（最重要）

予算制約線と無差別曲線の接点が最適消費点。そこでは「予算線の傾き＝無差別曲線の傾き」が成り立つ。

MRS = MUx/MUy = Px/Py 限界代替率＝価格比
📊 予算制約と無差別曲線の接点（最適消費点）
予算を使い切る点のうち、最も右上の無差別曲線に届くのが接点。そこで MRS＝価格比 が成立する。
①効用 → ②無差別曲線 → ③限界代替率 → ④効用最大化 → ⑤特殊型 → ⑥需要導出 → ⑦所得/価格効果

🔍 特殊な無差別曲線（4タイプ）

📊 4タイプを並べて比較（標準／L字／直線／凹）

① 完全補完財（L字型・レオンチェフ型）

X財とY財を固定比率（α:β）でしか使わない財。少ない方が増えなければ効用は上がらない。無差別曲線はL字型、各L字の角を結ぶ線が最適消費の通り道。

U = min(αX, βY)例：U=min(X,Y)。少ない方の値が効用

例：右手と左手の手袋（1:1）、車とガソリン、コーヒーと砂糖

📊 完全補完財の無差別曲線と最適消費点

② 完全代替財（直線）

X財α個とY財β個が完全に同じ価値。MRSは常に一定（α/β）→無差別曲線は直線。

U = αX + βY線形効用関数。傾き-α/βの平行な直線群

例：コーラとペプシ、500円玉2枚と1000円札

📊 完全代替財：3ケースで「どこを選ぶか」が決まる

③ 原点に凹（角点解）

両方少しずつより、どちらか一方に集中した方が満足度が高い財。無差別曲線は原点に対して凹（外側に膨らむ）、MRSは逓増。最適は軸上の角点解。

U = X² + Y² など原点から遠いほど高い・凹形状

例：ビールとカレー、ロックとクラシック（同時だと満足度↓）

📊 原点に凹：最適消費点はX軸またはY軸上（角点解）

📐 4形状の総括

形状効用関数MRS最適消費点例
原点に凸（標準）U=X^a·Y^b（コブ＝ダグラス）逓減内点（接点）普通の財
L字（完全補完）U=min(αX,βY)不連続L字の角左右の靴
直線（完全代替）U=αX+βY一定角点 or 任意点コーラ↔ペプシ
原点に凹U=X²+Y²逓増角点（軸上）ビール↔カレー
①効用 → ②無差別曲線 → ③限界代替率 → ④効用最大化 → ⑤特殊型 → ⑥需要導出 → ⑦所得/価格効果

📉 需要曲線の導出（4ステップ）

  1. X財の価格 Px を段階的に下げる（Px₁ → Px₂ → Px₃）
  2. そのたびに予算制約線が x切片側に右へ拡大（回転）する（y切片 M/Py は不変）
  3. 新しい予算制約と無差別曲線の接点（最適消費点）が右へシフト
  4. 価格と消費量の組合せ（Px, x）をプロット → 右下がりの需要曲線
📊 需要曲線の導出：上下2段の対応
予算制約の「シフト」と「回転」は区別！
所得が変わると平行シフト、価格が変わると回転（一方の切片だけ動く）。
📊 シフト vs 回転の比較
①効用 → ②無差別曲線 → ③限界代替率 → ④効用最大化 → ⑤特殊型 → ⑥需要導出 → ⑦所得/価格効果

🔬 スルツキー分解（価格効果＝代替効果＋所得効果）

分解の手順

  1. 価格変化後の予算線を引く
  2. それに平行で、元の無差別曲線 U₁ に接する仮想予算線を引く
  3. 仮想予算線とU₁の接点を x'（代替効果による点）
  4. 新予算線と新無差別曲線 U₂ の接点を x₂（最終点）
代替効果＝x' − x₁、所得効果＝x₂ − x'、価格効果＝x₂ − x₁
