🛒 所得が増えると消費はどう増える？

🧒
学習者
収入が増えたら、その分まるごと使うわけじゃないよね？
🧑‍🏫
先生
そう。所得が1円増えると、消費はそのうち c円（0<c<1） だけ増える。この c が 限界消費性向。残り (1−c) は貯蓄に回ります。

🐣 つかみ

  • 消費 C ＝ 独立消費 C₀（所得ゼロでも必要な最低消費）＋所得に比例する消費
  • 限界消費性向 c＝所得が1円増えたときに消費が増える額（傾き）
  • 残り (1−c)＝限界貯蓄性向 s（c＋s＝1）

ケインズ型消費関数（絶対所得仮説）

C = c·Y + C₀ C₀＝独立消費（最低限必要な消費水準） c＝限界消費性向（0<c<1）
税金 T を考慮すると C = c(Y−T) + C₀（可処分所得 Y−T に依存）。
📊 ケインズ型消費関数の構造

貯蓄関数（消費の裏返し）

S = (1−c)·Y − C₀ ＝ s·Y − C₀s＝限界貯蓄性向＝1−c

所得 Y のうち消費に回さない分が貯蓄 S。所得が小さいうちは S が負（貯蓄を取り崩す）、C=Y 点で S=0、それより右で S が正になる。

①消費関数 → ②45度線・均衡 → ③乗数 → ④デフレ/インフレギャップ

⚖️ 国全体の所得はどこで決まる？

🧒
学習者
「国民所得が決まる」ってどういうこと？
🧑‍🏫
先生
作った量（総供給 Y_S）と買いたい量（総需要 Y_D）が一致する所得で落ち着く。これが 均衡国民所得 Y*。総供給は「Y_S＝Y」なので 45度線で表せます。

🐣 つかみ

  • 総供給 Y_S＝生産面から見たGDP＝Y（だから 45度線）
  • 総需要 Y_D＝C＋I＋G（＋EX−IM）。傾きは c（45度より緩い）
  • 均衡：Y_S = Y_D の交点が 均衡国民所得 Y*

均衡国民所得 Y* の導出（閉鎖経済 EX−IM=0）

  1. Y_D = c(Y−T) + C₀ + I + G
  2. 均衡：Y_S = Y_D ⇒ Y = c(Y−T) + C₀ + I + G
  3. 整理：(1−c)Y = C₀ + I + G − cT
Y* = (C₀ + I + G − cT) / (1−c) 1/(1−c)＝乗数（次のタブで詳説）
📊 45度線分析（Y_S=Y_D で均衡国民所得 Y* が決まる）
📖 45度線分析の読み方
  • 黒の45度線：「総供給＝総生産＝Y」を表す（傾き1）
  • 赤の Y_D 線：総需要。傾き c（0<c<1）なので45度線より緩い
  • ⭐青丸：両線の交点＝均衡国民所得 Y*
  • Y切片 A=C₀+I+G−cT（独立支出の合計）が大きいほど Y* も大きくなる
①消費関数 → ②45度線・均衡 → ③乗数 → ④デフレ/インフレギャップ

🔁 1の支出が何倍にもなる「乗数効果」

🧒
学習者
10億円の公共事業で、なぜ所得が10億円以上増えるの？
🧑‍🏫
先生
受け取った人が、その一部を 消費する。すると別の人の所得になり、また一部が消費される…この連鎖で 1/(1−c) 倍 に膨らむ。これが乗数効果です。

📘 各種乗数

政策乗数意味
政府支出 G が ΔG 増1/(1−c)政府支出乗数
投資 I が ΔI 増1/(1−c)投資乗数（Gと同じ）
租税 T が ΔT 増−c/(1−c)租税乗数（マイナス）
減税 T が ΔT 減+c/(1−c)減税の効果
均衡予算乗数の定理：ΔG = ΔT のとき所得の変化は ΔG に等しい（乗数＝1）。
＝政府支出乗数 1/(1−c) と租税乗数 −c/(1−c) を足すと (1−c)/(1−c)=1。
📊 乗数効果の波及（c=0.8 → 乗数=5 の場合）
📖 乗数効果の仕組み
  • 政府が10億円の公共事業 → 受注企業の従業員の所得が10億円増（①）
  • その人達が 所得の80%（c=0.8） を消費 → 別の企業に8億円の収入（②）
  • 連鎖して 1+0.8+0.8²+… = 1/(1−0.8) = 5倍 に膨らむ
  • これが 政府支出乗数 1/(1−c) の正体（c が大きいほど乗数も大きい）
①消費関数 → ②45度線・均衡 → ③乗数 → ④デフレ/インフレギャップ

📏 「完全雇用」とのズレを縦で測る

🧒
学習者
働きたい人が全員働ける所得（完全雇用 Y_F）と、実際の均衡 Y* がズレるとどうなるの？
🧑‍🏫
先生
Y_F の所で需要が足りなければ デフレギャップ（失業）、需要が多すぎれば インフレギャップ（物価上昇）。どちらも Y_F での 縦方向の差で測ります。

🐣 つかみ（縦方向で測る）

  • デフレギャップ＝Y_F で総需要が 不足する高さ → 失業（非自発的失業）
  • インフレギャップ＝Y_F で総需要が 超過する高さ → 物価上昇圧力
  • 覚え方：均衡 Y* が Y_F より 右ならインフレ（「右でサイン」）
📊 デフレギャップ（Y_F で需要が不足 → 失業）
📊 インフレギャップ（Y_F で需要が超過 → 物価上昇）

🔍 デフレ vs インフレ 比較

デフレギャップ

総需要 Y_D が完全雇用国民所得 Y_F より 少ない状態。
→ 失業発生（非自発的失業）
→ 政策：公共事業 G↑、減税 T↓、投資促進 I↑

インフレギャップ

総需要 Y_D が完全雇用国民所得 Y_F より 多い状態。
→ 物価上昇圧力
→ 政策：緊縮財政 G↓、増税 T↑

📖 覚え方：「右でサイン（イン）」
  • 均衡 Y* が Y_F より 左（少ない）＝デフレギャップ（需要不足・失業）
  • 均衡 Y* が Y_F より 右（多い）＝インフレギャップ（需要過剰・物価上昇）
  • ギャップはいずれも Y_F での タテ方向の差で測る
