📘 弾力性で財の性質が見える

🐣 「価格が動くと需要・供給はどれだけ動く？」

🧒
学習者
同じ値上げでも、お米とブランド品では消費の減り方が違いそう。
🧑‍🏫
先生
その「動きやすさ」を測るのが 弾力性。必需品は弾力性小（あまり動かない）、奢侈品は弾力性大（よく動く）。

🐣 つかみ

  • 需要曲線＝右下がり（買いたい：価格が高いほど少ない）
  • 供給曲線＝右上がり（売りたい：価格が高いほど多い）
  • 弾力性＝価格1%変化に対して何%量が変わるか

📘 弾力性の定義

需要の価格弾力性 ε

ε = -(ΔD/D) / (ΔP/P) = -(ΔD/ΔP)·(P/D) 価格が1%上がったとき需要量が何%減るか（マイナス符号で正の値にする）
  • ε > 1：弾力的（奢侈品、ブランド品）
  • ε = 1：単位弾力的
  • ε < 1：非弾力的（必需品）

需要の所得弾力性 η

η = (ΔD/D) / (Δm/m) 所得が1%増えたとき需要量が何%変わるか
η分類
η > 0上級財
η = 0中立財
η < 0下級財
📊 弾力性の5パターン（傾きで判別）
📷 対応する手書きノート/教科書ページ：IMG_2410
📖 一言で覚える
  • 傾きが垂直＝価格変化に動かない＝ε=0（医薬品など代替がきかない財）
  • 傾きが急＝ε<1（必需品）／傾きが緩＝ε>1（奢侈品）
  • 傾きが水平＝価格を1円でも上げたら需要0＝ε=∞（完全競争市場の単一企業）

🔍 需要曲線の傾きと弾力性

📐 「傾きが急＝弾力性小」「傾きが緩い＝弾力性大」

同じ需要量Dでも、需要曲線の 傾き によって弾力性が変わる。
縦軸（P）と横軸（D）の取り方に注意：通常の需要曲線は 縦P・横D なので、傾き ΔP/ΔD を使うときは逆数になる。

傾き弾力性典型例
急（垂直に近い）小（非弾力的）必需品（米、塩、医薬品）
緩い（水平に近い）大（弾力的）奢侈品、代替財が多い財

🔍 弾力性と総収入の関係

値上げで売上は増える？減る？

  • 非弾力的（ε<1）：値上げ → 売上 増加（必需品なので量があまり減らない）
  • 単位弾力的（ε=1）：値上げ → 売上 不変
  • 弾力的（ε>1）：値上げ → 売上 減少（量が大きく減る）
📊 値上げで総収入はどう変わる？（長方形＝P×Q＝売上）
📷 対応する手書きノート/教科書ページ：IMG_2410
📖 売上分析の読み方
  • 売上＝価格 × 数量＝長方形の面積（縦＝価格、横＝数量）
  • ①非弾力的：値上げで縦が伸びるが横はあまり減らない → 面積が 増える
  • ②単位弾力的：縦の伸びと横の減りがちょうど釣り合う → 面積が 変わらない
  • ③弾力的：値上げで横が大きく減る → 面積が 減る
  • 経営的示唆：必需品(ε<1)を扱う企業は値上げ可能、奢侈品(ε>1)は値上げ慎重に

🔍 供給の価格弾力性

需要側だけでなく供給側にも弾力性

供給の価格弾力性 ε_s ＝ (ΔS/S) / (ΔP/P)。価格1%上昇で供給量が何%増えるか。

  • 短期：弾力性は小さい（生産能力に余裕がない）
  • 長期：弾力性は大きい（設備投資で増産可能）
  • 農産物：短期は非弾力的（収穫期固定）、長期は弾力的

✏️ 穴埋め演習

基礎
① 需要曲線は 右下がり、供給曲線は 右上がり
② 価格1%変化に対する需要量の%変化を 需要の価格弾力性 という。
③ 必需品は弾力性が 小さい。
④ 所得弾力性 η > 0 の財を 上級財 という。
⑤ ε < 1（非弾力的）の財は値上げで総収入が 増加 する。

❓ ミニクイズ

Q1
価格が10%上昇し、需要量が5%減少した。需要の価格弾力性 ε は？
  • 2.0（弾力的）
  • 0.5（非弾力的＝必需品的）
  • 1.0（単位弾力的）
  • -0.5
ε=-(-5%)/(10%)=0.5。1未満なので非弾力的。値上げで総収入は増える。
Q2
所得弾力性が負である財はどれか。
  • 上級財
  • 中立財
  • 下級財
  • 完全代替財
所得が増えると消費が減る財＝下級財（η<0）。
