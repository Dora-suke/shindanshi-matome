🔍 倹約のパラドックス／恒常所得仮説

📕 倹約のパラドックス

貯蓄意欲↑ → 消費↓ → 均衡GDP↓ → 貯蓄も増えない。合成の誤謬。
R05追試-7-2 出題。

📕 恒常所得仮説（フリードマン）

消費は一時所得ではなく 恒常所得 で決まる → 一時的給付は乗数効果が小。R03-4 出題。

🎭 暗記用 語呂・フレーズ集

① 政府支出乗数 1/(1−c)

「いっぴき・いんちのc」1/(1−c)＝政府支出乗数

「1ぴき」＝分子の1。「1−c」＝c=0.8なら0.2、で「1割る0.2＝5」と即計算。

② 租税乗数 −c/(1−c)

「税金マイナスcぶんのc」租税乗数の符号はマイナス

分子に「c」が乗る＝政府支出乗数より「c倍」分小さい、というイメージ。

③ 均衡予算乗数定理 = 1

「ジー＝ティーなら、ぴったり1」ΔG=ΔTなら乗数=1（cによらず）

どんなcでも常に1。これは政府支出乗数 − 租税乗数の絶対値 = (1−c+c)/(1−c) ÷ ... と暗算しなくていい。

④ 開放経済の乗数 1/(1−c+m)

「輸入のmで漏れて、乗数小さく」分母が大きくなる→乗数縮小

⑤ デフレ・インフレギャップ

「不足デフレ／超過インフレ」需要不足＝デフレ、需要超過＝インフレ

デフレ＝需要が足りなくて失業／インフレ＝需要が多すぎて物価上昇。

⑥ 倹約のパラドックス

「みんなで節約、所得も貯蓄も消える」合成の誤謬

⑦ ケインズ型消費関数 C=C₀+cY

「コゼロたすシーワイ」基礎消費＋限界消費性向×所得

🎵 語呂・フレーズ穴埋め

📝 語呂穴埋め
① 政府支出乗数 ＝ 1/(1−c)
② 租税乗数 ＝ −c/(1−c)。符号は マイナス。
③ ΔG=ΔTの均衡予算乗数定理は 1（cによらない）。
④ 開放経済の乗数 ＝ 1/(1−c+m)。mは 限界輸入性向。
⑤ ケインズ型消費関数 C＝C₀＋cY。
⑥ 完全雇用GDPに対し均衡GDPが小さい→デフレギャップ。大きい→インフレギャップ。
⑦ c=0.8 のとき政府支出乗数＝5、租税乗数＝−4。
⑧ c=0.75 のとき政府支出乗数＝4、租税乗数＝−3。
⑨ 個人の貯蓄が社会全体ではGDPを下げる現象を 倹約のパラドックス という。
⑩ 累進課税・失業保険などの景気自動安定化装置を ビルトイン・スタビライザー という。

✏️ 穴埋め問題（初級・中級・上級）

📝 初級（用語の確認）
① ケインズ型消費関数：C = C₀ + cY。
② cは 限界消費性向、(1−c)は 限界貯蓄性向。
③ 45度線は「生産＝所得」を表す。
④ 45度線と総需要曲線の交点が 均衡国民所得。
⑤ 政府支出乗数 ＝ 1/(1−c)。
📝 中級（公式と性質）
① 租税乗数 ＝ −c/(1−c)、符号は マイナス。
② 均衡予算乗数定理：ΔG=ΔTのとき乗数 ＝ 1（cによらない）。
③ 開放経済の乗数 ＝ 1/(1−c+m)。mは限界 輸入性向。
④ 完全雇用GDPで需要不足の状態を デフレギャップ、需要超過を インフレギャップ という。
⑤ デフレギャップ解消には 政府支出増・減税 など総需要拡大政策。
📝 上級（計算と応用）
① c=0.8、政府支出を10兆円増→ ΔY = 50兆円。
② c=0.75、減税1兆円→ ΔY = 3兆円（=1×0.75/0.25）。
③ c=0.8、ΔG=ΔT=10兆円 → ΔY = 10兆円（均衡予算乗数定理）。
④ デフレギャップ50兆円、c=0.8 → 必要なΔG = 10兆円。
⑤ c=0.8、m=0.2 の開放経済の乗数 = 2.5（=1/(1−0.8+0.2)=1/0.4）。
⑥ 一時的所得は消費に回りにくいというのは 恒常所得仮説（フリードマン）。

🏢 事例で学ぶ乗数効果

📕 事例1：100億円の公共事業が地域に与える効果

A県が100億円の道路工事を発注。地域の限界消費性向 c=0.7 とする。

  • 第1段階：建設会社の所得 +100億円
  • 第2段階：従業員が70億円消費 → 商店に+70億円
  • 第3段階：商店主が49億円消費 → +49億円
  • … 無限級数の和 ＝ 100/(1−0.7) ＝ 約333億円のGDP増

気づき：地方に消費性向の高い地域があれば、同じ財政出動でも効果が大きい。

📕 事例2：定額給付金10万円（2009年・2020年）

国民1人10万円配布。総額 約12兆円。

  • 租税乗数 c/(1−c)。実証研究では実質的な c は 0.2〜0.3 と低かった（恒常所得仮説）
  • → 実際のGDP押し上げ効果は理論値の半分以下と試算された
  • 「一時的給付」は消費喚起効果が限定的、という現代経済学の知見

📕 事例3：消費税増税の影響（2014年5%→8%、2019年8%→10%）

租税乗数 −c/(1−c) によりGDPが減少。実際、増税後の四半期GDPは大幅マイナス。

対策として「軽減税率」「ポイント還元」などで消費の落ち込みを緩和（疑似的な減税）。

📕 事例4：コロナ禍の財政出動（2020年）

日本では補正予算 約57兆円超。乗数効果でGDP水準を維持しようとした。

ただし将来不安で c が低下、貯蓄率が急上昇 → 倹約のパラドックス が現実化。

📕 事例5：開放経済の輸入漏れ

給付金で家電を購入 → 海外メーカーの製品（輸入）→ 日本のGDPには寄与しない。

「Made in Japan」の購入が乗数効果を高める、という政策的含意（地域振興券・買い物券）。

⭐ 今日のこれだけ覚えて（事例）
  • ❶ 地方の公共事業＝消費性向が高ければ乗数効果も大きい
  • ❷ 一時給付金＝恒常所得仮説により乗数効果が低い（理論値より小）
  • ❸ 輸入漏れ＝開放経済では乗数 1/(1−c+m) で縮小する

💡 身近な実例で「乗数効果」を体感

🏗 ニューディール政策（米国・1933〜）

大恐慌対策としてフランクリン・ルーズベルトが大規模公共事業（テネシー川流域開発など）を実施。ケインズ理論の実証実験と評される。失業率は25%→15%程度まで改善。

🏛 日本の補正予算（毎年恒例）

景気対策として数兆〜数十兆円規模の補正予算が組まれる。1兆円の公共事業で c=0.6 なら2.5兆円のGDP押上効果と試算される。

🛍 地域振興券・プレミアム付商品券

地域内でしか使えないため 輸入漏れを防ぐ工夫。乗数効果を最大化する政策設計。

📉 リーマンショック後の財政出動（2008〜2009）

米国「アメリカ復興・再投資法」7870億ドル、日本「定額給付金」12兆円。乗数の理論を実際の政策で検証した時代。

🦠 コロナ禍の経済対策（2020〜2022）

日本：3度の補正予算で総額100兆円超。米国：CARES法2.2兆ドル。「乗数効果」「ヘリコプターマネー」が主要キーワードに。

💴 消費税の景気影響

2014年5→8%、2019年8→10%の増税後、いずれもGDPが大きく落ち込み（租税乗数効果）。経済学のテキストの予測と一致した実例。

🖼 主要な点

   Y_S, Y_D
      ↑                           Y_S=Y（45度線）

  • 横軸：国民所得 Y、縦軸：総需要 Y_D / 総供給 Y_S
  • 45度線（傾き1）と総需要曲線（傾きc）の交点が 均衡GDP Y*
  • 独立需要 A が増えると、総需要曲線が上方シフト → Y* が 1/(1−c) 倍 増加

📐 デフレ／インフレギャップの図イメージ

[デフレギャップ]         [インフレギャップ]
   Y_S=Y                    Y_S=Y
    /  Y_D'(完全雇用に必要) /
   /  /                    /  Y_D(現状)
  / /                     / /
 //  ↕デフレ              //  ↕インフレ
 |   ギャップ             |    ギャップ
 |  Y_D(現状)            |   Y_D'(完全雇用に必要)
 |                        |
 +---Y*--Y_F---→ Y       +---Y_F--Y*---→ Y
  • デフレギャップ：Y* < Y_F。Y_F水準で「需要が足りない縦の距離」。失業発生。
  • インフレギャップ：Y* > Y_F。Y_F水準で「需要が超過している縦の距離」。物価上昇。
  • 解消にはギャップ × (1−c) ＝ 必要な財政出動額（または増税額）
