1業況判断DIの読み方

🔁 入門タブの「DI＝好転％−悪化％」を試験論点で深掘り。

📐 DIの計算式
DI ＝ 「好転」と答えた企業の割合（％） − 「悪化」と答えた企業の割合（％）
※「不変」は計算に含まない／「前年同期比」で判定
  • DI > 0 → 好転超過（景気が良い方向）
  • DI < 0 → 悪化超過（景気が悪い方向）
  • 白書の業況判断DIは長期にわたりマイナス圏で推移

📝 【1次】試験ポイント：DIの計算式（好転−悪化）が頻出。「不変」を含めるか／「前年同期比 vs 前期比」の違いに注意。

⚠️ 【1次】引っかけ：「DI＝好転％＋悪化％」「不変も計算に含む」は誤り。正は好転−悪化。

「DI＝好転−悪化」って単純なんだけど、選択肢で「足し算」にして引っかけてくる年がある。「マイナスにもなる指標＝引き算」と覚えれば一発で切れる！

5外部環境（金利・為替・物価）の影響

2025年版白書は「金利のある世界」の到来を強調。中小企業の業績に与えうる影響：

  • 政策金利上昇 → 借入金の支払利息増加 → 経常利益の下押し
  • 円安・物価高 → 輸入比率の高い中小にはコスト増のリスク
  • 労働分配率は8割近く → 賃上げ余力は厳しい
  • 対策：価格転嫁＋設備投資・デジタル化で付加価値・労働生産性向上
📚 出典：2025年版中小企業白書 はじめに／第1部総論／第2節 金利・為替・物価

📝 【1次】試験ポイント：白書のメッセージは「コストカット戦略から付加価値向上戦略への転換」。文脈設定の選択肢で出やすい。

  1. DI＝好転％−悪化％。マイナスにもなる引き算の指標
  2. 規模区分は「10億／1億／1千万」の資本金ライン（基本法とは別物）

1大企業との「差の拡大」を理解する

🔁 基本タブの「差は拡大」の根因を分解。

指標大企業中小企業（中規模＋小規模）差
業況判断DIマイナス幅小マイナス圏が定常恒常的なギャップ
売上高顕著な増加増加だが緩やか拡大
経常利益顕著な増加増加だが大企業に劣後拡大
労働生産性増加傾向横ばい（30年前比で減）拡大
🔍 差が拡大する3大要因（白書の整理）
  1. 価格転嫁力の差：大企業は原材料費・労務費を価格に転嫁しやすい
  2. 設備投資・デジタル化の差：大企業の方が設備・DX投資が進む
  3. 労働分配率の差：中小は約8割と高く、利益への振り向けが難しい

📝 【1次】試験ポイント：「中小は大企業を上回る労働生産性」と書かれた選択肢に注意。平均ではNGだが、個別企業では大企業を上回る中小が一定数存在する（白書の累積分布図）。

2労働生産性の累積分布（白書の独自論点）
📊 第1-1-33図（累積分布）
労働生産性を100万円/人ごとに区切って累積分布を見ると、大企業の平均値を上回る中小企業も一定数存在する。
  • 平均だけ見ると「大企業＞中小企業」だが、分布の上位帯では中小も大企業並みに高い
  • 「中小だから生産性が低い」と単純化はできない
  • 白書のメッセージ：「中小内の格差」にも目を向けるべき
📚 出典：2025年版中小企業白書 第1-1-33図／経済産業省「企業活動基本調査」

📝 【1次】試験ポイント：「大企業の平均値を上回る中小企業が存在する」（正）／「全ての中小は大企業に劣る」（誤）の引っかけが頻出。

1事例①：建設業 田中工務店

登場：地方の建設業 田中工務店（資本金3,000万円）の社長 田中さん。

💬 田中さんの実感：「公共工事の発注が増え、足下の経常利益は伸びています。ただ人手不足と資材高で利益は思ったほど残らない…」

👨‍💼 診断士の解説

  • 白書（第1-1-10図）でも建設業は経常利益が上昇傾向と整理されている
  • ただし労働生産性は横ばい（第1-1-32図）→ 売上拡大に対し人手不足で生産性が伸びない構造
  • 対策の方向：設備投資・デジタル化で1人当たりの付加価値を高める
2事例②：飲食店 鈴木食堂

登場：個人経営の飲食店 鈴木食堂（資本金なし／個人事業主）。

💬 鈴木さんの実感：「コロナ後にお客さんが戻ったけど、原材料費とエネルギーコストで利益が出ません。値上げは怖い…」

👨‍💼 診断士の解説

  • 白書（第1-1-10図）で宿泊業、飲食サービス業の経常利益は伸び悩みと名指し
  • 労働生産性もサービス業の伸びが小さい（第1-1-32図）
  • 白書のメッセージ：適切な価格転嫁＋付加価値向上で「コストカット→付加価値向上戦略」へ転換
3事例③：製造業 佐藤精機

登場：佐藤精機（資本金8,000万円・従業員80人）。中規模企業。

💬 佐藤社長の悩み：「業況DIはプラスに転じたが、大企業の取引先と比べると当社の利益率はあまり改善しない。なぜ？」

  • 累積分布（第1-1-33図）では大企業平均を上回る中小も存在 → 個別の経営力で差はつく

1実例：33年ぶりの賃上げ（2024年春闘）

2024年春季労使交渉では、約33年ぶりの高水準賃上げが実現。連合集計の賃上げ率は5%超。

  • 中小は労働分配率がすでに約8割で賃上げ余力が厳しい
  • 白書は「業績改善を伴わない賃上げ」も増えたと指摘
  • 結果：大企業との賃金格差は拡大

2実例：「金利のある世界」の到来

2024年3月、日本銀行はマイナス金利政策を解除し、約30年ぶりに政策金利を引き上げました。

  • 白書の試算：政策金利上昇は中小企業の収益を圧迫しうる
  • 輸入比率・借入金依存度が高い中小には特に影響
  • 対策：価格転嫁の推進＋設備投資・DXで生産性向上
📚 出典：2025年版中小企業白書 第1部第1章第2節（金利・為替・物価）
