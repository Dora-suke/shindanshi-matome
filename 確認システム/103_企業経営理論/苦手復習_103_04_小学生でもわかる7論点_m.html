🌱 まずはたとえ話

「シナジー」＝2つの仕事を一緒にやると、バラバラにやるより得をすること（相乗効果）。

たとえば、あなたがレモネード屋さんとクッキー屋さんを両方やるとする。それぞれの「道具・場所・コツ」を貸し借りすると、1人で2つやってるのに、すごく効率がよくなるよね。

🍋 ＋ 🍪 ＝ 💪 (1+1が2より大きい！)

アンゾフ先生は、その「貸し借りできるもの」を4種類に分けたんだ。これが「シナジーの4類型」。

🔑 強み（場所・道具・お金・コツ）の貸し借りで「合計より大きく」なる

図：シナジーの4類型（アンゾフ）を屋台でイメージ

🍋 レモネード＋🍪 クッキー=2より大きい！
🏪① 販売シナジー 同じお店・お客さんを使い回す。レモネードを買いに来た子にクッキーも売れる。広告も1回でOK。
🏭② 生産シナジー 同じキッチン・機械・材料を使い回す。砂糖もオーブンも両方に使える。
💰③ 投資シナジー 一度買った設備・研究を使い回す。高いミキサーを2つの商品で使えばムダがない。
🧑‍🏫④ 経営管理シナジー お母さんの「お店のやり方・コツ・経験」を両方に活かす。失敗の知恵を共有。
ひっかけ：4類型は販売・生産・投資・経営管理の4つ。「財務（お金集め）シナジー」はこの4つに入らないので、選択肢に出たら×。
🎵 覚え方：「ハ・セ・ト・ケ」ハンバイ（販売）・セイサン（生産）・トウシ（投資）・ケイエイ管理
💼 実践：事例で使うなら
事例企業：地方の老舗和菓子製造業A社。本業の売上が伸び悩み、観光客向けに直営カフェを新設したい。「この多角化に勝算はあるか？」を診断する。

▶ 4類型に分けて「効くシナジー」を1つずつ説明する

  • 販売：既存の和菓子ファンの顧客名簿・店頭にカフェを告知。カフェ来店客に手土産の和菓子も売れる（クロスセル）。
  • 生産：和菓子工房の餡（あん）・厨房設備をカフェの甘味メニューに転用。新規投資を抑えられる。
  • 投資：すでに持つ冷蔵設備・包装機を両事業で共用＝設備の稼働率が上がる（範囲の経済）。
  • 経営管理：老舗の品質管理・接客のノウハウをカフェ運営にそのまま活かせる。
📝 助言の型：「A社は生産・投資シナジー（既存設備の転用）と販売シナジー（顧客基盤の活用）が見込め、初期投資を抑えつつ既存事業との相乗効果が期待できる」と、どのシナジーがどこに効くかを具体的に書くと得点になる。
逆に「シナジーが薄い多角化」は新規投資・新規顧客開拓の負担が重く非推奨と判断する。
🔑 2次・1次のキーワード：範囲の経済経営資源の共有・転用既存顧客基盤財務シナジーは含まない
📝 1行過去問チェック
問1「アンゾフのシナジーは、販売・生産・投資・経営管理・財務の5類型に分類される。」
✕ 正しくは4類型（販売・生産・投資・経営管理）。財務シナジーは含まない。「5つ」「財務」と書いてあったら誤り。
問2「販売シナジーとは、同一の流通チャネルや顧客基盤を共有することで得られる相乗効果である。」
○ 正しい。販売＝売る場所・お客の共有。生産（工場・材料）／投資（設備）／経営管理（ノウハウ）と取り違えないこと。
⚠ 取り組む時の注意：①「財務」を混ぜる選択肢が定番のワナ。②類型名と中身（販売＝顧客／生産＝設備…）の組み合わせ入れ替えに注意。まず「4つだけ」と指を折って確認する。
✏️ ミラー穴埋め
シナジー4類型
① 同じお店・お客で得＝販売シナジー
② 同じ工場・材料で得＝生産シナジー
③ 買った設備の使い回し＝投資シナジー
④ やり方・コツの共有＝経営管理シナジー
※4つに入らないのは＝財務シナジー
🧩 まずはたとえ話

「多角化」＝今の商売だけじゃなく、別の商売にも手を広げること。広げ方には4つの型があるよ。

あなたがカレー屋さんをやっているとして、次のどれで広げる？

🍛 → 🍝 / 🌾 / 🍜 / 🎮
🔑 「今の強みを使い回せる近い分野（関連型）」に広げると、いちばん儲かりやすい！

図：多角化の4つの型

↔️① 水平型 今と似たものを横に広げる。カレー屋がハヤシライスも。同じお客・同じ作り方。
↕️② 垂直型 作る→売るの流れの上下へ。自分で小麦を作る（後方）／自分のお店を出す（前方）。
⭐③ 集中（関連）型 技術や販路を使い回せる近い分野。カレー屋がレトルト食品工場へ。いちばん儲かる！
🎲④ 集成（非関連）型 全然ちがう分野へ。カレー屋がゲーム会社も。危険の分散になるが儲けは小さめ。コングロマリット。

図：垂直型の「向き」をまちがえない

🌾 材料川上
→
🍛 自分（製造）本業
→
🏪 お店川下
材料側（左・川上）へ伸ばす＝後方統合／お店側（右・川下）へ伸ばす＝前方統合。
「お客さんに近づく＝前」と覚えると逆にしない。
🎵 ルメルト先生の結論：「関連（集中）型がいちばん儲かる」「非関連型が高収益」は×。バラバラだと強みが使い回せないので儲けは小さい
💼 実践：事例で使うなら
事例企業：従業員30名の印刷会社B社。デジタル化で印刷需要が減少。新規事業で生き残りたいが「どの方向に多角化すべきか」を助言する。

▶ 「自社の強みが使い回せる近い分野（＝関連型）」を選ぶ

  • ◎関連（集中）型を推奨：印刷で培ったデザイン力・地元企業の顧客基盤を活かし、Web制作・販促支援・パッケージ企画へ。技術と顧客を使い回せてシナジー大。
  • △垂直型：自社で用紙調達（後方）や直販EC（前方）へ。コスト・販路は強化できるが投資負担に注意。
  • ✕非関連（集成）型：飲食・不動産など畑違いは、印刷の強みが一切使えずリスクだけ高い。資金・人材の乏しい中小には不向き。
📝 助言の型：「B社は既存の強み（デザイン力・顧客基盤）を転用できる関連分野へ多角化すべき。ルメルトの研究でも関連型が最も高収益。畑違いの非関連多角化はシナジーが効かず、限られた経営資源を分散させるため避ける」。
＝中小の多角化は「強みの隣（範囲の経済）」が鉄則。
🔑 キーワード：コアコンピタンスの転用関連多角化経営資源の分散回避シナジー
📝 1行過去問チェック
問1「ルメルトの研究では、非関連（集成）型多角化が最も高い収益性を示した。」
✕ 最も高収益なのは関連（集約）型。非関連はシナジーが効かず収益は低め。「非関連＝分散効果で高収益」は典型的なワナ。
問2「製造業が自社製品の販売店を展開することは前方統合にあたる。」
○ 正しい。お客側（川下）へ＝前方統合。原材料側（川上）へ＝後方統合。「お客に近づく＝前」で覚える。
⚠ 取り組む時の注意：①関連/非関連の収益性を逆にした文に注意。②前方/後方の向きの入れ替えが頻出。③「同業を買収＝垂直統合」は誤りで水平統合。まず方向（川上/川下）を矢印で図に書くとミスが減る。

💴 まずはたとえ話

「賃金支払の5原則」＝会社が給料を払うときの5つの約束（労働基準法24条）。
「おこづかいのルール」だと思うと一気にわかるよ。

💴 → 🧒 (まちがいなく・ちゃんと・きちんと)
🔑 給料は「お金で・本人に・全部・毎月・決まった日に」払う！

図：5原則＝おこづかいの5つの約束

💴① 通貨払 ちゃんとお金で払う。おもちゃ券や品物で払うのはダメ。
例外：本人がOKなら銀行振込・スマホ払いも可
🤝② 直接払 本人に直接渡す。お兄ちゃんや親が代わりに受け取るのはダメ。
例外なし！代理は不可
💯③ 全額払 全部払う。勝手に天引きしない。
例外：税金・社会保険料、労使協定があれば天引きOK
📅④ 毎月1回以上払 毎月かならず1回は払う。
ボーナス・決算手当は対象外
🗓️⑤ 一定期日払 決まった日に払う。「毎月25日」はOK。
「第4金曜日」は月によって日付がズレるのでダメ！
🎵 覚え方：「通・直・全・毎・期」通貨・直接・全額・毎月1回以上・一定期日
テストの急所：①「第4金曜日に支払う」は×（日が動くから一定期日に反する）。②直接払は例外なし＝家族への代理払いも違反（※本人同意の銀行振込はOK）。
💼 実践：事例で使うなら（労務トラブル診断）
事例企業：資金繰りが苦しい中小企業C社。社長が「コスト削減」のつもりで始めた給与の扱いに、どこが違法かを指摘する。

▶ 「やりがちなNG」を5原則で1つずつ斬る

  • 業績悪化で給料の一部を自社商品券で支給 → ①通貨払違反。
  • 本人が忙しいので親に給料を渡した → ②直接払違反（例外なし）。
  • 制服代・親睦会費を勝手に天引き → ③全額払違反（天引きには労使協定が必要）。
  • 資金繰りで「今月は来月とまとめて2か月分」 → ④毎月1回以上払違反。
  • 支払日を「毎月第4金曜」に設定 → ⑤一定期日払違反（日付がズレる）。
📝 助言の型：「賃金規程・就業規則に支払方法を明記し、5原則を順守する。天引きが必要な項目（社宅費・財形等）は労使協定（24協定）を締結し、法定控除（税・社会保険料）以外は本人同意を得る」。
＝資金繰り対策を“給与のいじり方”で解決しようとすると即・労基法違反になる、と止めるのが診断士の役割。
🔑 キーワード：賃金規程・就業規則労使協定（24協定）法定控除のみ天引き可
📝 1行過去問チェック
問1「賃金は、労働者の同意があれば銀行口座への振込で支払うことができる。」
○ 正しい。通貨払の原則の例外として、本人同意があれば口座振込が可能（2023年〜一定要件でスマホ払いも）。
問2「賃金の支払日を『毎月第4金曜日』と定めても一定期日払の原則に反しない。」
✕ 反する。「第4金曜」は月により日付が変動するため不可。「毎月25日」のように日付を固定すればOK。
⚠ 取り組む時の注意：①直接払は例外なし＝代理人・家族への支払いは振込でも不可（本人名義口座への振込はOK）。②天引きは税・社会保険料以外は労使協定が必要。③「一定期日＝日付固定」。5原則のどれに反するかを1つ特定するのがコツ。
✏️ ミラー穴埋め
賃金支払の5原則
① お金で払う＝通貨払
② 本人に渡す＝直接払（例外なし）
③ 天引き原則禁止＝全額払
④ 毎月かならず＝毎月1回以上払
⑤ 決まった日に＝一定期日払（第4金曜は×）

サイモン先生は「人が物事を決めるときの順番」を整理したよ。
これは「晩ごはんを決める手順」とまったく同じ！

🔑 順番は「情報 → 設計 → 選択」。この順番を入れかえた問題はひっかけ！

図：サイモンの意思決定プロセス

①情報活動
🔍 何が問題か調べる
（冷蔵庫を見る）
→
②設計活動
📝 案を作る
（メニューを考える）
→
③選択活動

元ノートでは、選んだ後に「④実行活動（実際にやってみて確認）」まで描くこともある。
かんたん解説（おまけ）：人間は神様じゃないから、全部の案を比べてベストを選ぶ（最適化）なんて無理。だから「これで十分OK」という案で決める＝満足化。これも同じサイモン先生の考え（限定合理性）。
🎵 覚え方：「じょう→せっ→せん」情報→設計→選択（＝晩ごはん：見る→考える→決める）
💼 実践：事例で使うなら
事例企業：勘と度胸で決める社長のD社。新商品が当たり外れ続きで、意思決定の質を上げたい。なぜ失敗するのかをプロセスで説明する。

▶ 「どの活動が抜けているか」を診断する

  • ①情報活動が薄い：市場・顧客・競合を調べず（＝SWOTや市場調査をせず）、いきなり「これを作る」と決めている → まず情報活動の充実を助言。
  • ②設計活動が単線：代替案を1つしか作らない → 複数の代替案を比較する仕組みを作る。
  • ③選択活動：基準なく社長の好みで決定 → 判断基準（採算・適合性）を事前に決める。
📝 ここがポイント：中小企業は情報も人材も限られる＝限定合理性が強い。だから全部を分析し尽くす「最適化」は現実的でなく、必要な範囲で調べて「これで十分」で素早く決める＝満足化がむしろ合理的。
※診断士の助言フロー（現状把握→課題抽出→施策提案）自体が、この「情報→設計→選択」と同じ構造。
🔑 キーワード：情報活動の充実（SWOT）代替案の比較限定合理性満足化＝スピード
📝 1行過去問チェック
問1「サイモンの意思決定プロセスは、設計→情報→選択の順で進む。」
✕ 正しい順は情報→設計→選択（→実行）。順番の入れ替えがそのままワナになる。
問2「限定合理性のもとでは、意思決定者は最適解ではなく満足解を選ぶ。」
○ 正しい。完全に比較し尽くす「最適化」は不可能なので、願望水準を満たした案で打ち切る＝満足化。
⚠ 取り組む時の注意：①プロセスの順番を入れ替えた文に注意（「情報」が最初）。②最適化（経済人）/満足化（経営人）の取り違え。③「人名」＝サイモンか確認（ゴミ箱モデルはコーエン＝マーチ＝オルセン）。

🏛️ まずはたとえ話

会社の「決めごと」は、立場（えらさ）によって3段階に分かれるよ。
学校でたとえると一発でわかる！

🏫 校長先生 ／ 👩‍🏫 担任の先生 ／ 🧒 日直の子

上の人ほど「会社をどっちへ進めるか」という大きく・先の・めったにない決定。下の人ほど「今日やること」という細かく・毎日くり返す決定をするよ。

🔑 上＝戦略（長期）／中＝管理（資源の配分）／下＝業務（毎日の作業）

図：意思決定の3階層（アンゾフ）

① 戦略的意思決定トップ＝校長先生
② 管理的意思決定ミドル＝担任の先生
③ 業務的意思決定現場＝日直の子
階層だれが決めること（学校でいうと）性質
戦略的トップ新規参入・多角化・全社方針
（＝学校の方針・新コース新設）		長期・めったにない
管理的ミドル人や予算の配分・組織編成
（＝クラスの席替え・予算配分）		中期
業務的現場日々の作業・在庫・受発注
（＝今日の日直・掃除当番）		短期・毎日くり返す
取り違え注意：サイモンの「定型／非定型」は“決め方”の話。アンゾフの「戦略的/管理的/業務的」は“立場・レベル”の話＝別物。人名と切り口をまぜない。
🎵 覚え方：「戦・管・業（せん・かん・ぎょう）＝上から順」戦略的→管理的→業務的。えらい順＝大きい決定の順
💼 実践：事例で使うなら（よくある中小の悩み）
事例企業：創業社長のE社。社長が見積り・発注・クレーム対応まで全部こなし、毎日忙殺されて将来の事業計画を考える時間がない。事業承継も控える。

▶ 「社長が下の階層の決定までやってしまっている」と診断

  • 社長が業務的意思決定（日々の発注・段取り）に時間を取られ、本来やるべき戦略的意思決定（新市場・後継者育成・全社方針）が手薄。
  • ＝「社長のプレイヤー化」。中小に超頻出のパターン。
📝 助言の型：「業務的・管理的意思決定を幹部・現場へ権限委譲し、社長は戦略的意思決定に専念できる体制をつくる。これにより日常業務が回りつつ、事業承継・新規事業など長期課題に経営資源を振り向けられる」。
＝権限委譲と階層に応じた役割分担が定番の打ち手。
🔑 キーワード：権限委譲社長は戦略に専念事業承継プレイヤー化の解消

問1「アンゾフは、意思決定を戦略的・管理的・業務的の3階層に分類した。」

🔑 安定した環境＝機械的／変化の激しい環境＝有機的

図：機械的組織 vs 有機的組織

⚙️ 機械的組織

厳しいルール／集権・階層的（上から命令）／タテの命令
👉 安定した環境に向く
例：毎日同じことをする給食当番・工場のライン

🌿 有機的組織

柔軟な責任／分権・現場に任せる／ヨコの連携
👉 変化の激しい環境に向く
例：その場で動きが変わる運動会の準備・ベンチャー
頻出ひっかけ：「安定＝有機的／変動＝機械的」と逆にした選択肢がよく出る。
正しくは安定＝機械的・変動＝有機的。「どんな時も最適な1つの形はない（唯一最適は無い）」＝コンティンジェンシー（条件適合）理論。
🎵 覚え方：「キカイは安定、ユウキは変化」機械的⚙️＝安定・かたい／有機的🌿＝変化・やわらかい（生き物っぽい）
💼 実践：事例で使うなら
事例企業：大手の安定受注に支えられてきた下請製造業F社（機械的組織）。主要取引先の海外移転で受注が激減し、自社製品開発・新規顧客開拓へ舵を切る必要が出てきた。

▶ 「環境が変わったのに組織が古いまま」と診断

  • これまでは安定環境＝機械的組織（決められた仕様を正確に・効率よく作る）で正解だった。
  • しかし環境が不安定化（市場変化・新事業）した今は、有機的な要素が必要 → 開発プロジェクトチーム、現場への権限委譲、部門横断の連携。
📝 助言の型：「環境変化に組織を適合させる（コンティンジェンシー）。新規開発には有機的なプロジェクト組織を設け、現場の裁量と部門間連携を高める。一方、量産部門は効率重視の機械的運用を残す＝両方を使い分ける」。
＝中小は規模が小さくもともと有機的に動きやすいのが強み、と前向きに位置づけられる。
🔑 キーワード：環境適合（コンティンジェンシー）プロジェクト組織権限委譲・部門間連携両利き
📝 1行過去問チェック
問1「バーンズ＆ストーカーによれば、変化の激しい環境には有機的組織が適合する。」
○ 正しい。不安定・変化が激しい＝有機的（柔軟・分権）。安定＝機械的（集権・階層）。
問2「安定した環境には有機的組織が、変化の激しい環境には機械的組織が適している。」
✕ 完全に逆。安定＝機械的／変動＝有機的。この“逆さま選択肢”が最頻出のワナ。
⚠ 取り組む時の注意：①安定⇄変動と機械的⇄有機的の対応を逆にした文を必ず疑う。②「環境に関わらず最適な組織形態が1つある」は誤り＝コンティンジェンシー理論（唯一最適は無い）。「キカイ＝安定」と先に固定して読む。

💼 実践：事例で使うなら
事例企業：機能別組織のままの製造業G社。複数の製品分野に手を広げた結果、すべての判断が社長に集中し意思決定が遅延、製品ごとの採算も見えない。

▶ 「成長段階に組織が追いついていない」と診断

  • 機能別は専門性に優れるが、複数事業になると調整がトップに集中→遅延。今のG社の症状そのもの。
  • 事業部制へ移行すれば、製品ごとに権限委譲＋独立採算（利益責任）でスピードと採算管理が改善。
  • ただし規模が小さいと資源・機能の重複コストが重い → 会社の規模・事業数しだいで、機能別維持が合理的な場合もある。
📝 助言の型：「多角化・複数事業化が進んだG社は、事業部制へ移行し各事業に権限委譲・利益責任を持たせることで、意思決定の迅速化と製品別採算の明確化を図る（チャンドラー＝組織は戦略に従う）。一方、重複コストに留意し、小規模なら機能別の維持も選択肢」。
＝「正解の形は1つでない・戦略と規模に合わせて選ぶ」が結論。
🔑 キーワード：組織は戦略に従う権限委譲・独立採算意思決定の迅速化資源重複に留意
